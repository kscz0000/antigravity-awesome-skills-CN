# Filen
## 配置
Filen 的初始设置要求你为自己的账户获取一个 API key，
目前仅可通过 [Filen CLI](https://github.com/FilenCloudDienste/filen-cli) 完成。
这意味着你必须先下载 CLI，登录，然后运行 `export-api-key` 命令。

**注意：** 如果你更改了 Filen 账户密码，则需要使用新密码和一个重新导出的 API key（通过 `export-api-key`）来更新你的 rclone 配置。

下面以创建一个名为 `FilenRemote` 的 remote 为例。首先运行：

     rclone config

### 修改时间和哈希值
文件完全支持修改时间，对于目录，仅创建时间是有效的。

Filen 支持 Blake3 哈希。

### 受限的文件名字符

### 标准选项

以下是 filen (Filen) 特有的标准选项。

#### --filen-email

Filen 账户的邮箱

属性:

- Config:      email
- Env Var:     RCLONE_FILEN_EMAIL
- Type:        string
- Required:    true

#### --filen-password

Filen 账户的密码

**注意** 此项输入必须经过混淆处理 — 参见 [rclone obscure](/commands/rclone_obscure/)。

属性:

- Config:      password
- Env Var:     RCLONE_FILEN_PASSWORD
- Type:        string
- Required:    true

#### --filen-api-key

Filen 账户的 API Key

使用 Filen CLI 的 export-api-key 命令获取
你可以从 https://github.com/FilenCloudDienste/filen-cli 下载 Filen CLI

**注意** 此项输入必须经过混淆处理 — 参见 [rclone obscure](/commands/rclone_obscure/)。

属性:

- Config:      api_key
- Env Var:     RCLONE_FILEN_API_KEY
- Type:        string
- Required:    true

### 高级选项

以下是 filen (Filen) 特有的高级选项。

#### --filen-upload-concurrency

分块上传的并发数。

这是同一文件并发传输数量的上限。
将此值设置为小于 1 的值将导致上传死锁。

如果你正通过高速链路上传少量大文件，
且这些上传未能充分利用你的带宽，那么增大该值
可能有助于提升传输速度。

属性:

- Config:      upload_concurrency
- Env Var:     RCLONE_FILEN_UPLOAD_CONCURRENCY
- Type:        int
- Default:     16

#### --filen-encoding

后端的编码方式。

更多信息请参阅 [概述中的编码章节](/overview/#encoding)。

属性:

- Config:      encoding
- Env Var:     RCLONE_FILEN_ENCODING
- Type:        Encoding
- Default:     Slash,Del,Ctl,InvalidUtf8,Dot

#### --filen-master-keys

Master Keys（仅供内部使用）

属性:

- Config:      master_keys
- Env Var:     RCLONE_FILEN_MASTER_KEYS
- Type:        string
- Required:    false

#### --filen-private-key

- Type:        string

#### --filen-public-key

属性:

- Config:      base_folder_uuid

- Type:        string
- Required:    false

#### --filen-description

remote 的描述。

属性:

- Config:      description

- Type:        string
- Required:    false
