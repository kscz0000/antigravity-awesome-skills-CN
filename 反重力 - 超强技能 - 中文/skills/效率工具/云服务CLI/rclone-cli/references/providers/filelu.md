# FileLu

[FileLu](https://filelu.com/) 是一个可靠的云存储提供商，
提供安全文件上传与下载、灵活的存储选项以及共享能力等功能。
凭借对大容量存储的支持以及与 rclone 的无缝集成，FileLu 让
云端文件管理变得轻松。
其跨平台文件备份服务让你可以从任何联网设备
上传和备份文件，无论是电脑、手机还是其他终端，
都能轻松完成。

**注意** FileLu 现已提供一个功能完备的 S3 后端 [FileLu S5](/s3#filelu-s5)，
它是一个符合行业标准的 S3 兼容对象存储。

## 配置


### 路径

不以 `/` 开头的路径将在 `Rclone` 目录中操作。

以 `/` 开头的路径将在根目录操作，你可以在那里看到
`Rclone` 目录。

```console
$ rclone lsf TestFileLu:/
CCTV/
Camera/
Documents/
Music/
Photos/
Rclone/
Vault/
Videos/
```

### 示例命令

在 `Rclone` 目录下创建一个名为 `foldername` 的新文件夹：

```console
rclone mkdir filelu:foldername
```

在 FileLu 上删除一个文件夹：

```console
rclone rmdir filelu:/folder/path/
```

在 FileLu 上删除一个文件：

```console
rclone delete filelu:/hello.txt
```

列出你 FileLu 账户中的文件：

```console
rclone ls filelu:
```

列出所有文件夹：

```console
rclone lsd filelu:
```

将特定文件复制到 FileLu 根目录：

```console
rclone copy D:\hello.txt filelu:
```

将本地目录中的文件复制到 FileLu 目录：

```console
rclone copy D:/local-folder filelu:/remote-folder/path/
```

从 FileLu 下载一个文件到本地目录：

```console
rclone copy filelu:/file-path/hello.txt D:/local-folder
```

将本地目录中的文件移动到 FileLu 目录：

```console
rclone move D:\local-folder filelu:/remote-path/
```

将本地目录中的文件同步到 FileLu 目录：

```console
rclone sync --interactive D:/local-folder filelu:/remote-path/
```

将远程挂载到本地 Linux：

```console
rclone mount filelu: /root/mnt --vfs-cache-mode full
```

将远程挂载到本地 Windows：

```console
rclone mount filelu: D:/local_mnt --vfs-cache-mode full
```

获取 FileLu 账户的存储信息：

```console
rclone about filelu:
```

该后端支持所有其他 rclone 命令。

### 使用 FolderID 代替文件夹路径

我们使用 FolderID 而不是文件夹名称，以避免在用户存在
相同文件夹名称或路径时出错。例如，如果一个用户拥有
两个或三个名为 "test_folders" 的文件夹，系统可能
产生混淆而不知道要移动哪一个。在大型存储
系统中，某些客户端拥有数十万个文件夹和数百万个文件，
重复的文件夹名称或路径相当常见。

### 修改时间和哈希

FileLu 同时支持修改时间和 MD5 哈希。

FileLu 仅支持最长 255 个字符的文件名和文件夹名，
其中字符为 Unicode 字符。

### 重复文件

在使用 Rclone 上传和同步时，FileLu 不允许在同一目录中
上传重复的文件。但是，你仍可以上传
重复文件，只要它们位于不同的目录（文件夹）中。

### 标准选项

以下是 filelu（FileLu 云存储）特有的标准选项。

#### --filelu-key

你在 My Account 中获得的 FileLu Rclone key

Properties:

- Config:      key
- Env Var:     RCLONE_FILELU_KEY
- Type:        string
- Required:    true

## 局限性

该后端使用一个实现了 FileLu API 的自定义库。虽然
它支持文件传输，但部分高级功能可能尚不可用。
如遇任何问题，请到 [rclone 论坛](https://forum.rclone.org/)
报告以便进行故障排查和获取更新信息。

更多信息，请访问 [FileLu 官方网站](https://filelu.com/)。
