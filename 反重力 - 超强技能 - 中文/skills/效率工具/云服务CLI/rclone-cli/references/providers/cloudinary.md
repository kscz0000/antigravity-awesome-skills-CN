# Cloudinary

这是 [Cloudinary](https://cloudinary.com/) 平台的一个后端

## 关于 Cloudinary

[Cloudinary](https://cloudinary.com/) 是一个图像与视频 API 平台。
被 150 万开发者以及 1 万家企业和高增长公司所信赖，
作为其技术栈中交付富有视觉吸引力
体验的关键组成部分。

## 账户与定价

要使用此后端，你需要在 Cloudinary 上
[创建一个免费账户](https://cloudinary.com/users/register_free)。从具有宽松使用额度的免费套餐开始。然后，随着你的
需求增长，升级到最适合你的套餐。
参阅 [定价详情](https://cloudinary.com/pricing)。

## 保护你的凭据

请参阅 [文档](/docs/#configuration-encryption-cheatsheet)

## 配置

下面是一个 Cloudinary 配置的示例。

首先，创建一个 [cloudinary.com](https://cloudinary.com/users/register_free)
账户并选择一个套餐。

你需要登录并从开发者部分获取账户的 `API Key` 和 `API Secret`。

现在运行

```console
rclone config
```

按照交互式设置流程操作：


列出你的媒体库顶层中的目录。

```console
rclone lsd cloudinary-media-library:
```

创建一个新目录。

```console
rclone mkdir cloudinary-media-library:directory
```

列出目录的内容。

```console
rclone ls cloudinary-media-library:directory
```

### 标准选项

以下是 cloudinary (Cloudinary) 特有的标准选项。

#### --cloudinary-cloud-name

Cloudinary 环境名称

属性:

- Config:      cloud_name
- Env Var:     RCLONE_CLOUDINARY_CLOUD_NAME
- Type:        string
- Required:    true

#### --cloudinary-api-key

Cloudinary API Key

属性:

- Config:      api_key
- Env Var:     RCLONE_CLOUDINARY_API_KEY
- Type:        string
- Required:    true

#### --cloudinary-api-secret

Cloudinary API Secret

属性:

- Config:      api_secret
- Env Var:     RCLONE_CLOUDINARY_API_SECRET
- Type:        string
- Required:    true

#### --cloudinary-upload-prefix

指定美国境外环境的 API 端点

属性:

- Config:      upload_prefix
- Env Var:     RCLONE_CLOUDINARY_UPLOAD_PREFIX
- Type:        string
- Required:    false

#### --cloudinary-upload-preset

用于在上传时选择资源操作的上传预设

属性:

- Config:      upload_preset
- Env Var:     RCLONE_CLOUDINARY_UPLOAD_PRESET
- Type:        string
- Required:    false

### 高级选项

以下是 cloudinary (Cloudinary) 特有的高级选项。

#### --cloudinary-encoding

后端的编码方式。

参见 [概述中的编码部分](/overview/#encoding) 了解更多信息。

属性:

- Config:      encoding
- Env Var:     RCLONE_CLOUDINARY_ENCODING
- Type:        Encoding
- Default:     Slash,LtGt,DoubleQuote,Question,Asterisk,Pipe,Hash,Percent,BackSlash,Del,Ctl,RightSpace,InvalidUtf8,Dot

#### --cloudinary-eventually-consistent-delay

等待 N 秒以使支持后端操作的数据库达到最终一致性

属性:

- Config:      eventually_consistent_delay
- Env Var:     RCLONE_CLOUDINARY_EVENTUALLY_CONSISTENT_DELAY
- Type:        Duration
- Default:     0s

#### --cloudinary-adjust-media-files-extensions

Cloudinary 将媒体格式作为文件属性处理，并将其从名称中剥离，这与大多数其他文件系统不同

属性:

- Config:      adjust_media_files_extensions
- Env Var:     RCLONE_CLOUDINARY_ADJUST_MEDIA_FILES_EXTENSIONS
- Type:        bool
- Default:     true

#### --cloudinary-media-extensions

Cloudinary 支持的媒体扩展名

属性:

- Config:      media_extensions
- Env Var:     RCLONE_CLOUDINARY_MEDIA_EXTENSIONS
- Type:        stringArray
- Default:     [3ds 3g2 3gp ai arw avi avif bmp bw cr2 cr3 djvu dng eps3 fbx flif flv gif glb gltf hdp heic heif ico indd jp2 jpe jpeg jpg jxl jxr m2ts mov mp4 mpeg mts mxf obj ogv pdf ply png psd svg tga tif tiff ts u3ma usdz wdp webm webp wmv]

#### --cloudinary-description

远程配置的描述。

属性:

- Config:      description
- Env Var:     RCLONE_CLOUDINARY_DESCRIPTION
- Type:        string
- Required:    false
