# Linkbox

Linkbox 是一个[私有云盘](https://linkbox.to/)。

## 配置

下面是为 Linkbox 创建一个 remote 的示例。

你需要准备：
- 来自 https://www.linkbox.to/admin/account 的 API token
- 你的 Linkbox 账号的邮箱和密码

**注：** 从 v1.74 起，除了 API token 外，还需要邮箱和密码。API token 用于上传和修改，而邮箱/密码认证用于列出和下载文件。如果你已有 Linkbox remote，需要使用 `rclone config` 重新配置，编辑该 remote 并补上邮箱和密码。

首次运行：

```console
rclone config
```

这将引导你完成交互式配置流程：

### 标准选项

以下是 linkbox (Linkbox) 特有的标准选项。

#### --linkbox-token

来自 https://www.linkbox.to/admin/account 的 token

Properties:

- Config:      token
- Env Var:     RCLONE_LINKBOX_TOKEN
- Type:        string
- Required:    true

### 高级选项

以下是 linkbox (Linkbox) 特有的高级选项。

#### --linkbox-description

remote 的描述。

Properties:

- Config:      description
- Env Var:     RCLONE_LINKBOX_DESCRIPTION
- Type:        string
- Required:    false
