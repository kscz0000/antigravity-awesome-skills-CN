# Internxt Drive

[Internxt Drive](https://internxt.com) 是一项零知识加密的云存储服务。

路径以 `remote:path` 的形式指定。

路径可以根据需要指定到任意深度，例如 `remote:directory/subdirectory`。

## 局限性

**注意：** Internxt 后端可能无法在所有账户类型上使用。请参考 [Internxt 套餐详情](https://internxt.com/pricing) 或联系 [Internxt 客服](https://help.internxt.com) 来验证 rclone 与你的订阅套餐的兼容性。

### 安全注意事项

认证过程会将你的密码和助记词存储到 rclone 配置文件中。**强烈建议**对 rclone 配置进行加密，以保护这些敏感凭据：

```
rclone config password
```

这会提示你设置一个密码，用于加密整个配置文件。

### 使用示例

```
# List files
rclone ls internxt:

# Copy files to Internxt
rclone copy /local/path internxt:remote/path

# Sync local directory to Internxt
rclone sync /local/path internxt:remote/path

# Mount Internxt Drive as a local filesystem
rclone mount internxt: /path/to/mountpoint

# Check storage usage
rclone about internxt:
```

### 修改时间和哈希值

Internxt 后端不支持哈希。

修改时间从服务器读取，但无法设置。后端报告 `ModTimeNotSupported` 精度，因此修改时间不会用于同步比较。

### 标准选项

以下是 internxt（Internxt Drive）特有的标准选项。

#### --internxt-email

你的 Internxt 账户邮箱。

属性：

- Config:      email
- Env Var:     RCLONE_INTERNXT_EMAIL
- Type:        string
- Required:    true

#### --internxt-pass

密码。

**注意** 此项输入必须经过混淆处理 - 参见 [rclone obscure](/commands/rclone_obscure/)。

属性：

- Config:      pass
- Env Var:     RCLONE_INTERNXT_PASS
- Type:        string
- Required:    true

### 高级选项

以下是 internxt（Internxt Drive）特有的高级选项。

#### --internxt-mnemonic

助记词（仅供内部使用）

属性：

- Config:      mnemonic
- Env Var:     RCLONE_INTERNXT_MNEMONIC
- Type:        string
- Required:    false

#### --internxt-skip-hash-validation

下载文件时跳过哈希校验。

默认情况下，哈希校验是关闭的。将此项设为 false 可启用校验。

属性：

- Config:      skip_hash_validation
- Env Var:     RCLONE_INTERNXT_SKIP_HASH_VALIDATION
- Type:        bool
- Default:     true

#### --internxt-encoding

后端的编码方式。

更多信息请参见 [overview 中的编码章节](/overview/#encoding)。

属性：

- Config:      encoding
- Env Var:     RCLONE_INTERNXT_ENCODING
- Type:        Encoding
- Default:     Slash,BackSlash,CrLf,RightPeriod,InvalidUtf8,Dot

#### --internxt-description

远程的描述。

属性：

- Config:      description
- Env Var:     RCLONE_INTERNXT_DESCRIPTION
- Type:        string
- Required:    false
