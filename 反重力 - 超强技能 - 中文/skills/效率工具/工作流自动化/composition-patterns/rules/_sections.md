# 章节

本文件定义了所有章节、它们的排序、影响级别以及描述。
括号中的章节 ID 是用于对规则进行分组的文件名前缀。

---

## 1. 组件架构（architecture）

**影响：** HIGH  
**描述：** 用于结构化组件的基础模式，以避免 props 膨胀并启用灵活的组合。

## 2. 状态管理（state）

**影响：** MEDIUM  
**描述：** 用于在已组合组件间提升状态和管理共享上下文的模式。

## 3. 实现模式（patterns）

**影响：** MEDIUM  
**描述：** 实现复合组件和上下文 Provider 的具体技术。

## 4. React 19 APIs（react19）

**影响：** MEDIUM  
**描述：** 仅适用于 React 19+。不要使用 `forwardRef`；使用 `use()` 代替 `useContext()`。
