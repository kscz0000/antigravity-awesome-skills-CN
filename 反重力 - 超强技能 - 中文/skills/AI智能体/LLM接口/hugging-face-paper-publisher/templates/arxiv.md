## III. 方法

### A. 问题形式化

设 $X = \{x_1, x_2, ..., x_n\}$ 为输入空间，$Y = \{y_1, y_2, ..., y_m\}$ 为输出空间。我们的目标是学习一个函数 $f: X \rightarrow Y$，使下式最小化：

$$
\mathcal{L}(\theta) = \sum_{i=1}^{N} \ell(f(x_i; \theta), y_i) + \lambda R(\theta)
$$

其中 $\theta$ 表示模型参数，$\ell$ 是损失函数，$R(\theta)$ 是正则化项。

其中 $\sigma$ 是激活函数，$W^{(l)}$ 是权重矩阵，$b^{(l)}$ 是偏置向量。

### C. 训练算法

**算法 1**：训练过程

```
1: 输入：训练数据 D = {(xi, yi)}
2: 初始化参数 θ
3: for epoch = 1 to max_epochs do
4:     for each mini-batch B ⊂ D do
5:         计算损失：L(θ) = 1/|B| Σ ℓ(f(xi; θ), yi)
6:         更新：θ ← θ - η∇θL(θ)
7:     end for
8: end for
9: 返回：训练后的参数 θ*
```
