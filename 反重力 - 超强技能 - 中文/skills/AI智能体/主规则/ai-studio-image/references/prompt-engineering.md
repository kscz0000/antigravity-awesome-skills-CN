# AI Studio Image — 拟人化图像提示词高级指南

## 基本原则（来自 Google）

> "描述场景，不要只列出关键词。"

叙述性和描述性的段落总是优于关键词列表，因为这能充分利用模型对语言的深度理解。

## 官方模板

### 1. 真实照片场景

```
一张真实照片风格的[景别]拍摄[主体]，[动作/表情]，
场景设置在[环境]。由[光照]照亮，营造[氛围/情绪]。
使用[相机/镜头]拍摄，强调[纹理/细节]。
```

### 2. 产品样机

```
[产品]的高分辨率产品照片，放置在[表面]上。
光照：[设置]以[目的]。相机角度：[角度]展示[特征]。
超真实，清晰聚焦于[细节]。
```

### 3. 教育素材

```
创建一个[视觉类型]解释[概念]，风格参考[参考]。
展示[关键元素]和[结果]。设计类似[示例]，
适合[目标受众]。
```

### 4. 图像中的文字

```
为[品牌]创建一个[类型]，文字为"[精确文字]"，使用[字体风格]。
设计应为[风格]，配色方案为[配色方案]。
```

**Imagen 文字限制：25个字符，最多3个不同短语。**
**复杂文字：使用 gemini-pro-image。**

## 拟人化技术

### 手机相机

拟人化图像的秘诀在于模拟手机相机：

- **浅景深**：小镜头创造自然虚化
- **传感器噪点**：尤其在低光环境中
- **镜头畸变**：广角镜头边缘轻微变形
- **不完美的自动曝光**：区域轻微过曝/欠曝
- **颗粒感**：增加画面生命力的有机纹理

### 真实表情

避免影棚摆拍。描述真实瞬间：

- "和朋友聊天时被抓拍到正在大笑"
- "低头看手机时带着淡淡的微笑"
- "专注于工作，没注意到相机"
- "转头看向画面外的东西"

### 真实环境

描述有生活气息的场景：

- "咖啡馆，背景中有模糊的其他顾客"
- "厨房，用过的砧板和切了一半的蔬菜"
- "书桌，有咖啡渍圈、散落的笔和便利贴"
- "公园长椅，地上有落叶，附近有鸽子"

## 提示词摄影术语

### 光照
- 黄金时刻、蓝调时刻、阴天漫射光
- 窗边光、室内混合光
- 逆光带眩光
- 开阔阴影、斑驳林间光

### 镜头和相机
- 85mm 人像镜头、35mm 广角
- f/1.8 浅景深
- 智能手机相机、iPhone 质量
- 自然虚化、奶油般背景

### 构图
- 三分法、偏离中心主体
- 引导线、自然框架
- 负空间、呼吸空间
- 层次深度：前景/中景/背景

### 纹理和细节
- 可见的皮肤毛孔和自然瑕疵
- 面料纹理、材质质感
- 环境纹理：木纹、混凝土、砖块
- 水珠、蒸汽、大气颗粒

## 复杂度级别

### 简单提示词（好）
```
年轻女性在咖啡馆喝咖啡，窗户自然光
```

### 中级提示词（更好）
```
年轻女性坐在温馨咖啡馆的大窗户旁，手拿热拿铁，
晨光创造柔和阴影，真实放松的微笑，穿着休闲毛衣，
用智能手机拍摄
```

### 高级提示词（最佳）
```
一张中近景照片，拍摄一位二十多岁后期的年轻女性，
坐在咖啡馆大窗户旁的木桌边。她双手捧着一个陶瓷拿铁杯，
可见蒸汽，微微侧头带着真实温暖的微笑。柔和的晨光透过窗户
在桌上投下自然的阴影。她穿着一件休闲奶油色针织毛衣，
袖子微微卷起。头发自然造型，不完美。背景显示模糊的
咖啡馆内部和其他顾客。用智能手机相机拍摄，自然景深，
无专业灯光或闪光灯。可见真实皮肤纹理，淡淡的雀斑。
图像感觉温暖、真实、完全不做作——就像朋友在对面
随手拍下的照片。
```

## 常见错误避免

1. **提示词太短** → 结果泛泛
2. **关键词列表** → 不如叙述自然
3. **要求"完美"** → AI 生成看起来人工的东西
4. **忘记背景** → 背景泛泛/空白
5. **不指定相机** → 模型假设专业单反
6. **"完美"皮肤** → 恐怖谷效应，看起来假
7. **影棚灯光** → 失去自然感
8. **模特姿势** → 图库照片风格

## 高级功能

### 多轮对话（Gemini）
使用聊天进行迭代：
1. 生成基础图像
2. "把咖啡杯移到左边"
3. "让光照更暖"
4. "在背景加一株小植物"

### 参考图像（Gemini Pro）
发送最多14张参考图像：
- 6张用于物体（高保真）
- 5张用于人物（角色一致性）

### 思考模式（Gemini Pro）
模型在生成前"思考"——创建中间构图来优化最终结果。适合复杂场景。

### 搜索增强（Gemini Pro）
基于网络实时信息生成图像。
