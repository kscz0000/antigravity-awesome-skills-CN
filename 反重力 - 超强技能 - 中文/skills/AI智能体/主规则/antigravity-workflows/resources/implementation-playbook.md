# Antigravity Workflows 实现手册

本文档说明智能体应如何执行基于工作流的编排。

## 执行契约

对于每个工作流：

1. 确认目标和范围。
2. 选择最匹配的工作流。
3. 按顺序执行工作流步骤。
4. 每个步骤产出一个具体产物。
5. 继续前进行验证。

## 步骤产物示例

- 规划步骤 -> 范围文档或里程碑检查清单。
- 构建步骤 -> 代码变更和实现说明。
- 测试步骤 -> 测试结果和失败分类。
- 发布步骤 -> 上线检查清单和风险日志。

## 安全护栏

- 未经用户明确批准，切勿运行破坏性操作。
- 如果缺少所需技能，说明差距并回退到最接近的可用技能。
- 当涉及安全测试时，确保授权是明确的。

## 建议的完成格式

工作流完成时，返回：

1. 已完成的步骤
2. 产出的产物
3. 验证证据
4. 未解决的风险
5. 建议的后续行动
