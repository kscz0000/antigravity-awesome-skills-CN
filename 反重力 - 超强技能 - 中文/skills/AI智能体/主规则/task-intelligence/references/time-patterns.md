# 按任务类型的历史时间模式

基于生态系统中的真实执行数据。

## 技能创建

| 任务 | 实际耗时 | 备注 |
|------|----------|------|
| 简单技能（仅 SKILL.md，无脚本） | 5-10 分钟 | ~200-400 行 |
| 中等技能（SKILL.md + 1-2 个脚本） | 15-25 分钟 | 如 telegram、whatsapp |
| 复杂技能（多个脚本+参考资料） | 30-60 分钟 | 如 instagram（29个文件） |
| 技能迭代（v1→v2） | 20-40 分钟 | 重写整个章节 |
| 相关技能生态系统（5-6个） | 2-4 小时 | 如 leiloeiro-ia + 5 个模块 |

## 安装与基础设施

| 任务 | 实际耗时 | 备注 |
|------|----------|------|
| skill-installer（1个技能） | 1-3 分钟 | 含10项验证 |
| 生成1个技能的 ZIP | 30秒 | 不使用最大压缩 |
| 生成完整生态系统 ZIP（31个技能） | 1-2 分钟 | 342个文件，~1 MB |
| 完整生态系统审计 | 10-20 分钟 | 扫描+验证+修复 |
| 手动更新 registry | 2-5 分钟 | 当 scan_registry 有 bug 时 |

## API 与集成

| 任务 | 实际耗时 | 备注 |
|------|----------|------|
| API 初始设置（认证+测试） | 10-20 分钟 | 需要有效的 API 密钥 |
| 实现带 HMAC 的 webhook | 15-25 分钟 | 含测试 |
| 完整 OAuth 流程 | 30-60 分钟 | 取决于平台 |
| API 调试（token 过期、403） | 5-30 分钟 | 方差较大 |

## AI 智能体（角色）

| 任务 | 实际耗时 | 备注 |
|------|----------|------|
| 创建角色 v1（500-700 行） | 15-25 分钟 | 使用子智能体 |
| 迭代角色至 v2（1200+ 行） | 20-35 分钟 | 扩展 80-180% |
| 并行创建6个角色（v1） | 25-45 分钟总计 | 并行子智能体 |
| 并行迭代5个角色（v2） | 30-60 分钟总计 | 高质量 |

## 分析与研究

| 任务 | 实际耗时 | 备注 |
|------|----------|------|
| 代码分析（1个文件） | 2-5 分钟 | |
| 完整安全审计 | 15-30 分钟 | 使用 007 |
| 网络调研（5-10 个来源） | 5-10 分钟 | 使用 Explore 子智能体 |
| 复杂 bug 调试 | 15-45 分钟 | 方差较大 |

## 时间乘数

| 因素 | 乘数 |
|------|------|
| 第三方 API 不稳定 | ×1.5 到 ×2.0 |
| 未记录的依赖 | ×1.3 到 ×1.8 |
| Windows vs Linux 路径问题 | ×1.2 |
| 并行子智能体（4+） | ÷2 到 ÷3 |
| 首次接触该领域 | ×1.5 到 ×2.5 |
| 之前做过该任务（活跃记忆） | ×0.5 到 ×0.7 |
