# 配额参考卡

每次任务中估算和管理配额的快速参考。

---

## 冲刺 vs 每周限制

| 限制 | 大小 | 重置周期 |
|-------|------|--------|
| 冲刺限制 | 250 单位 | 每 5 小时 |
| 每周基线 | 2,800 单位 | 每周一次（硬上限） |

5 小时刷新仅补充冲刺——不补充每周基线。

| 操作 | 约计单位 |
|--------|-------------|
| 生成 1 个智能体 | 1–2 |
| 读取 1 个文件（小型） | 1 |

## 任务规模指南

| 任务规模 | 智能体 | 预估冲刺占比 | 安全？ |
|-------------|--------|---------------|-------|
| 微型（1 个文件修复） | 1 Flash | < 5% | 始终安全 |
| 小型（1 个功能） | 2–3 Flash | 15–25% | 是 |
| 中型（完整流程） | 3–4 Mixed | 30–45% | 是 |
| 大型（整个模块） | 5+ Mixed | 50–70% | 需谨慎 |
| 超大（完整应用构建） | 6+ | 70–100% | 分天执行 |

## 配额节省技巧

1. **`.antigravityignore`** → 最大的单项节省。永远优先做这个。
2. **每个任务开启新对话** → 防止历史膨胀消耗 token。
3. **规格优先并行** → 智能体同时运行而非顺序等待。
4. **修复用 Flash** → 永远不要用 Sonnet 修小 bug。
5. **跳过浏览器智能体** → 除非任务确实需要实时网页。
6. **外挂 AI Studio 密钥** → 每周限制用尽时获得免费 Gemini 配额。

## 外挂免费配额（当你碰壁时）

1. 访问 https://aistudio.google.com
2. 创建免费 API 密钥（Gemini Flash，免费层级）
3. 在 Antigravity → 设置 → 模型 → 粘贴 API 密钥
4. 现在运行在你自己的独立配额池上

在每周基线用尽后继续工作时效果很好。
