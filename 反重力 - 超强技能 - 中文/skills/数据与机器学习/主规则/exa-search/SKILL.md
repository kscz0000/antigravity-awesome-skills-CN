# exa-search

## 概述
通过 Exa API 进行语义搜索、相似内容发现和结构化研究

## 何时使用
- 当需要语义/嵌入搜索时
- 当查找相似内容时
- 当按类别搜索（公司、人物、研究论文等）时

## 安装
```bash
npx skills add -g BenedictKing/exa-search
```

## 分步指南
1. 使用上述命令安装技能
2. 配置 Exa API 密钥
3. 在 Claude Code 对话中自然使用
