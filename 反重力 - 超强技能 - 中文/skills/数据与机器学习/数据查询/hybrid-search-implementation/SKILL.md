# 混合搜索实现

结合向量相似度与关键词搜索的模式。

## 适用场景

- 构建需要更高召回率的 RAG 系统
- 需要语义理解与精确匹配相结合
- 查询包含特定术语（名称、代码）
- 改善领域专业词汇的搜索效果
- 纯向量搜索遗漏关键词匹配时

## 不适用场景

- 任务与混合搜索实现无关
- 需要此范围之外的其他领域或工具
