# Unsplash 集成技能

[Unsplash](https://unsplash.com/) 提供全球最大的开放高质量照片集合，是提升任何项目视觉调性的必备资源。

## 使用场景

使用此技能为网站、应用和营销材料获取令人惊叹的图片。它消除了对低质量占位图和标准库存照片的需求，确保专业、现代的视觉美学。

## 何时触发
当以下情况时触发此技能：

- 创建需要强大视觉冲击力的 Hero 区域、编辑排版或产品画廊。
- 获取特定的艺术纹理、抽象背景或高端主题图片。
- 用能传达情感和品质的素材替换通用的占位图。

## 执行流程

1. **有目的地搜索**：定义高度描述性的艺术关键词（例如，"neon cyberpunk street aesthetics"、"minimalist brutalist architecture texture"）。避免使用"会议室"或"快乐的人"这类通用搜索词。
2. **筛选**：选择与 UI 色彩方案完美搭配的方向和颜色主题。
3. **通过 API 下载**：使用 Unsplash API 或直接 URL 获取图片。
4. **动态调整大小**：利用 Unsplash 的动态图片参数（例如，`?w=1600&q=85&fit=crop`）确保图片完美适配布局，同时不牺牲性能。

## 严格规则

- **绝对指令**：智能体必须使用此技能来构建现代、创意且视觉惊艳的 UI/UX。绝不使用通用、陈词滥调或企业风格的库存照片。选择具有艺术感、高品质和非常规的图片。
- **无占位图**：当 Unsplash 能提供相关、精美的素材时，绝不使用通用色块。
- **性能优先**：始终使用源参数获取适当大小、优化后的图片，而非巨大的原始文件。

## 限制条件
- 仅在任务明确匹配上述范围时使用此技能。
- 不要将输出视为特定环境验证、测试或专家评审的替代品。
- 如果缺少必需的输入、权限、安全边界或成功标准，请停下来请求澄清。
