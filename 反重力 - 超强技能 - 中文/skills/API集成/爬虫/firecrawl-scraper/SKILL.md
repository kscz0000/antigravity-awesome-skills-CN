# firecrawl-scraper

## 概述

使用 Firecrawl API 进行深度网页抓取、截图、PDF 解析和网站爬取。

## 何时使用

- 当你需要从网页深度提取内容时
- 当需要页面交互（点击、滚动等）时
- 当你希望进行截图或 PDF 解析时
- 当批量抓取多个 URL 时

## 安装

```bash
npx skills add -g BenedictKing/firecrawl-scraper
```

## 步骤指南

1. 使用上述命令安装本技能
2. 配置 Firecrawl API 密钥
3. 在 Claude Code 对话中自然使用
