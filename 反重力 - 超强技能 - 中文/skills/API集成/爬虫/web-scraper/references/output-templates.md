# 输出模板参考

所有支持的输出格式的完整格式化模板。
每个输出都必须用含元数据的交付封装包裹。

---

## 交付封装（必需）

每个提取结果都必须包含此元数据封装，无论输出格式：

```markdown
## 提取结果

**来源：** [页面标题](https://example.com/page)
**日期：** 2026-02-25 14:30 UTC
**条目数：** 47 条记录
**置信度：** HIGH
**格式：** Markdown 表格

---

[数据在此]

---

**备注：**
- 任何空缺、异常或观察
- 已应用的过滤或排序
- 已抓取的页面（如分页）
```

## Markdown 表格格式

### 标准表格

```markdown
| 名称           | 价格    | 评分 | 库存状态    |
|:---------------|---------:|:------:|:-------------|
| 商品 Alpha    |   $29.99 |  4.5   | 有货         |
| 商品 Beta     |   $49.99 |  4.2   | 有货         |
| 商品 Gamma    |  $119.00 |  4.8   | 预订         |
| 商品 Delta    |   $15.50 |  3.9   | 缺货         |
```

### 对齐规则

| 数据类型    | 对齐方式 | Markdown 语法 |
|:-------------|:----------|:----------------|
| 文本         | 左对齐    | `:---`          |
| 数字         | 右对齐    | `---:`          |
| 居中         | 居中      | `:---:`         |
| 混合/状态   | 左对齐    | `:---`          |

### 含汇总行的表格

```markdown
| 商品          | 销售单位 | 收入       |
|:---------------|----------:|-----------:|
| Widget A       |     1,234 |  $12,340   |
| Widget B       |       567 |   $8,505   |
| Widget C       |     2,890 |  $57,800   |
| **合计**      | **4,691** | **$78,645**|
```

### 宽数据（拆分表格）

当数据列数超过 10 时，按逻辑组拆分：

```markdown
### 基础信息

| 名称    | 类别     | 品牌     | SKU       |
|:--------|:---------|:--------|:----------|
| 项目 A  | 工具     | Acme    | ACM-001   |

### 价格与库存

| 名称    | 价格   | 折扣价    | 库存 | 发货时间 |
|:--------|--------:|-----------:|:------|:---------|
| 项目 A  | $49.99  |    $39.99  | 142   | 2 天     |
```

### 多 URL 对比表

```markdown
| 来源         | 商品        | 价格   | 评分 |
|:-------------|:-----------|--------:|:------:|
| store-a.com  | Laptop X   | $999    |  4.3   |
| store-b.com  | Laptop X   | $949    |  4.5   |
| store-c.com  | Laptop X   | $1,029  |  4.1   |
```

### 截断规则

对于超过 60 个字符的值：
```markdown
| 标题                                                          | 作者     |
|:--------------------------------------------------------------|:--------|
| Introduction to Advanced Machine Learning Techni...            | J. Smith|
```

## JSON 格式

### 标准 JSON 输出

```json
{
  "metadata": {
    "source": "https://example.com/products",
    "title": "商品目录 - Example Store",
    "extractedAt": "2026-02-25T14:30:00Z",
    "itemCount": 3,
    "confidence": "HIGH",
    "fields": ["name", "price", "rating", "availability"],
    "notes": []
  },
  "data": [
    {
      "name": "商品 Alpha",
      "price": 29.99,
      "currency": "USD",
      "rating": 4.5,
      "availability": "有货"
    },
    {
      "name": "商品 Beta",
      "price": 49.99,
      "currency": "USD",
      "rating": 4.2,
      "availability": "有货"
    },
    {
      "name": "商品 Gamma",
      "price": 119.00,
      "currency": "USD",
      "rating": 4.8,
      "availability": "预订"
    }
  ]
}
```

### JSON 键命名

| 规则                 | 示例                            |
|:-----------------------|:----------------------------------|
| camelCase              | `productName`、`unitPrice`        |
| 数字保持为数字         | `29.99` 而非 `"29.99"`            |
| 布尔保持为布尔         | `true` 而非 `"true"`              |
| 缺失值 = null          | `null` 而非 `""` 或 `"N/A"`       |
| 数组用于多项           | `"tags": ["sale", "new"]`         |
| ISO-8601 表示日期      | `"2026-02-25T14:30:00Z"`          |

### 嵌套 JSON（带详情的商品）

```json
{
  "metadata": { "..." : "..." },
  "data": [
    {
      "name": "Laptop Pro X",
      "brand": "TechCo",
      "pricing": {
        "current": 999.99,
        "original": 1299.99,
        "currency": "USD",
        "discount": "23%"
      },
      "rating": {
        "score": 4.5,
        "count": 1234
      },
      "specifications": {
        "processor": "M3 Pro",
        "ram": "16 GB",
        "storage": "512 GB SSD",
        "display": "14.2 inch Retina"
      },
      "availability": {
        "inStock": true,
        "shipsIn": "2-3 个工作日"
      }
    }
  ]
}
```

### 多 URL JSON

```json
{
  "metadata": {
    "sources": [
      "https://store-a.com/laptop-x",
      "https://store-b.com/laptop-x"
    ],
    "extractedAt": "2026-02-25T14:30:00Z",
    "itemCount": 2,
    "confidence": "HIGH"
  },
  "data": [
    {
      "source": "store-a.com",
      "name": "Laptop X",
      "price": 999,
      "currency": "USD",
      "rating": 4.3
    },
    {
      "source": "store-b.com",
      "name": "Laptop X",
      "price": 949,
      "currency": "USD",
      "rating": 4.5
    }
  ]
}
```

## CSV 格式

### 标准 CSV

```csv
# 来源：https://example.com/products
# 提取时间：2026-02-25 14:30 UTC
# 条目数：3 | 置信度：HIGH
name,price,currency,rating,availability
"商品 Alpha",29.99,USD,4.5,"有货"
"商品 Beta",49.99,USD,4.2,"有货"
"商品 Gamma",119.00,USD,4.8,"预订"
```

### CSV 规则

| 规则                              | 示例                          |
|:-------------------------------------|:-------------------------------|
| 始终包含表头行                      | `name,price,rating`            |
| 含逗号字段需加引号                  | `"Smith, John"`                |
| 含引号字段（需转义）                | `"He said ""hello"""`          |
| 含换行字段需加引号                  | `"Line 1\nLine 2"`            |
| UTF-8 编码带 BOM                    | `\xEF\xBB\xBF` 前缀            |
| 使用 `,` 作为分隔符（标准）          | `,`                            |
| 元数据以注释形式（# 前缀）          | `# Source: URL`                |
| null/缺失用空字段表示                | `field1,,field3`               |

### 多 URL CSV

```csv
# 来源：store-a.com, store-b.com
# 提取时间：2026-02-25 14:30 UTC
source,name,price,currency,rating
"store-a.com","Laptop X",999,USD,4.3
"store-b.com","Laptop X",949,USD,4.5
```

## 摘要统计模板

当提取的数据包含数值字段时，包含一个摘要块：

```markdown
### 摘要统计

| 指标    | 价格     | 评分 |
|:----------|----------:|-------:|
| 数量     |        47 |     47 |
| 最小值   |    $12.99 |    2.1 |
| 最大值   |   $299.99 |    5.0 |
| 平均值   |    $67.42 |    4.1 |
| 中位数   |    $54.99 |    4.3 |
```

仅在以下情况包含：
- 数据有数值列
- 提取条目超过 5 条
- 用户可能从聚合视图中受益（价格、评分、数量）

## 文章提取模板

```markdown
## 文章：[标题]

**作者：** 作者名
**发布日期：** YYYY-MM-DD
**来源：** [站点名](URL)

### 摘要
[2-3 句文章内容摘要]

### 关键数据点
- [事实数据点 1]
- [事实数据点 2]
- [统计发现]

### 标签
`tag1` `tag2` `tag3`
```

注意：概括文章内容。不要复制全文以避免版权问题。

## FAQ 提取模板

```markdown
### FAQ：[页面标题]

**来源：** [站点名](URL)
**条目数：** 12 个问题

| # | 问题 | 答案（节选） |
|--:|:---------|:-----------------|
| 1 | 如何重置我的密码？ | 前往 Settings > Security 并点击 "Reset..." |
| 2 | 你们接受哪些付款方式？ | 我们接受 Visa、Mastercard、PayPal 和银行转账... |
```

或以 JSON 形式（FAQ 模式默认）：
```json
{
  "metadata": { "source": "URL", "itemCount": 12, "confidence": "HIGH" },
  "data": [
    { "question": "如何重置我的密码？", "answer": "前往...", "category": "账户" },
    { "question": "接受哪些付款方式？", "answer": "我们接受...", "category": "账单" }
  ]
}
```

## 差异（Diff）输出模板

对比当前提取与上一次运行时：

```markdown
## 提取结果（差异）

**来源：** [页面标题](URL)
**日期：** 2026-02-25 14:30 UTC
**对比时间：** 2026-02-20 10:00 UTC
**变化：** +5 新增、-2 删除、3 修改

---

### 新增条目 (+5)

| 名称           | 价格    | 评分 |
|:---------------|--------:|:------:|
| 商品 Eta    |  $39.99 |  4.6   |
| 商品 Theta  |  $24.99 |  4.1   |
| ...            |         |        |

### 删除条目 (-2)

| 名称           | 价格    | 评分 |
|:---------------|--------:|:------:|
| ~~商品 Alpha~~ | ~~$29.99~~ | ~~4.5~~ |
| ~~商品 Beta~~  | ~~$49.99~~ | ~~4.2~~ |

### 修改条目 (3)

| 名称           | 字段   | 原值        | 新值        |
|:---------------|:--------|:-----------|:-----------|
| 商品 Gamma  | 价格   | $119.00    | $109.00    |
| 商品 Gamma  | 评分  | 4.8        | 4.9        |
| 商品 Delta  | 库存   | 缺货 | 有货 |

---

**摘要：**
- 自上次提取以来新增 5 个商品
- 删除 2 个商品（可能已停产）
- 商品 Gamma 价格下降 $10，评分上升
- 商品 Delta 重新有货
```

## 错误 / 部分结果模板

当提取部分失败时：

```markdown
## 提取结果（部分）

**来源：** [页面标题](URL)
**日期：** 2026-02-25 14:30 UTC
**条目数：** 约 50 条中提取 23 条
**置信度：** LOW
**策略：** A（WebFetch）-> 升级到 B（浏览器）

---

[部分数据]

---

**问题：**
- 27 条无法提取（内容在 JS 渲染之后）
- 5 条价格字段缺失（标记为 N/A）
- 从 WebFetch 自动升级到浏览器后恢复了 15 条

**建议：**
- 显式使用浏览器自动化重新运行以获得完整结果
- 检查站点是否有 API 端点以直接访问数据
- 若被限流可换时间重试
```
