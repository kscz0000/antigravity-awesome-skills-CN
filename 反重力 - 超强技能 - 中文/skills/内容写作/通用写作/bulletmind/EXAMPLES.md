# Bulletmind 输出参考

## 示例 1 - 主题：气候变化

- 气候变化
  - 定义
    - 全球气温和天气模式的长期变化
  - 原因
    - 人类活动
      - 燃烧化石燃料
      - 森林砍伐
      - 工业排放
    - 自然因素（影响较小）
      - 火山活动
      - 太阳变化
  - 影响
    - 全球气温上升
    - 冰川和冰盖融化
    - 海平面上升
    - 极端天气事件
  - 解决方案
    - 减少碳排放
    - 转向可再生能源
    - 重新造林
    - 可持续实践

## 示例 2 - 主题：操作系统

- 操作系统 (OS)
  - 定义
    - 管理计算机硬件和软件资源，并为应用程序运行提供平台和接口的软件。
  - 核心功能
    - 进程管理
      - 处理运行中程序的执行
    - 内存管理
      - 分配和跟踪内存使用
    - 文件系统管理
      - 组织和存储数据
    - 设备管理
      - 控制和协调硬件组件
  - 类型
    - 批处理操作系统
    - 分时操作系统
    - 分布式操作系统
    - 实时操作系统
  - 示例
    - Windows
    - macOS
    - Linux
