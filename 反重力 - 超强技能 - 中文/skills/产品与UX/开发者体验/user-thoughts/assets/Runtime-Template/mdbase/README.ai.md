# user-thoughts mdbase 索引

此目录存储用户提供的项目决策、约束、偏好和计划。请保持用户原始意图完整。

最近更新：从未

## 维护规则

- 保留用户措辞中的细节。不要简化掉约束、数字、否定或示例。
- 按维度对条目进行分类，方便未来智能体快速查找。
- 优先使用现有维度。仅当用户想法确实需要时才创建新维度。
- 新记录按日期追加到 `## yyyy-mm-dd` 标题之下。
- 除非用户明确要求删除，否则不要删除历史内容。
