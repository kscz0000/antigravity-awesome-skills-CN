# 发布前审查清单

使用此清单从 PR 或 git diff 中发现生产发布风险。仅报告已确认问题或需要确认的合理风险对应的类别。

## 数据库与数据变更

- schema 或 ORM 模型变更但缺少对应迁移文件。
- 新增列、枚举、约束、索引、分区、触发器、函数或扩展，需要部署时执行 DDL。
- 破坏性迁移、列重命名、类型变更、约束收紧或数据重写，缺少向后兼容的发布方案或回滚说明。
- 新增必填数据但缺少 seed、回填、管理员设置或一次性 SQL。
- 索引变更可能锁大表或需要并发/在线创建。
- 查询变更依赖生产环境不保证的数据形态。
- 迁移文件存在但未被部署系统或迁移运行器引用。

## 环境与配置

- 新增环境变量、配置键、feature flag、密钥名称或运行时选项，缺少示例/默认值、部署平台更新或 CI/CD 密钥更新。
- 配置键重命名/删除可能破坏现有生产变量。
- 代码读取仅生产环境才有的值但缺少验证或安全失败行为。
- feature flag 缺少文档化的默认状态、负责人、发布计划或关闭开关。
- Docker、Kubernetes、Helm、Terraform、Railway、Vercel、GitHub Actions 等部署配置变更需要手动环境变更。

## 安全与敏感材料

- diff 中新增了私钥、token、密码、证书、cookie、`.pem`、`.key`、`.p12`、`.env`、服务账号 JSON 或云凭据。
- 日志、错误、分析、webhook 或追踪可能暴露 PII、token、session ID 或支付数据。
- 调试端点、管理员绕过、宽松的 CORS、禁用认证、弱 TLS 或临时开发标志。
- IAM、ACL、桶策略、数据库角色、队列权限、webhook 签名或 API key scope 变更未经发布协调。
- 依赖或容器变更存在已知安全敏感行为、原生二进制或 postinstall 脚本。

## 缓存、CDN 与派生状态

- Redis key 格式、命名空间、TTL、序列化或值形态变更，缺少失效策略或向后兼容。
- 代码假设已预热缓存、预计算数据、物化视图、搜索索引或派生表已存在。
- CDN/静态资源路径、缓存头、ETag、版本化或清除需求变更。
- 部分部署期间可能同时提供新旧缓存值。
- 功能移除后遗留的过期缓存 key 可能恢复旧行为。

## 队列、事件与调度器

- 新增 topic、队列、routing key、exchange、订阅、事件类型、cron 作业或定时 worker。
- 生产者与消费者契约变更缺少兼容的部署顺序。
- 缺少 DLQ、重试策略、幂等性、去重 key 或毒消息处理。
- worker 并发、超时、速率限制或背压变更可能导致依赖过载。
- 事件 payload 形态变更缺少版本化或旧消费者兼容。

## 外部服务与资源

- 新增对象存储、CDN、S3、OSS、GCS 或静态资源引用，缺少上传或权限确认。
- 邮件、短信、推送、PDF、图片、翻译或通知模板变更，缺少生产素材更新。
- webhook URL、回调域名、重定向 URI、CORS origin、OAuth 应用、支付提供商或第三方白名单变更。
- 新增云资源、桶、DNS 记录、证书、API 产品、SaaS 设置或配额需求。
- 前端构建资源依赖尚未部署的后端路由或配置。

## 服务依赖与部署顺序

- API 契约变更影响 web、移动端、worker、索引器、调度器或第三方。
- 数据库迁移必须在特定服务版本之前或之后执行。
- worker 应在生产者之后暂停、排空或部署。
- 滚动部署期间的读/写兼容性风险。
- 新的后台作业、队列或缓存消费者需要在应用部署前准备基础设施。
- 回滚不安全——schema、数据、缓存或队列 payload 不向后兼容。

## CI/CD 与发布自动化

- 工作流、Dockerfile、构建脚本、部署脚本、包管理器、lockfile 或构件路径变更。
- 构建时所需的环境变量、密钥、二进制、系统包或运行时版本变更。
- tag/release 工作流依赖 diff 中未更新的文件或输出。
- 迁移、seed、资源上传或缓存清除步骤为手动但未文档化。
- 测试、lint 或类型检查被禁用或缩小范围，涉及发布关键代码。

## 可观测性与运维

- 新增关键路径但缺少日志、指标、追踪、健康检查、仪表盘或告警。
- 错误处理变更但缺少可操作日志或回滚信号。
- 运维变更缺少 runbook、发布清单、事件响应或支持说明。
- 风险功能缺少关闭开关、feature flag 或紧急禁用路径。
- SLO、速率限制、配额或容量影响未被处理。

## 可报告的"无法验证"情形

- 无法拉取远程 PR diff。
- 无法找到 release tag，审计降级为最近提交。
- 部署平台配置、生产密钥、云桶、队列或外部 SaaS 设置无法从本地仓库访问。
- 无法从 blame/log 证据推断负责人。
- diff 过大，无法在可用时间或工具限制内完整检查。
