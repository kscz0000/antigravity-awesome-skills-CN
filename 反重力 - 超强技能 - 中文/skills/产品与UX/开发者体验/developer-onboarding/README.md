# 开发者入门

通过精心设计的快速入门、教程和示例应用，让开发者快速完成 "Hello World"。

## 本技能涵盖内容

- **首次价值时间优化** - 定义并衡量首次价值时刻
- **快速入门清单设计** - 结构、进度指示器、常见失败
- **交互式 vs 静态教程** - 何时使用哪种、混合方式
- **示例应用与模板** - 极简、入门级与生产级模板
- **优雅处理入门失败** - 错误信息、恢复流程、主动检测
- **衡量激活指标** - 激活定义、漏斗、先行指标

## 核心原则

开发者给你的注意力大约只有 10 分钟。每一秒的困惑、每一条没有指引的错误、每一个意外出现的要求，都会让你失去用户。优秀的入门体验像是和一位早已预判你每个问题的伙伴结对编程。

## 速查

**首次价值时间目标：**
- 简单 API：< 5 分钟
- 需要安装的 SDK：< 10 分钟
- 复杂基础设施：< 30 分钟

**激活率基准：**
- 自服务开发者产品为 20-40%

## 何时使用本技能

- 设计新产品入门体验
- 改进激活指标
- 编写快速入门和教程
- 制作示例应用与模板
- 排查开发者激活率低的原因
- 规划入门邮件序列
