# 开发者注册流程

为开发者设计无缝的注册体验，包括 GitHub OAuth、API key 生成以及新手引导个性化。

## 本技能涵盖的内容

- **OAuth 选项** - GitHub 优先策略、Google 考量、邮箱兜底
- **减少表单字段** - 零字段理想、渐进式收集
- **API key 生成体验** - 立即可见、复制模式、代码示例
- **新手引导个性化** - 用例路径、语言检测、行为适配
- **渐进式资料完善** - 何时收集什么，场景化询问
- **衡量注册转化率** - 漏斗指标、激活率、A/B 测试

## 核心原则

每一个多余的字段、每一次多余的点击、每一道验证步骤都在告诉开发者"你不尊重他们的时间"。最优秀的注册流程几乎让人感受不到——开发者从好奇心到写代码用时不超过 60 秒。

## 速查参考

**目标：OAuth 注册到仪表盘**
- 30 秒以内
- 除 OAuth 外零必填字段
- API key 立即可见

**目标：注册到首次 API 调用**
- 10 分钟以内
- 一次点击即可复制 API key
- 仪表盘上展示可运行的代码示例

## 何时使用本技能

- 设计新的开发者产品注册流程
- 优化现有注册转化率
- 评估身份认证服务
- 规划渐进式资料完善策略
- 排查注册漏斗流失点
