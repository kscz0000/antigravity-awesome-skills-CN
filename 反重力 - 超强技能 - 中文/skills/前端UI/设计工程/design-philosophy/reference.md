# 设计哲学参考（Design Philosophy Reference）

在进行哲学导向强或艺术指导类工作时使用。

## 哲学范例（节选）

**具体诗学（Concrete Poetry）** — 通过纪念碑式的形态与大胆的几何传达信息。大块色面、雕塑感字体（极少的巨型单词、极小的标签）、粗野主义的空间划分。融合波兰海报的能量与勒·柯布西耶的语言。文字作为稀少而有力的姿态——永远不要段落，只有视觉架构中不可或缺的词汇。

**色彩语言（Chromatic Language）** — 色彩作为首要信息系统。几何精准；色彩区域创造意义。字体极简——小号无衬线标签；色彩场负责传达。约瑟夫·阿尔伯斯的色彩交互遇上数据可视化。文字只用来锚定色彩已经表达的内容。

**模拟冥想（Analog Meditation）** — 通过质感与留白实现安静的视觉沉思。纸张纹理、墨水晕染、广阔的负空间。摄影与插画主导。字体低语般出现（小而克制）。日式摄影集美学。文字点到为止——短促的短语，绝不长篇解释。

**有机系统（Organic Systems）** — 自然聚集与模块化生长。圆润形态、有机排列、从自然与建筑中汲取色彩。通过图表、空间关系、图标系统传达信息。文字只用于悬浮在空间中的关键标签。

**几何静默（Geometric Silence）** — 纯粹的秩序与克制。基于网格的精准、大胆的摄影或冷峻的图形、戏剧化的负空间。字体精准但极简——小号必要文字、大片安静区域。瑞士形式主义遇上粗野主义的材质诚实。结构本身传达，文字退居其次。
