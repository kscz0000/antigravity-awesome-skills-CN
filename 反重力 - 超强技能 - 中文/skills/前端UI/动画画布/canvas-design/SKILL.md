本指南用于创建设计哲学——即美学运动——并以视觉方式呈现。仅输出 .md、.pdf 和 .png 文件。

分两步完成：
1. 设计哲学创建（.md 文件）
2. 在画布上视觉呈现（.pdf 或 .png 文件）

首先，完成以下任务：

## 设计哲学创建

首先，创建一个视觉哲学（而非布局或模板），通过以下元素诠释：
- 形式、空间、色彩、构图
- 图像、图形、形状、图案
- 极简文字作为视觉点缀

### 核心理解
- 接收内容：用户的微妙输入或指令，作为基础参考，但不应限制创作自由。
- 创建内容：设计哲学/美学运动。
- 后续步骤：同一版本接收哲学后，以视觉方式呈现——创作 90% 视觉设计、10% 必要文字的作品。

考虑以下方式：
- 为艺术运动撰写宣言
- 下一阶段是创作艺术作品

哲学必须强调：视觉表达。空间传达。艺术诠释。极简文字。

### 如何生成视觉哲学

**命名运动**（1-2 个词）："Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams"

**阐述哲学**（4-6 段——简洁但完整）：

捕捉视觉本质，表达哲学如何通过以下维度呈现：
- 空间与形式
- 色彩与材质
- 尺度与节奏
- 构图与平衡
- 视觉层级

**关键准则：**
- **避免冗余**：每个设计要点只提及一次。除非增加新深度，否则避免重复色彩理论、空间关系或排版原则。
- **反复强调工艺**：哲学必须多次强调，最终作品应呈现为耗时无数、精心打磨、出自顶尖专家之手。这种框架至关重要——重复使用"精雕细琢"、"深厚专业积淀"、"一丝不苟"、"大师级执行"等表述。
- **留出创作空间**：对美学方向保持具体，但足够简洁，让下一个 Claude 有空间做出同样高水准工艺的诠释选择。

哲学必须引导下一版本以视觉而非文字表达理念。信息存在于设计中，而非段落里。

### 哲学示例

**"Concrete Poetry"**
哲学：通过纪念碑式形式与大胆几何进行传达。
视觉呈现：巨大色块、雕塑式排版（超大单词、微小标签）、粗野主义空间分割、波兰海报能量与 Le Corbusier 相遇。理念通过视觉重量与空间张力表达，而非解释。文字作为稀有而有力的姿态——绝无段落，仅有融入视觉建筑的必要词汇。每个元素都以大师级工匠的精准度放置。

**"Chromatic Language"**
哲学：色彩作为主要信息系统。
视觉呈现：几何精准，色彩区域创造意义。排版极简——小型无衬线标签让色彩场域传达信息。联想 Josef Albers 的交互与数据可视化相遇。信息以空间和色彩编码。文字仅用于锚定色彩已展示的内容。这是精心校准的色彩成果。

**"Analog Meditation"**
哲学：通过纹理与呼吸空间实现静谧视觉沉思。
视觉呈现：纸张纹理、墨水晕染、广阔留白。摄影与插画主导。排版轻声细语（小、克制、服务于视觉）。日本摄影书美学。图像在页面间呼吸。文字稀疏出现——短句，绝无解释性段落。每幅构图都以冥想般的用心平衡。

**"Organic Systems"**
哲学：自然聚集与模块化生长模式。
视觉呈现：圆润形态、有机排列、从自然到建筑的色彩。信息通过视觉图解、空间关系、图标呈现。文字仅作为漂浮在空间中的关键标签。构图通过专家级空间编排讲述故事。

**"Geometric Silence"**
哲学：纯粹秩序与克制。
视觉呈现：网格精准、大胆摄影或冷峻图形、戏剧性留白。排版精准但极简——小型必要文字、大片静默区域。瑞士形式主义与粗野主义材料诚实相遇。结构传达，而非文字。每个对齐都是无数次精修的成果。

*这些是精简示例。实际设计哲学应为 4-6 个充实段落。*

### 核心原则
- **视觉哲学**：创建一个通过设计表达的美学世界观
- **极简文字**：始终强调文字稀疏、仅限必要、作为视觉元素融入——绝不冗长
- **空间表达**：理念通过空间、形式、色彩、构图传达——而非段落
- **艺术自由**：下一版本以视觉诠释哲学——提供创作空间
- **纯粹设计**：这是创作艺术作品，而非装饰文档
- **专家工艺**：反复强调最终作品必须呈现为精雕细琢、精心打磨、顶尖专家耗时无数的成果

**设计哲学应为 4-6 段。** 以诗意的设计哲学填充，凝聚核心愿景。避免重复相同要点。保持设计哲学通用，不提及艺术意图，仿佛可应用于任何场景。将设计哲学输出为 .md 文件。

## 推导微妙主题

**关键步骤**：创建画布前，识别原始请求中的微妙概念线索。

**核心原则**：
主题是**嵌入艺术本身的微妙、小众主题**——不总是字面呈现，始终精致。熟悉主题的人应直觉感知，其他人则体验一幅精湛的抽象构图。设计哲学提供美学语言。推导出的主题提供灵魂——悄然编织进形式、色彩与构图的概念 DNA。

这**非常重要**：主题必须精致，增强作品深度而不自我宣告。像爵士乐手引用另一首歌——只有懂的人会察觉，但所有人都欣赏音乐。

## 画布创建

哲学与概念框架确立后，在画布上呈现。花片刻整理思绪、清空头脑。使用创建的设计哲学与以下指令，以专家工艺打造杰作，体现哲学的所有方面。

**重要**：对于任何类型内容，即使用户请求电影/游戏/书籍相关内容，方法仍需精致。始终牢记这是艺术，而非卡通或业余作品。

要创作博物馆或杂志级作品，以设计哲学为基础。创建单页、高度视觉化、设计导向的 PDF 或 PNG 输出（除非要求更多页面）。通常使用重复图案与完美形状。将抽象哲学设计视为科学圣经，借用系统观察的视觉语言——密集的标记积累、重复元素或分层图案，通过耐心重复构建意义，回报持续观看。添加稀疏、冷静的排版与系统参考标记，暗示这可能是来自想象学科的图解，以通常用于记录可观察现象的敬意对待不可见主题。用简单短语或细节微妙定位锚定作品，使用有限且有意、协调的色彩调色板。拥抱用分析性视觉语言表达人类体验理念的悖论：结果应感觉像一件文物，证明某种短暂之物可通过仔细观察被研究、绘制与理解。这是真正的艺术。

**文字作为语境元素**：文字始终极简且视觉优先，但让语境引导是轻声细语的标签还是大胆的排版姿态。朋克场地海报可能比极简陶艺工作室形象有更大、更激进的字体。大多数时候，字体应纤细。所有字体使用必须设计导向，优先视觉传达。无论文字尺度如何，无内容溢出页面，无重叠。每个元素必须包含在画布边界内，留有适当边距。仔细检查所有文字、图形与视觉元素有呼吸空间与清晰分隔。这是专业执行的不可妥协要求。**重要：若书写文字，使用不同字体。搜索 `./canvas-fonts` 目录。无论方法如何，精致度不可妥协。**

下载并使用所需字体使其成为现实。发挥创意，让排版真正成为艺术的一部分——若艺术抽象，将字体带入画布，而非数字排版。

要突破边界，遵循设计直觉，以哲学为指导原则。拥抱终极设计自由与选择。将美学与设计推向前沿。

**关键**：要实现人工创作质量（非 AI 生成），创作看起来耗时无数的作品。呈现为顶尖专家一丝不苟打磨每个细节的成果。确保构图、间距、色彩选择、排版——一切彰显专家级工艺。反复检查无重叠、格式完美、每个细节无瑕。创作可向人展示以证明专业、无可否认令人印象深刻的作品。

输出最终结果为单个可下载的 .pdf 或 .png 文件，同时输出使用的设计哲学作为 .md 文件。

## 最终步骤

**重要**：用户已说"还不够完美。必须无瑕，工艺杰作，仿佛即将在博物馆展出。"

**关键**：优化作品时，避免添加更多图形；而是精修已创作内容，使其极其锐利，完全尊重设计哲学与极简原则。与其添加有趣滤镜或重构字体，不如思考如何让现有构图与艺术更协调。若本能是调用新函数或绘制新形状，停下，改为问："如何让已有内容更像一件艺术作品？"

进行第二轮。回到代码，进一步精修/打磨，使其成为哲学设计的杰作。
