# Golden Hour — 黄金时刻

浓郁温暖的秋日色盘，营造亲切而精致的氛围。

## 配色方案

- **Mustard Yellow**：`#f4a900` — 醒目的主强调色
- **Terracotta**：`#c1666b` — 温暖的次级色
- **Warm Beige**：`#d4b896` — 中性背景色
- **Chocolate Brown**：`#4a403a` — 深色文字与锚点

## 字体搭配

- **标题**：FreeSans Bold
- **正文**：FreeSans

## 适用场景

餐饮演示、酒店品牌、秋季营销、温馨生活方式内容、手工艺产品。
