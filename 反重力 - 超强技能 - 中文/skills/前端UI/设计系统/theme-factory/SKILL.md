# 主题工厂技能

提供精选的专业字体与配色主题合集，每套主题均含精心搭配的调色板和字体组合。选定主题后可应用于任意作品。

## 用途

为演示文稿应用统一、专业的样式。每套主题包含：
- 完整的配色方案（含十六进制色值）
- 标题与正文的互补字体组合
- 适用于不同场景和受众的独立视觉风格

## 使用方法

为演示文稿或其他作品应用样式：

1. **展示主题预览**：打开 `theme-showcase.pdf` 文件，让用户直观浏览所有可用主题。不做任何修改，仅供查看。
2. **询问用户选择**：询问要将哪套主题应用于演示文稿
3. **等待确认**：获取用户对所选主题的明确确认
4. **应用主题**：确认主题后，将所选主题的配色和字体应用到演示文稿/作品

## 主题详情

每套主题在 `themes/` 目录下有完整定义，包含：
- 完整的配色方案（含十六进制色值）
- 标题与正文的互补字体组合
- 适用于不同场景和受众的独立视觉风格

## 应用流程

选定主题后：
1. 从 `themes/` 目录读取对应主题文件
2. 在整个演示文稿中统一应用指定的配色和字体
3. 确保对比度和可读性达标
4. 在所有幻灯片中保持主题的视觉一致性

## 创建自定义主题
当现有主题均不适用时，可创建自定义主题。根据用户提供的输入，生成与上述主题风格一致的新主题。为主题取一个能描述字体/配色特征的名称。根据用户提供的描述选择合适的配色和字体。生成主题后展示给用户审阅确认，然后按上述流程应用。
