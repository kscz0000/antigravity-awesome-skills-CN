- 使用 Tailwind 创建组件库
- 实现设计 token 和主题系统
- 构建响应式和无障碍组件
- 统一代码库中的 UI 模式
- 迁移或扩展 Tailwind CSS
- 设置深色模式和配色方案

## 不适用场景
