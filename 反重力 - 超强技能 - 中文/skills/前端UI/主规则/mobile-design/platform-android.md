# Android平台指南

> Material Design 3要点、Android设计规范、Roboto排版与原生模式。
> **构建Android应用时阅读本文件。**

---

## 1. Material Design 3理念

### Material核心原则

```
材质隐喻：
├── 表面存在于3D空间
├── 光影定义层级
├── 动效提供连贯性
└── 大胆、图形化、有意图的设计

自适应设计：
├── 响应设备能力
├── 一套UI适配所有形态
├── 壁纸提取动态色彩
└── 按用户个性化

默认无障碍：
├── 大触控目标
├── 清晰的视觉层级
├── 语义色
└── 动效尊重用户偏好
```

### Material Design价值观

| 价值观 | 实现方式 |
|--------|----------|
| **动态色彩** | 色彩适配壁纸/用户偏好 |
| **个性化** | 用户专属主题 |
| **无障碍** | 内置于每个组件 |
| **响应式** | 适配所有屏幕尺寸 |
| **一致性** | 统一的设计语言 |

## 2. Android排版

### Roboto字体家族

```
Android系统字体：
├── Roboto：默认无衬线
├── Roboto Flex：可变字体（API 33+）
├── Roboto Serif：衬线替代
├── Roboto Mono：等宽
└── Google Sans：Google产品（特殊许可）
```

### Material字号体系

| 角色 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| **Display Large** | 57sp | Regular | 64sp | 英雄区文字、闪屏 |
| **Display Medium** | 45sp | Regular | 52sp | 大标题 |
| **Display Small** | 36sp | Regular | 44sp | 中标题 |
| **Headline Large** | 32sp | Regular | 40sp | 页面标题 |
| **Headline Medium** | 28sp | Regular | 36sp | 分区标题 |
| **Headline Small** | 24sp | Regular | 32sp | 子分区 |
| **Title Large** | 22sp | Regular | 28sp | 对话框、卡片 |
| **Title Medium** | 16sp | Medium | 24sp | 列表、导航 |
| **Title Small** | 14sp | Medium | 20sp | Tab、次要 |
| **Body Large** | 16sp | Regular | 24sp | 主要内容 |
| **Body Medium** | 14sp | Regular | 20sp | 次要内容 |
| **Body Small** | 12sp | Regular | 16sp | 说明文字 |
| **Label Large** | 14sp | Medium | 20sp | 按钮、FAB |
| **Label Medium** | 12sp | Medium | 16sp | 导航 |
| **Label Small** | 11sp | Medium | 16sp | Chip、徽章 |

### 缩放无关像素（sp）

```
sp = 缩放无关像素

sp自动随以下设置缩放：
├── 用户字号偏好
├── 显示密度
└── 无障碍设置

规则：文字始终用sp，其他一切用dp。
```

### 字重使用

| 字重 | 用途 |
|------|------|
| Regular (400) | 正文、展示 |
| Medium (500) | 按钮、标签、强调 |
| Bold (700) | 极少使用，仅强强调 |

## 3. Material色彩系统

### 动态色彩（Material You）

```
Android 12+ 动态色彩：

用户壁纸 → 色彩提取 → 应用主题

你的应用自动适配：
├── 主色（来自壁纸）
├── 辅色（互补色）
├── 第三色（强调色）
├── 表面色（派生）
└── 所有语义色自动调整

规则：实现动态色彩以提供个性化体验。
```

### 语义色角色

```
表面色：
├── Surface → 主背景
├── SurfaceVariant → 卡片、容器
├── SurfaceTint → 高度叠加
├── InverseSurface → Snackbar、工具提示

On-Surface色：
├── OnSurface → 主要文字
├── OnSurfaceVariant → 次要文字
├── Outline → 边框、分隔线
├── OutlineVariant → 微妙分隔线

主色：
├── Primary → 关键操作、FAB
├── OnPrimary → 主色上的文字
├── PrimaryContainer → 较弱强调
├── OnPrimaryContainer → 容器上的文字

辅色/第三色：类似模式
```

### 错误、警告、成功色

| 角色 | 浅色 | 暗色 | 用途 |
|------|------|------|------|
| Error | #B3261E | #F2B8B5 | 错误、破坏性操作 |
| OnError | #FFFFFF | #601410 | 错误色上的文字 |
| ErrorContainer | #F9DEDC | #8C1D18 | 错误背景 |

### 暗黑主题

```
Material暗黑主题：

├── 背景：#121212（默认非纯黑）
├── 表面：#1E1E1E、#232323等（按高度）
├── 高度：越高 = 越浅叠加
├── 降低色彩饱和度
└── 检查对比度

暗黑模式高度叠加：
├── 0dp → 0%叠加
├── 1dp → 5%叠加
├── 3dp → 8%叠加
├── 6dp → 11%叠加
├── 8dp → 12%叠加
├── 12dp → 14%叠加
```

## 4. Android布局与间距

### 布局网格

```
Android使用8dp基准网格：

所有间距为8dp的倍数：
├── 4dp：组件内部（半步）
├── 8dp：最小间距
├── 16dp：标准间距
├── 24dp：分区间距
├── 32dp：大间距

边距：
├── 紧凑（手机）：16dp
├── 中等（小平板）：24dp
├── 展开（大屏）：24dp+或分栏
```

### 响应式布局

```
窗口尺寸类别：

紧凑（< 600dp宽度）：
├── 竖屏手机
├── 单栏布局
├── 底部导航

中等（600-840dp宽度）：
├── 平板、折叠屏
├── 考虑双栏
├── Navigation Rail可选

展开（> 840dp宽度）：
├── 大平板、桌面
├── 多栏布局
├── 导航抽屉
```

### 经典布局

| 布局 | 用途 | 窗口类别 |
|------|------|----------|
| **列表-详情** | 邮件、消息 | 中等、展开 |
| **信息流** | 社交、新闻 | 所有 |
| **辅助面板** | 参考内容 | 中等、展开 |

## 5. Android导航模式

### 导航组件

| 组件 | 用途 | 位置 |
|------|------|------|
| **底部导航** | 3-5个顶级目的地 | 底部 |
| **Navigation Rail** | 平板、折叠屏 | 左侧竖排 |
| **导航抽屉** | 多目的地、大屏 | 左侧，隐藏/可见 |
| **顶部应用栏** | 当前上下文、操作 | 顶部 |

### 底部导航

```
┌─────────────────────────────────────┐
│                                     │
│         内容区域                     │
│                                     │
├─────────────────────────────────────┤
│  🏠     🔍     ➕     ❤️     👤    │ ← 80dp高度
│ 首页   搜索   FAB   收藏   个人     │
└─────────────────────────────────────┘

规则：
├── 3-5个目的地
├── 图标：Material Symbols（24dp）
├── 标签：始终显示（无障碍）
├── 选中态：填充图标 + 指示药丸
├── 角标：用于通知
├── FAB可集成（可选）
```

### 顶部应用栏

```
类型：
├── 居中对齐：Logo应用、简洁
├── 小型：紧凑，滚动时消失
├── 中型：标题 + 操作，折叠
├── 大型：展示标题，折叠为小型

┌─────────────────────────────────────┐
│  ☰   应用标题               🔔 ⋮  │ ← 64dp（小型）
├─────────────────────────────────────┤
│                                     │
│         内容区域                     │
└─────────────────────────────────────┘

操作：最多3个图标，更多用溢出菜单（ ⋮ ）
```

### Navigation Rail（平板）

```
┌───────┬─────────────────────────────┐
│  ≡    │                             │
│       │                             │
│  🏠   │                             │
│ 首页  │       内容区域               │
│       │                             │
│  🔍   │                             │
│ 搜索  │                             │
│       │                             │
│  👤   │                             │
│ 个人  │                             │
└───────┴─────────────────────────────┘

宽度：80dp
图标：24dp
标签：图标下方
FAB：可在顶部
```

### 返回导航

```
Android提供系统返回：
├── 返回按钮（三键导航）
├── 返回手势（从边缘滑动）
├── 预测性返回（Android 14+）

你的应用必须：
├── 正确处理返回（弹出栈）
├── 支持预测性返回动画
├── 绝不意外劫持/覆盖返回
└── 丢弃未保存数据前确认
```

## 6. Material组件

### 按钮

```
按钮类型：

┌──────────────────────┐
│    填充按钮           │  ← 主要操作
└──────────────────────┘

┌──────────────────────┐
│    色调按钮           │  ← 次要，较弱强调
└──────────────────────┘

┌──────────────────────┐
│    描边按钮           │  ← 第三级，更低强调
└──────────────────────┘

    文字按钮               ← 最低强调

高度：
├── 小型：40dp（空间受限时）
├── 标准：40dp
├── 大型：56dp（需要FAB尺寸时）

最小触控目标：48dp（即使视觉更小）
```

### 浮动操作按钮（FAB）

```
FAB类型：
├── 标准：56dp直径
├── 小型：40dp直径
├── 大型：96dp直径
├── 扩展：图标 + 文字，可变宽度

位置：右下角，距边缘16dp
高度：浮于内容之上

┌─────────────────────────────────────┐
│                                     │
│         内容                        │
│                                     │
│                              ┌────┐ │
│                              │ ➕ │ │ ← FAB
│                              └────┘ │
├─────────────────────────────────────┤
│       底部导航                      │
└─────────────────────────────────────┘
```

### 卡片

```
卡片类型：
├── 提升型：阴影，静止态
├── 填充型：背景色，无阴影
├── 描边型：边框，无阴影

卡片结构：
┌─────────────────────────────────────┐
│           头部图片                   │ ← 可选
├─────────────────────────────────────┤
│  标题 / 标题文字                    │
│  副标题 / 辅助文字                  │
├─────────────────────────────────────┤
│      [ 操作 ]    [ 操作 ]           │ ← 可选操作
└─────────────────────────────────────┘

圆角：12dp（M3默认）
内边距：16dp
```

### 文本输入框

```
类型：
├── 填充型：背景填充，下划线
├── 描边型：四周边框

┌─────────────────────────────────────┐
│  标签                               │ ← 聚焦时上浮
│  ________________________________________________
│  │     输入文字...                │ ← 前置/后置图标
│  ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
│  辅助文字或错误信息                  │
└─────────────────────────────────────┘

高度：56dp
标签：从占位符动画到顶部
错误：红色 + 图标 + 消息
```

### Chip

```
类型：
├── 辅助型：智能操作（导航、拨号）
├── 筛选型：切换筛选
├── 输入型：代表实体（标签、联系人）
├── 建议型：动态推荐

┌───────────────┐
│  🏷️ 筛选     │  ← 32dp高度，8dp圆角
└───────────────┘

状态：未选、已选、禁用
```

## 7. Android专属模式

### Snackbar

```
位置：底部，导航栏上方
时长：4-10秒
操作：一个可选文字操作

┌─────────────────────────────────────────────────┐
│  已归档1项                          [ 撤销 ]    │
└─────────────────────────────────────────────────┘

规则：
├── 简短消息，尽量单行
├── 最多2行
├── 一个操作（文字，非图标）
├── 可滑动关闭
└── 不堆叠，排队显示
```

### 底部弹出层

```
类型：
├── 标准：交互内容
├── 模态：阻挡背景（有遮罩）

模态底部弹出层：
┌─────────────────────────────────────┐
│                                     │
│        （内容上方遮罩）              │
│                                     │
├═════════════════════════════════════┤
│  ─────  （拖拽手柄，可选）           │
│                                     │
│        弹出层内容                    │
│                                     │
│        操作 / 选项                   │
│                                     │
└─────────────────────────────────────┘

圆角：28dp（顶部两角）
```

### 对话框

```
类型：
├── 基础：标题 + 内容 + 操作
├── 全屏：复杂编辑（移动端）
├── 日期/时间选择器
├── 确认对话框

┌─────────────────────────────────────┐
│              标题                    │
│                                     │
│       说明文字                       │
│                                     │
│           [ 取消 ]  [ 确认 ]        │
└─────────────────────────────────────┘

规则：
├── 居中显示
├── 背景遮罩（变暗）
├── 最多2个操作，右对齐
├── 破坏性操作可在左侧
```

### 下拉刷新

```
Android使用SwipeRefreshLayout模式：

┌─────────────────────────────────────┐
│         ○ （旋转器）                │ ← 圆形进度
├─────────────────────────────────────┤
│                                     │
│         内容                        │
│                                     │
└─────────────────────────────────────┘

旋转器：Material圆形指示器
位置：顶部居中，随内容下拉
```

### 涟漪效果

```
每个可触控元素都需要涟漪：

按下 → 涟漪从触控点扩散
抬起 → 涟漪完成并淡出

颜色：
├── 浅色上：黑色约12%不透明度
├── 暗色上：白色约12%不透明度
├── 彩色上：适当对比度

这是Android体验的强制要求。
```

## 8. Material Symbols

### 使用指南

```
Material Symbols：Google图标库

风格：
├── Outlined：默认，最常见
├── Rounded：更柔和、友好
├── Sharp：棱角分明、精确

可变字体轴：
├── FILL：0（描边）到1（填充）
├── wght：100-700（字重）
├── GRAD：-25到200（强调）
├── opsz：20、24、40、48（光学尺寸）
```

### 图标尺寸

| 尺寸 | 用途 |
|------|------|
| 20dp | 密集UI、行内 |
| 24dp | 标准（最常见） |
| 40dp | 更大触控目标 |
| 48dp | 强调、独立使用 |

### 状态

```
图标状态：
├── 默认：全不透明度
├── 禁用：38%不透明度
├── 悬停/聚焦：容器高亮
├── 选中：填充变体 + 着色

非活跃 vs 活跃：
├── 非活跃：描边
├── 活跃：填充 + 指示器
```

## 9. Android无障碍

### TalkBack要求

```
每个交互元素需要：
├── contentDescription（它是什么）
├── 正确的语义（button、checkbox等）
├── 状态播报（selected、disabled）
└── 逻辑分组

Jetpack Compose：
Modifier.semantics {
    contentDescription = "播放按钮"
    role = Role.Button
}

React Native：
accessibilityLabel="播放按钮"
accessibilityRole="button"
accessibilityState={{ disabled: false }}
```

### 触控目标尺寸

```
强制：48dp × 48dp最小

即使视觉元素更小：
├── 图标：24dp视觉，48dp触控区域
├── 复选框：20dp视觉，48dp触控区域
└── 加内边距达到48dp

目标间距：最少8dp
```

### 字体缩放

```
Android支持字体缩放：
├── 85%（更小）
├── 100%（默认）
├── 115%、130%、145%...
├── 最高200%（最大）

规则：在200%字号下测试UI。
使用sp单位，避免固定高度。
```

### 减少动效

```kotlin
// 检查动效偏好
val reduceMotion = Settings.Global.getFloat(
    contentResolver,
    Settings.Global.ANIMATOR_DURATION_SCALE,
    1f
) == 0f

if (reduceMotion) {
    // 跳过或减少动画
}
```
