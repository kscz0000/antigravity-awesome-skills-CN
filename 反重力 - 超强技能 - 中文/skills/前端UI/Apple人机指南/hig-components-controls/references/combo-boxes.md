# 组合框

组合框将文本字段与下拉按钮组合在单个控件中。

![显示城市列表的组合框控件的风格化表示。图像带有红色色调，以微妙地反映原始六色 Apple 标志中的红色。](https://docs-assets.developer.apple.com/published/8f828536eb509b9cd023583ecd2f8eca/components-combobox-intro%402x.png)

用户可以在字段中输入自定义值，或点击按钮从预定义值列表中选择。当用户输入自定义值时，它不会添加到选择列表中。

## [最佳实践](https://developer.apple.com/design/human-interface-guidelines/combo-boxes#Best-practices)

**用列表中有意义的默认值填充字段。** 虽然字段默认可以为空，但最好让默认值引用隐藏的选择。默认值不必是列表中的第一项。

**使用介绍性标签让用户知道期望的项目类型。** 通常，标签使用标题样式大写并以冒号结尾。有关相关指导，请参阅[标签](https://developer.apple.com/design/human-interface-guidelines/labels)。

**提供相关的选择。** 用户希望有输入自定义值的能力，以及从最可能的选择列表中选择的便利。

**确保列表项目不比文本字段宽。** 如果项目太宽，文本字段可能会截断它，这对用户来说很难阅读。

有关指导，请参阅[文本字段](https://developer.apple.com/design/human-interface-guidelines/text-fields)和[下拉按钮](https://developer.apple.com/design/human-interface-guidelines/pull-down-buttons)。

## [平台注意事项](https://developer.apple.com/design/human-interface-guidelines/combo-boxes#Platform-considerations)

_iOS、iPadOS、tvOS、visionOS 或 watchOS 不支持。_
