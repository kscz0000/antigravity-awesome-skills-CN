# 图像井

图像井是图像视图的可编辑版本。

![图像井的风格化呈现。图像带有红色色调，以微妙地反映原始六色 Apple 标志中的红色。](https://docs-assets.developer.apple.com/published/101ad769eaeccadb3758ca0b05a4fc5b/components-image-well-intro%402x.png)

选择图像井后，用户可以复制和粘贴其图像或删除它。用户还可以将新图像拖入图像井而无需先选择它。

## [最佳实践](https://developer.apple.com/design/human-interface-guidelines/image-wells#Best-practices)

**必要时恢复为默认图像。** 如果你的图像井需要图像，当用户清除图像井的内容时再次显示默认图像。

**如果你的图像井支持复制和粘贴，确保标准复制和粘贴菜单项可用。** 用户通常期望选择这些菜单项——或使用标准键盘快捷键——来与图像井交互。有关指导，请参阅[编辑菜单](https://developer.apple.com/design/human-interface-guidelines/the-menu-bar#Edit-menu)。

有关相关指导，请参阅[图像视图](https://developer.apple.com/design/human-interface-guidelines/image-views)。

## [平台注意事项](https://developer.apple.com/design/human-interface-guidelines/image-wells#Platform-considerations)

_iOS、iPadOS、tvOS、visionOS 或 watchOS 不支持。_
