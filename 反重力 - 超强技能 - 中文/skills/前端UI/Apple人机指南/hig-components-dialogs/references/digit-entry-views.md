# 数字输入视图

数字输入视图填充整个屏幕，提示用户使用数字专用键盘输入一系列数字，如 PIN。

![Apple TV 五位数密码输入屏幕的概念图。图像带有红色色调，以微妙地反映原始六色 Apple 标志中的红色。](https://docs-assets.developer.apple.com/published/ca37bff0e0176a587241e4a6ced6090d/components-digit-entry-view-intro%402x.png)

可以在数字行上方添加可选标题和提示。

## [最佳实践](https://developer.apple.com/design/human-interface-guidelines/digit-entry-views#Best-practices)

**使用安全数字字段。** 安全数字字段在屏幕上显示星号而非输入的数字。当应用请求敏感数据时，始终使用安全数字字段。

**清楚说明数字输入视图的目的。** 使用标题和提示解释为什么需要输入数字。

## [平台注意事项](https://developer.apple.com/design/human-interface-guidelines/digit-entry-views#Platform-considerations)

 _iOS、iPadOS、macOS、visionOS 或 watchOS 不支持。_
