# Python 包管理

使用现代打包工具、pyproject.toml 创建、组织和发布 Python 包的完整指南。

## 适用场景

- 创建用于分发的 Python 库
- 构建带 entry points 的命令行工具
- 发布包到 PyPI 或私有仓库
- 搭建 Python 项目结构
- 创建带依赖的可安装包
- 构建 wheel 和 source distribution
- 管理 Python 包的版本和发布
- 创建 namespace packages
- 实现包元数据和 classifiers

## 不适用场景

- 任务与 Python 打包无关
- 需要本范围之外的其他领域或工具
