# Angular

面向 AI 智能体和 LLM 优化的现代 Angular 开发（v20+）综合指南。

## 概述

本技能涵盖现代 Angular 模式，包括：

- **Signals** — Angular 的响应式原语，用于状态管理
- **Standalone Components** — 无需 NgModule 的现代组件架构
- **Zoneless Applications** — 无 Zone.js 的高性能应用
- **SSR & Hydration** — 服务端渲染和客户端水合模式
- **Modern Routing** — 函数式守卫、解析器和懒加载
- **Dependency Injection** — 使用 `inject()` 函数的现代 DI
- **Reactive Forms** — 类型安全的表单处理

## 结构

本技能是一个综合性的 `SKILL.md` 文件，包含：

1. 使用 Signal 输入/输出的现代组件模式
2. 使用 Signals 和计算值的状态管理
3. 性能优化技术
4. SSR 和水合最佳实践
5. 从旧版 Angular 模式迁移的策略

## 使用方式

本技能设计为可完整阅读以理解现代 Angular 开发的完整方法，也可在需要时参考特定模式。
