# 分流标签

技能使用五个标准分流角色来表述。本文件将这些角色映射到本仓库 Issue 跟踪器中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 我们跟踪器中的标签   | 含义                                   |
| -------------------------- | -------------------- | -------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此 Issue                 |
| `needs-info`               | `needs-info`         | 等待报告者补充更多信息                 |
| `ready-for-agent`          | `ready-for-agent`    | 已完全描述，可供 AFK 智能体处理        |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现                           |
| `wontfix`                  | `wontfix`            | 不会处理                               |

当技能提及某个角色时（例如"应用 AFK 就绪分流标签"），使用本表中对应的标签字符串。
