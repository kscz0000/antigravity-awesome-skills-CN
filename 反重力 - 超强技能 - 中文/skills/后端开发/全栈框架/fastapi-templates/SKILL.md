- 从零开始创建新的 FastAPI 项目
- 使用 Python 实现异步 REST API
- 构建高性能 Web 服务和微服务
- 创建使用 PostgreSQL、MongoDB 的异步应用
- 搭建具有规范结构和测试的 API 项目

## 何时不使用此技能
