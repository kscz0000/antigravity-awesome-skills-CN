# CQRS 实现

实现 CQRS（命令查询职责分离）模式的综合指南。

## 使用此技能的场景

- 分离读写关注点
- 独立扩展读取与写入
- 构建事件溯源系统
- 优化复杂查询场景
- 需要不同的读写数据模型
- 需要高性能报表

## 不使用此技能的场景

- 领域简单，CRUD 已足够
- 无法运维独立的读写模型
- 所有地方都需要强即时一致性

## 指令

- 识别读写工作负载和一致性需求。
- 定义具有清晰边界的命令和查询模型。
- 实现读取模型投影和同步机制。
- 验证性能、恢复能力和故障模式。
- 如需详细模式，打开 `resources/implementation-playbook.md`。

## 资源

- `resources/implementation-playbook.md` 提供详细的 CQRS 模式和模板。

## 限制
- 仅当任务明确符合上述描述范围时使用此技能。
- 输出内容不能替代环境特定的验证、测试或专家评审。
- 如缺少必要的输入、权限、安全边界或成功标准，请停止并请求澄清。
