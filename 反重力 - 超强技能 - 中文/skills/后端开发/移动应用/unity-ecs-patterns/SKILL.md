# Unity ECS 模式

Unity 数据导向技术栈 (DOTS) 生产级模式，涵盖实体组件系统、Job 系统和 Burst 编译器。

## 使用场景

- 构建高性能 Unity 游戏
- 高效管理数千个实体
- 实现数据导向的游戏系统
- 优化 CPU 密集型游戏逻辑
- 将 OOP 游戏代码转换为 ECS
- 使用 Jobs 和 Burst 实现并行化

## 不适用场景

- 任务与 Unity ECS 模式无关
- 需要此范围外的其他领域或工具
