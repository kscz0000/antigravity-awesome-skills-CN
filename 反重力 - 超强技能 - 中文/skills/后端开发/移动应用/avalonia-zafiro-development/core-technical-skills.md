# 核心技术技能与架构

## 必备专业知识

开发者必须具备以下方面的扎实专业知识：
- **C# 和现代 .NET**：利用语言和框架的最新特性。
- **Avalonia UI**：用于跨平台 UI 开发。
- **MVVM 架构**：保持 UI 和业务逻辑的严格分离。
- **整洁代码与整洁架构**：关注可维护性和向内依赖流。
- **C# 函数式编程**：拥抱不可变性和函数式模式。
- **响应式编程**：DynamicData 和 System.Reactive 专业知识。

## 架构原则

- **纯 MVVM**：所有 UI 代码强制要求。逻辑必须独立于 UI 关注点。
- **组合优于继承**：优先使用模块化构建块而非深层继承层次结构。
- **向内依赖流**：抽象不能依赖于实现。
- **不可变性**：在可行的情况下优先使用不可变结构以确保可预测性。
- **稳定的公共 API**：精心设计 API 以确保长期稳定性和清晰性。
