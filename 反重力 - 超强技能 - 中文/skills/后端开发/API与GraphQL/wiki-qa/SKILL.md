# Wiki Q&A

完全基于源代码证据回答仓库问题。

## 何时使用
- 用户询问代码库相关问题
- 用户想了解特定文件、函数或组件
- 用户问"X如何工作"或"Y在哪里定义"

## 流程

1. 检测问题的语言；使用相同语言回答
2. 在代码库中搜索相关文件
3. 阅读这些文件以收集证据
4. 综合答案并提供内联引用

## 响应格式

- 使用 `##` 标题、带语言标签的代码块、表格、项目符号列表
- 内联引用来源：`(src/path/file.ts:42)`
- 包含"关键文件"表格，将文件映射到其角色
- 如果信息不足，请说明并建议需要检查的文件

## 规则

- 仅使用来自实际源文件的信息
- 永远不要编造、猜测或使用外部知识
- 回答前逐步思考

## 何时使用
此技能适用于执行概述中描述的工作流或操作。

## 限制
- 仅当任务明确匹配上述范围时使用此技能。
- 不要将输出视为环境特定验证、测试或专家审查的替代品。
- 如果缺少必需的输入、权限、安全边界或成功标准，请停止并请求澄清。
