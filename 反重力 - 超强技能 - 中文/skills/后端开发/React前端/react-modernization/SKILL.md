# React 现代化

精通 React 版本升级、class 到 hooks 迁移、并发特性采用及 codemod 自动化重构。

## 使用场景

- 将 React 应用升级到最新版本
- 将 class 组件迁移为使用 hooks 的函数组件
- 采用并发 React 特性（Suspense、transitions）
- 使用 codemod 进行自动化重构
- 现代化状态管理模式
- 迁移到 TypeScript
- 利用 React 18+ 特性提升性能
