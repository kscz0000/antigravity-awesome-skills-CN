# 章节定义

本文件定义所有章节、排序、影响级别和描述。
章节 ID（括号内）是用于分组规则的文件名前缀。

---

## 1. 消除瀑布流 (async)

**影响：** CRITICAL
**描述：** 瓶布流是头号性能杀手。每个顺序 await 都会增加完整的网络延迟。消除它们能带来最大的收益。

## 2. 包体积优化 (bundle)

**影响：** CRITICAL
**描述：** 减少初始包体积可改善可交互时间（TTI）和最大内容绘制（LCP）。

## 3. 服务端性能 (server)

**影响：** HIGH
**描述：** 优化服务端渲染和数据获取可消除服务端瀑布流并减少响应时间。

## 4. 客户端数据获取 (client)

**影响：** MEDIUM-HIGH
**描述：** 自动去重和高效的数据获取模式可减少冗余的网络请求。

## 5. 重渲染优化 (rerender)

**影响：** MEDIUM
**描述：** 减少不必要的重渲染可最小化无用计算并提升 UI 响应速度。

## 6. 渲染性能 (rendering)

**影响：** MEDIUM
**描述：** 优化渲染过程可减少浏览器需要执行的工作量。

## 7. JavaScript 性能 (js)

**影响：** LOW-MEDIUM
**描述：** 对热路径的微优化可以累积成有意义的改进。

## 8. 高级模式 (advanced)

**影响：** LOW
**描述：** 针对特定场景的高级模式，需要谨慎实现。
