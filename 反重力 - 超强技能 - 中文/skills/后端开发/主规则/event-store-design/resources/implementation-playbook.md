# 事件存储设计操作手册

## Schema 与流策略

- 使用仅追加写入，配合乐观并发控制。
- 同时维护流内排序和全局排序索引。
- 包含元数据字段以记录 causation 和 correlation ID。

## 运维安全护栏

- 严禁在生产环境中修改历史事件。
- 通过明确的 upcasters/downcasters 策略对事件 schema 进行版本管理。
- 按流类型定义保留与归档策略。

## 订阅与投影安全

- 跟踪每个订阅者的检查点位置。
- 使处理器具备幂等性且可安全重放。
- 支持从干净检查点重建投影。

## 性能检查清单

- 为 stream id + version 建立索引。
- 为 global position 建立索引。
- 为长生命周期 aggregate 添加快照策略。
