# 现代 JavaScript 模式

全面掌握现代 JavaScript (ES6+) 特性、函数式编程模式及编写整洁、可维护、高性能代码的最佳实践。

## 使用此技能的场景

- 将旧版 JavaScript 重构为现代语法
- 实现函数式编程模式
- 优化 JavaScript 性能
- 编写可维护且可读的代码
- 处理异步操作
- 构建现代 Web 应用
- 从回调迁移到 Promise/async-await
- 实现数据转换管道

## 不使用此技能的场景

- 任务与现代 JavaScript 模式无关
- 需要此范围之外的其他领域或工具
