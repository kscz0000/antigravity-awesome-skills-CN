# UV 包管理器

使用 uv 的全面指南——一个用 Rust 编写的极速 Python 包安装器和解析器，用于现代 Python 项目管理和依赖工作流。

## 适用场景

- 快速搭建新的 Python 项目
- 比 pip 更高效地管理 Python 依赖
- 创建和管理虚拟环境
- 安装 Python 解释器
- 高效解决依赖冲突
- 从 pip/pip-tools/poetry 迁移
- 加速 CI/CD 流水线
- 管理 monorepo Python 项目
- 使用 lockfile 实现可复现构建
- 优化含 Python 依赖的 Docker 构建

## 不适用场景

- 任务与 uv 包管理器无关
- 需要使用本技能范围之外的其他领域或工具
