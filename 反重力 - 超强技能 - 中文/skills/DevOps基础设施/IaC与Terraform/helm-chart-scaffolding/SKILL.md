# Helm Chart 脚手架

创建、组织和管理 Helm Chart 以打包和部署 Kubernetes 应用的全面指南。

## 使用场景

以下情况使用此技能：
- 从零创建新的 Helm Chart
- 打包 Kubernetes 应用用于分发
- 使用 Helm 管理多环境部署
- 为可复用的 Kubernetes 清单实现模板化
- 设置 Helm Chart 仓库
- 遵循 Helm 最佳实践和约定

## 不适用场景

- 任务与 Helm Chart 脚手架无关
- 需要此范围之外的其他领域或工具

## 操作指引

- 明确目标、约束和所需输入
- 应用相关最佳实践并验证结果
- 提供可执行步骤和验证方法
- 如需详细示例，打开 `resources/implementation-playbook.md`

## 资源文件

- `resources/implementation-playbook.md` 提供详细模式和示例
