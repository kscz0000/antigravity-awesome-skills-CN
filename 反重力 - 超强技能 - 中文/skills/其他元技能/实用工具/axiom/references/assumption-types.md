# 假设分类手册

Axiom 阶段3使用的四种假设分类系统的详细参考。本手册提供识别方法、质疑策略、真实示例和边界情况指南。

---

## 为什么分类本身就是洞见

大多数第一性原理工具跳过分类，直接"质疑一切"。但**不同类型的假设需要根本不同的质疑策略**：

- 质疑**物理事实**浪费时间和信誉
- 质疑**惯例**需要检查环境是否已改变
- 质疑**信念**需要找到塑造它的经验来源
- 质疑**利益驱动**假设需要追踪金钱链

**最大的"顿悟"时刻**往往来自某人发现他们当成不可改变事实的东西其实只是惯例——甚至可能是利益驱动的叙事。

---

## 四种假设类型详解

---

### 🔵 类型1：物理事实

**定义：**
自然规律、数学定理、已验证的科学常数、逻辑恒等式。不能被人类意志、政策或信念改变。

**识别问题：**
- 你能在地球上找到一个真正的反例吗？
- 这在所有文化、时间、语境下都成立吗？
- 如果人类消失了，这还成立吗？
- 这是否基于可重复、可测量的证据？

**质疑策略：**
接受它。不要浪费精力。如果确实是物理事实，正确的做法是在它的约束内工作，而不是对抗它。

**示例：**

| ✅ 真正的物理事实 | ❌ 常被误分类为事实 |
|------------------|-------------------|
| 引力存在 (9.8 m/s²) | "房价永远涨"（惯例） |
| 熵随时间增加 | "你需要8小时睡眠"（惯例——因人而异） |
| 光速恒定 | "人是理性的经济主体"（信念） |
| 复利公式 | "创新需要颠覆"（利益驱动） |

**常见误分类陷阱：**
人们经常把**社会模式**或**统计趋势**分类为物理事实。"经济总会复苏"是历史规律，不是物理定律。"人抗拒改变"是一种倾向，不是常数。

---

### 🟡 类型2：历史惯例

**定义：**
曾经有效且被广泛采用的做法、规则或信念，但起源于特定语境。语境可能已经改变，使惯例过时或可选。

**识别问题：**
- 这个规则/做法是什么时候起源的？那时的环境是什么样的？
- 底层环境（技术、人口、市场、监管）变了吗？
- 这是普遍的，还是特定于某个地区/行业/时代？
- 谁因为这个惯例的延续而获益？
- 遵循这个惯例的人中，多少比例真正得到了承诺的结果？

**质疑策略：**
检查时间戳。问："这在20年前成立吗？现在还成立吗？10年后呢？"关键洞见：惯例有巨大的惯性——人们在原始理由消失后很久仍然遵循它们。

**示例：**

| 惯例 | 原始语境 | 已改变的语境 |
|------|----------|--------------|
| 大学学历是好工作的门票 | 1990s: 大学生稀缺 | 2020s: 本科普及，技能导向增强 |
| 你必须在办公室才能高效 | 2020年前: 远程工具有限 | 疫情后: 远程基础设施成熟 |
| 在一家公司待5-10年才算忠诚 | 终身雇佣时代 | 平均任期现在只有2-3年 |
| 房地产是中产阶级最好的投资 | 1980-2010: 人口增长+城市化 | 2020s+: 许多国家人口下降 |
| 创业要写完整的商业计划书 | 精益创业方法论之前 | 现在: MVP → 迭代 → 转型是标准 |
| 学STEM专业就业稳定 | AI时代之前 | AI现在也在颠覆STEM岗位 |

**跨文化注意：**
有些惯例是文化特有的：
- 🇨🇳 "35岁是职业分水岭" — 这是中国劳动力市场惯例，西方几乎不知道
- 🇺🇸 "你应该每2-3年换一次工作" — 这是美国科技行业惯例，在日本或德国不常见
- 🇨🇳 "有房才能结婚" — 中国婚姻惯例，不是全球现实
- 🇺🇸 "追随你的热情" — 美国文化叙事，在集体主义文化中较少见

### 🔴 类型3：主观信念

**定义：**
从个人经验中得出的结论，被投射为普遍真理。当事人真诚地相信它们，但它们是由有限的经验样本、认知偏差和情感联想塑造的。

**识别问题：**
- 谁告诉你的？
- 你亲自验证过吗，还是从小样本推断？
- 有不同经历的人能得出相反结论吗？
- 这个信念让你舒服吗？它是否保护你免受更难以面对的真相？
- 如果发现这个信念是错的，你必须做出什么改变？

**质疑策略：**
寻找反面证据。找到做了相反事情却成功的人，或遵循这个信念却失败的人。目标不是推翻信念，而是检验它的普遍性。

**示例：**

| 主观信念 | 实际是什么 |
|----------|-----------|
| "我不适合做管理" | 一次糟糕的管理经验 → 泛化为身份认同 |
| "我不是学数学的料" | 早期负面经验 → 自我限制的身份认同 |
| "创业太冒险了" | 听到失败故事 → 可得性偏差 |
| "我这个行业不看重X" | 基于一家公司的文化，不是整个行业 |
| "我的性格不适合社交" | 内向 ≠ 不能建立关系 |
| "好开发者不需要擅长沟通" | 保护自己免受不适的信念 |

**危险信号 — 伪装成事实的信念：**
如果有人说"就是这样的"或"大家都知道"，但说的不是物理定律，那几乎肯定是信念或惯例被提升成了"事实"地位。

### ⚫ 类型4：利益驱动

**定义：**
某些人（行业、公司、意见领袖或社会群体）积极传播的假设，因为他们从你的相信中获益。这并不意味着这些说法总是假的——但它们被传播的原因不是因为真，而是因为有利可图。

**识别问题：**
- 谁在推广这个观点？他们卖什么？
- 这个信息来源是否有经济动机让你相信？
- 如果没有人从中获利，这个观点还会流行吗？
- 这是否被包装成紧迫的？紧迫感通常服务于卖方。
- 零假设是什么——如果你什么都不做会怎样？

**质疑策略：**
追踪金钱链。从这个观点追溯到获益者。这并不自动推翻这个说法，但它改变了你应该给予它的权重。

**示例：**

| 利益驱动假设 | 谁获益 |
|-------------|--------|
| "你需要买保险来保护家人" | 保险公司和保险经纪人 |
| "AI将在5年内取代所有工作" | 正在融资的AI公司 |
| "创业是实现财务自由的唯一方式" | 创业培训课程、孵化器 |
| "你需要学编程" (2015-2023) | 编程训练营、科技公司 |
| "房价永远涨" | 房地产开发商、中介、银行 |
| "你需要这个认证才有竞争力" | 认证提供商 |
| "知识付费是最好的自我投资" | 知识付费平台和课程制作者 |
| "增长黑客对创业公司至关重要" | 增长咨询公司、SaaS工具 |

**跨文化注意：**
- 🇨🇳 中国互联网充斥着利益驱动假设：知识付费平台、房产中介、保险销售、创业导师、消费主义KOL
- 🇺🇸 美国媒体充斥着：VC支持的创业神话、大学推动的学历主义、广告推动的生活方式通胀

## 处理混合类型

许多现实世界的假设是**混合体**。遇到时：

1. **识别主要类型** — 哪种类型最能描述核心机制？
2. **标记次要类型** — 还有什么其他动态在起作用？
3. **先应用主要质疑策略**，再用次要策略补充

**混合类型示例：**

| 假设 | 主要 | 次要 | 原因 |
|------|------|------|------|
| "现在是创业的好时机" | 🟡 惯例 | ⚫ 利益驱动 | "好时机"叙事是惯例，但创业媒体为了点击和融资放大它 |
| "大学学位会回本" | 🟡 惯例 | ⚫ 利益驱动 | 统计上曾经成立，但大学有动机无论真假都推广 |
| "房价只涨不跌" | 🔴 信念 | 🟡 惯例 | 来自特定时代的个人经验，被全行业惯例强化 |
| "远程工作者效率更低" | 🔴 信念 | 🟡 惯例 | 管理者的个人偏见，被传统管理惯例强化 |

## 边界案例指南

### 统计趋势 vs 事实
"男性平均比女性高" — 这是**统计事实**（可测量、可重复），但应用于任何个体就是**主观信念**。根据用户使用它的方式分类。

### 过时的事实
"冥王星是行星" — 曾经是物理事实，后来被重新分类。如果某人的假设基于过时事实，把它当作尚未更新的**惯例**。

### 自证预言
"股市会崩盘" — 如果足够多的人相信，他们会卖出，然后真的崩盘。如果来源从预测中获益，分类为**利益驱动**；如果是用户的个人焦虑，分类为**主观信念**。

### 文化规范
"你应该30岁之前结婚" — 这是**惯例**（文化特有、有历史基础），可能有**利益驱动**元素（家庭压力、婚庆行业）。

```
🔴 主观信念  → 谁告诉你的？          → 寻找反面证据
⚫ 利益驱动  → 谁因此获利？          → 追踪利益链
```
