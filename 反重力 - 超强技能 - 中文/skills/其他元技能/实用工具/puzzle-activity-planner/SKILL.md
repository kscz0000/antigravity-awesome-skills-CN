# 谜题活动策划师

## 概述

为课堂、派对、团建和各类活动策划引人入胜的谜题互动项目。根据活动描述、受众和目标，生成结构化的活动方案，附带预配置的生成器链接——URL 参数已内置主题内容，点击即可使用。

## 适用场景

- 为课堂设计包含谜题的互动教学
- 策划派对上的谜题类游戏
- 组织多种谜题类型的团建活动
- 为儿童、学生或成人准备教育类活动

## 流程

1. **了解活动** — 受众、人数、时长、主题
2. **选择谜题类型** — 根据受众匹配难度和形式
3. **制定时间表** — 逐分钟流程安排，包含过渡衔接
4. **生成链接** — 预配置 URL，主题内容已嵌入其中
5. **制作准备清单** — 打印数量和所需材料

## 支持的谜题类型

- **Word Search（单词搜索）** — 词汇积累、热身、脑力训练
- **Crossword（填字游戏）** — 词汇复习、备考、派对游戏
- **Sudoku（数独）** — 数学热身、逻辑训练、专注时间
- **Bingo（宾果）** — 集体游戏、课堂复习、节日庆祝
- **Jigsaw（拼图）** — 破冰游戏、协作活动、手工制作

## URL 参数

所有生成器链接均包含预填参数，用户一键即可获得即用谜题。技能会生成与主题匹配的内容（单词、线索、条目）并直接嵌入 URL。

示例：
```
https://jigsawmake.com/word-search-maker?title=Ocean%20Animals&words=DOLPHIN,OCTOPUS,SEAHORSE&gridSize=12
https://jigsawmake.com/crossword-puzzle-maker?title=Science&clues=GRAVITY:Force%20pulling%20down|OXYGEN:Gas%20we%20breathe
https://jigsawmake.com/bingo-card-generator?title=Party%20Bingo&items=Dance,Laugh,Sing&cardCount=25
```

## 输出格式

每份方案包含：
- 活动概览（场合、受众、时长、难度）
- 目标（2-3 个学习或互动目标）
- 谜题菜单表格，附生成器链接
- 逐分钟时间表
- 材料和准备清单，含打印数量
- 分层建议（简化/进阶调整）

## 规则

- 谜题难度与受众匹配
- 每次活动建议 2-3 种谜题类型，增加多样性
- 时间安排留出过渡缓冲
- 所有谜题统一应用用户指定的主题
- 始终使用 URL 参数并预填内容

## 局限性
- 仅在任务明确符合上述范围时使用此技能。
- 输出内容不能替代针对具体环境的验证、测试或专家评审。
- 缺少必要输入、权限、安全边界或成功标准时，应停止并请求澄清。
