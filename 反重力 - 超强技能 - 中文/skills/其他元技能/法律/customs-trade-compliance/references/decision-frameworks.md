# 决策框架 — 海关与贸易合规

本参考资料提供详细的决策逻辑、归类方法、FTA 资格决策树、海关估价方法、受限方筛查协议以及海关和贸易合规运营的处罚风险评估框架。

所有阈值、时限和监管引用反映涵盖美国、欧盟、英国和主要亚太市场的多司法管辖区贸易运营。监管引用截至 2024 年有效，但在具体程序中依赖前必须核对最新修订。

## 1. HS 归类方法论

### 1.1 归类前信息收集

在应用 GRI 之前，汇编产品的完整技术档案。归类错误几乎总是追溯到产品信息不充分，而非规则误用。

**必要信息清单：**

| 数据点 | 重要性原因 | 示例影响 |
|---|---|---|
| 材料成分（按重量百分比） | 决定纺织、塑料、金属、复合材料的章 | 52% 棉 / 48% 涤纶的面料归为棉（第 52 章）而非合成纤维（第 54 章） |
| 主要功能 | 决定机器、装置、仪器的品目 | 同时称重和贴标的设备按赋予本质特征的功能归类 |
| 尺寸和重量 | 决定许多产品的子目 | 外径在 406.4mm 以上/以下的钢管归类不同 |
| 制造工艺 | 决定产品是否"已制备"、"已加工"或为原料 | 生咖啡（第 9 章）vs 烘焙咖啡（第 9 章）vs 咖啡提取物（第 21 章） |
| 预期用途 | 与"用于"或"适用于"品目相关 | 汽车玻璃 vs 建筑玻璃——相同产品，不同归类 |
| 零售包装 | 决定 GRI 3(b) 套装规则是否适用 | 零售包装中的急救包归为套装；相同物品散装运输则单独归类 |
| 制造国 | 影响国家关税行和 AD/CVD 适用性 | 中国产钢面临韩国产钢不需要的额外关税 |
| 技术规格 | 解决子目级别的区分 | 电压、频率、容量、分辨率、精度——均产生子目划分 |

**常见信息缺失：**

- 依赖供应商建议的 HS 编码而未独立验证——供应商为出口目的归类；进口归类在国家级别可能不同
- 使用产品营销名称而非技术描述——"SmartWidget Pro"对归类毫无帮助
- 接受"……的零件"归类而未验证该项目是否符合第十六类注释 2 的零件测试
- 基于产品行业而非其物理特征归类——汽车连接器和工业连接器可能有相同 HS 编码

### 1.2 GRI 应用决策树

```
起点：待归类产品
  │
  ├─ 步骤 1：阅读所有候选品目文本
  │   ├─ 阅读每个候选类的类注释
  │   ├─ 阅读每个候选章的章注释
  │   └─ 类/章注释是否明确包含或排除该产品？
  │       ├─ 是，仅一个品目明确涵盖 → 归类（GRI 1）。停止。
  │       └─ 否，无唯一品目或注释模糊 → 进入步骤 2
  │
  ├─ 步骤 2：产品是否不完整、未成品或未组装？
  │   ├─ 是 → 是否具有完整品的本质特征？
  │   │   ├─ 是 → 按完整品归类（GRI 2(a)）。停止。
  │   │   └─ 否 → 可能为"零件"——检查类/章注释的零件条款
  │   └─ 否 → 是否为材料/部件的混合物或组合？
  │       ├─ 是 → GRI 2(b) 将品目扩展至包括混合物。多个品目
  │       │         可能适用 → 进入步骤 3
  │       └─ 否 → 进入步骤 3
  │
  ├─ 步骤 3：仍有多个品目适用（表面上可归入 2 个或更多）
  │   ├─ 步骤 3(a)：是否一个品目比其他更具体？
  │   │   ├─ 是 → 归入最具体品目。停止。
  │   │   └─ 否（品目同等具体，或产品为组合/套装）→
  │   │
  │   ├─ 步骤 3(b)：产品是否为组合货品、为零售销售的套装
  │   │              或为零售销售成套装运的货物？
  │   │   ├─ 是 → 按赋予本质特征的材料或部件归类
  │   │   │   ├─ 本质特征可确定 → 归类。停止。
  │   │   │   └─ 本质特征不可确定 → 进入 3(c)
  │   │   └─ 否 → 进入 3(c)
  │   │
  │   └─ 步骤 3(c)：归入按数字顺序最后出现的品目。停止。
  │
  ├─ 步骤 4：经步骤 1-3 后仍无品目涵盖该产品
  │   └─ 归入最相似货物的品目（GRI 4）。停止。
  │
  ├─ 步骤 5：（适用于箱、容器和包装材料）
  │   ├─ GRI 5(a)：为特定物品专门设计的箱、盒及类似容器与该物品一起
  │   │            呈报时与该物品一起归类
  │   └─ GRI 5(b)：包装材料与货物一起归类，除非明显适合重复使用
  │
  └─ 步骤 6：子目级别归类
      ├─ 在已确定的品目内应用 GRI 1-5
      ├─ 在相同级别比较子目文本（一个破折号对一个破折号，两个破折号对两个破折号）
      └─ 子目注释在此级别优先。归类。停止。
```

### 1.3 本质特征确定（GRI 3(b)）

当组合货品或套装必须按赋予本质特征的部件归类时，评估以下因素。没有单一因素是决定性的——结合上下文权衡：

| 因素 | 决定性场景 | 示例 |
|---|---|---|
| **功能** | 一个部件定义产品的功能 | 带内置收音机的手电筒——若主要购买目的是照明，则手电筒部件赋予本质特征 |
| **价值** | 一个部件占主导成本 | 含 95 美元手表和 5 美元小袋的礼品套装——手表按价值赋予本质特征 |
| **体积/重量** | 一个部件在物理上占主导 | 混凝土占重量 90% 的混凝土-钢复合材料 |
| **使用中的角色** | 一个部件不可或缺 | 含墨盒和清洁布的打印机墨盒套件——墨盒不可或缺，布为辅助 |
| **消费者认知** | 产品为一个部件营销 | 含糖果的玩具——若作为玩具营销（糖果为附带），玩具赋予本质特征 |

**当本质特征无法确定时：** 若因素冲突或平衡（例如两个部件的价值、体积和功能重要性大致相等），则本质特征为"不可确定"，你必须进入 GRI 3(c)——按数字顺序最后的品目。

### 1.4 第十六类特殊规则（机器和机械器具）

第十六类（第 84-85 章）有覆盖一般 GRI 应用的复杂注释：

**注释 1——排除：** 通用零件（第十五类）、带和软管（第 39/40/59 章）以及其他具体排除的物品不归入第十六类，无论其与机器的用途如何。

**注释 2——零件归类层次：**
1. 属于第 84 或 85 章任何品目的零件（8409、8431、8448、8466、8473、8503、8522、8529、8538 除外）按其**自身品目**归类，而非作为零件。
2. 其他零件，如果仅或主要用于特定机器，则与该机器一起归类。
3. 可用于多台机器的零件归入"兜底"零件品目（8409、8431 等）或品目 8487（其他机器零件）。

**注释 3——组合机器：** 执行两项或多项互补功能的机器按代表**主要功能**的品目归类。若无法识别主要功能，注释 3 遵从 GRI 3(c)。

**注释 4——机器单元：** 由独立部件组成、设计为共同执行明确定义功能的机器，当部件一起呈报或设计为组装时，作为完整机器归类。

**注释 5——自动数据处理（ADP）机器：** 品目 8471 要求机器能够：(a) 存储处理程序，(b) 由用户自由编程，(c) 执行算术计算，(d) 无需物理干预即可执行用户修改的处理程序。必须满足全部四项标准。单功能设备（例如仅扫描的条码扫描器）不满足标准 (b)，按其功能特定品目归类，而非 ADP。

### 1.5 常见归类争议及解决

**争议：是"零件"还是"附件"？**
- 零件对机器功能至关重要，并在使用中被消耗或与机器物理集成。没有该零件，机器无法运行。
- 附件增强或补充机器功能，但非必需。没有附件，机器仍能运行。
- 归类影响：零件通常跟随机器的归类。附件可按其自身材料或功能品目独立归类，通常适用不同税率。

**争议：食品是"已制备"（第 16/19/20/21 章）还是原料（第 2-14 章）？**
- "已制备"通常指超出保存或运输所需的加工。
  冷冻生虾：第 3 章。熟虾：第 16 章。虾天妇罗（裹面糊油炸）：第 16 章。
- 不构成"制备"的简单操作：清洗、修剪、冷冻、冷藏、按大小分拣、包装。这些使产品保持在第 2-14 章。
- 注意章注释——第 20 章注释 1 将"用醋制备或保存的"蔬菜（品目 2001 是具体规定）从一般第 7 章蔬菜品目中排除。

**争议：是否为 GRI 3(b) 项下的"套装"？**
"为零售销售的套装"必须满足**全部**三个条件：
1. 由至少两个可归入不同品目的不同物品组成
2. 物品装在一起以满足特定需要或进行特定活动
3. 以适合直接销售给用户而无需重新包装的方式装运

若任何条件不满足，物品单独归类——不作为套装。工业组合（例如一盒混合紧固件）若非"为零售销售装运"，则不符合条件。

## 2. FTA 资格决策树

### 2.1 一般 FTA 资格流程

```
起点：该产品是否符合优惠待遇条件？
  │
  ├─ 步骤 1：识别所有潜在适用的 FTA
  │   ├─ 货物在哪里生产 / 最后实质性加工？
  │   ├─ 货物进口到哪里？
  │   └─ 两国是否都是一个或多个 FTA 的缔约方？
  │       ├─ 是 → 列出所有适用 FTA → 进入步骤 2
  │       └─ 否 → 无 FTA 优惠可用。检查 GSP 或其他单边项目。
  │
  ├─ 步骤 2：对每个适用 FTA，确定产品特定原产地规则
  │   ├─ 在 FTA 的产品特定规则附件中查找 HS 品目（4 位或 6 位）
  │   ├─ 确定规则类型：
  │   │   ├─ 关税转移（关税归类变化——CTC、CTH、CTSH）
  │   │   ├─ 区域价值含量（RVC）阈值
  │   │   ├─ 特定加工要求
  │   │   └─ 上述的组合
  │   └─ 进入步骤 3
  │
  ├─ 步骤 3：追溯物料清单（BOM）中的所有材料
  │   ├─ 对每种投入材料，确定：
  │   │   ├─ 投入的 HS 归类
  │   │   ├─ 投入的原产国
  │   │   ├─ 投入的价值（用于 RVC 计算）
  │   │   └─ 投入在同一 FTA 下是否"原产"（用于累积）
  │   └─ 进入步骤 4
  │
  ├─ 步骤 4：应用产品特定规则
  │   ├─ 关税转移：每种非原产材料是否经历了所需的关税转移？
  │   │   ├─ CTC（关税章变化）：所有非原产材料必须来自与成品
  │   │   │   不同的 2 位章
  │   │   ├─ CTH（关税品目变化）：不同的 4 位品目
  │   │   ├─ CTSH（关税子目变化）：不同的 6 位子目
  │   │   └─ 检查例外——许多规则列出从关税转移中**排除**的特定品目
  │   │       （例如，"从任何品目改变，品目 7208-7212 除外"）
  │   ├─ RVC：区域价值含量是否达到或超过阈值？
  │   │   ├─ 使用允许的方法计算
  │   │   └─ 若 FTA 提供多种方法，使用最有利的
  │   └─ 加工：所需制造工艺是否在该地区执行？
  │
  ├─ 步骤 5：应用累积（如适用）
  │   ├─ 在其他 FTA 伙伴国原产的材料可被视为原产
  │   ├─ 双边累积：仅在两方之间（EU-UK TCA）
  │   ├─ 对角累积：在所有缔约方之间（RCEP、PEM 公约）
  │   └─ 完全累积：即使在伙伴国的加工（不仅是原产材料）
  │       也计入原产（欧盟海外国家和领土）
  │
  ├─ 步骤 6：应用最低门槛 / 容差规则
  │   ├─ 大多数 FTA 允许不符合关税转移规则的非原产材料占
  │   │   较小百分比（通常为产品价值的 7-10%）
  │   ├─ 最低门槛**不适用**于产品特定规则中具体排除的材料
  │   └─ 纺织品有单独的最低门槛规则（通常按重量而非价值）
  │
  └─ 步骤 7：验证认证和文件要求
      ├─ USMCA：自认证，包含九项必要数据要素
      ├─ EU-UK TCA：发票上的原产地声明（低于 €6,000 的托运任何出口商；
      │   超过 €6,000 需要 REX 注册）
      ├─ RCEP：原产地证书（RCEP 表格）或原产地声明
      └─ 保留所有支持文件至 FTA 规定的保留期
```

### 2.2 USMCA 原产地规则——详细应用

**关税转移规则：**

USMCA 附件 4-B 包含每个 HS 品目的产品特定规则。许多规则将关税转移要求与 RVC 阈值相结合。常见模式：

- **纯关税转移：**"品目 9403 项下从任何其他章的改变。"所有非原产材料必须来自第 94 章（家具）以外的章。若使用来自第 44 章的非原产木材制作木桌，木材满足此规则，因为第 44 章 ≠ 第 94 章。但若进口已归入品目 9403 的非原产桌腿，关税转移失败。

- **带例外的关税转移：**"品目 6204 项下从品目 6201-6212 组以外的任何品目的改变，但品目 5106-5113 或 5204-5212 除外……"这些排除条款旨在防止简单装配操作获得原产资格。

- **关税转移或 RVC：**"品目 8471 项下从任何其他品目的改变；或者在关税归类中无要求的改变，但前提是按成交价格法区域价值含量不低于 45% 或按净成本法不低于 35%。"出口商可选择更易满足的替代方案。

**区域价值含量计算：**

*成交价格法：*
```
RVC = ((TV - VNM) / TV) × 100
```
其中：
- TV = 货物的成交价格（调整为 FOB）
- VNM = 非原产材料的价值（包括零件、生产中消耗的材料）

*净成本法：*
```
RVC = ((NC - VNM) / NC) × 100
```
其中：
- NC = 总成本减去销售推广、营销、售后服务、特许权使用费、运输和包装成本以及不允许的利息
- VNM = 同上

净成本法通常产生**更高的** RVC，因为它从分母中剔除膨胀成本。当利润率较低或存在重大特许权使用费/推广成本时使用。

**汽车规则（USMCA 特有）：**

USMCA 引入了有史以来最复杂的汽车原产地规则：
- 乘用车：使用净成本法的 75% RVC（从 62.5% 阶段性提高）
- 核心部件（发动机、变速箱、车身/底盘）：单独的 75% RVC 要求
- 主要部件（制动器、车轴、悬挂）：70% RVC
- 补充部件（空调、转向、电池）：70% RVC
- 钢铝：70% 必须在北美"冶炼并浇铸"
- 劳动价值含量：40% 的生产必须由工资 ≥ 16 美元/小时（高薪）的工人完成

### 2.3 EU-UK TCA 原产地规则

**与 USMCA 的主要差异：**

- 高价值托运无 REX 注册不进行自认证
- 仅双边累积（英国 + 欧盟，除非存在单独协议，否则不与第三国累积）
- "加工不足"清单：无论关税转移如何，**永远不**授予原产资格的操作（例如，简单装配、包装、混合、纺织品熨烫、锐化/研磨）
- 容差：不符合清单规则的非原产材料占出厂价的 10%（第 50-63 章产品，纺织品为 15%——但按重量而非价值计算）

**清单规则结构：**

EU-UK TCA 附件 ORIG-2 按四列提供规则：
1. 成品的 HS 品目
2. 产品描述
3. 授予原产资格的对非原产材料的加工或处理
4. 替代规则（如可用）

示例：品目 8418（冰箱）
- 规则："制造中所使用的所有材料归入产品品目以外的品目"且"所使用的所有非原产材料的价值不超过产品出厂价的 40%"
- 必须同时满足两个条件（累积，非替代）

### 2.4 RCEP 原产地规则

**主要特点：**

- 15 个成员国，具有对角累积
- 产品特定规则同时使用关税转移和 RVC（40% 阈值为典型）
- RVC 可使用以下任一方法计算：
  - 累加法：RVC = (VOM / FOB) × 100 ≥ 40%
  - 累减法：RVC = ((FOB - VNM) / FOB) × 100 ≥ 40%
- 原产地证书（RCEP 表格）或经批准的出口商原产地声明
- 允许"背靠背"证书，用于通过非缔约方转运的货物（对新加坡/香港枢纽运营至关重要）
- 第 50-63 章（纺织品）的产品特定规则明显更具限制性——许多要求两步加工（纱线 → 面料 → 服装）

### 2.5 AfCFTA 原产地规则（摘要）

- 一般规则：40% 的增加值门槛（或关税品目变化）
- 累积：高达 60% 的材料可来自其他 AfCFTA 成员国
- 最不发达国家成员国有简化规则
- 许多关税行的产品特定规则仍在谈判中——在依赖一般规则之前，查阅最新的原产地规则议定书附件

## 3. 海关估价方法

### 3.1 方法选择决策树

```
起点：确定进口货物的海关价值
  │
  ├─ 方法 1：成交价格
  │   ├─ 是否存在出口到进口国的销售？
  │   │   ├─ 否（寄售、借用、免费样品、礼品、租赁）→ 跳至方法 2
  │   │   └─ 是 → 销售是否在关联方之间进行？
  │   │       ├─ 否 → 销售是否受无法量化的条件约束？
  │   │       │   ├─ 否 → 是否有处置/使用限制（法律或地理除外）？
  │   │       │   │   ├─ 否 → 成交价格 = 实付或应付价格 + 增加项 - 扣除项
  │   │       │   │   │   ├─ 增加项（19 CFR § 152.103 / UCC 第 71 条）：
  │   │       │   │   │   │   ├─ 佣金（采购佣金除外）
  │   │       │   │   │   │   ├─ 协助（免费/降低成本提供的工具、模具、模型、工程）
  │   │       │   │   │   │   ├─ 与进口货物相关且作为销售条件的特许权使用费和许可费
  │   │       │   │   │   │   ├─ 包装成本
  │   │       │   │   │   │   ├─ 卖方获得的随后转售收益
  │   │       │   │   │   │   └─ 美国：到美国港口的运输成本（海运/空运、保险）
  │   │       │   │   │   │       欧盟：到欧盟边境的运输和保险**包含在内**
  │   │       │   │   │   │       （欧盟为 CIF 基础，美国为 FOB+）
  │   │       │   │   │   └─ 扣除项（不计入成交价格）：
  │   │       │   │   │       ├─ 进口后运输成本（内陆运费）
  │   │       │   │   │       ├─ 进口关税和税款
  │   │       │   │   │       ├─ 进口后建设/安装成本
  │   │       │   │   │       └─ 采购佣金（如明确识别）
  │   │       │   │   └─ 确定价值。停止。
  │   │       │   └─ 是 → 跳至方法 2
  │   │       └─ 是（关联方）→
  │   │           ├─ 该关系是否影响价格？
  │   │           │   ├─ 否（以"销售情况"测试或测试值证明）→
  │   │           │   │   成交价格可接受。应用上述增加/扣除项。
  │   │           │   └─ 是或无法证明 → 跳至方法 2
  │   │           └─ 或：成交价格是否接近"测试值"？
  │   │               （相同/类似货物销售给非关联方的成交价格、
  │   │                相同/类似货物的倒扣价格或计算价格）
  │   │               ├─ 是 → 成交价格可接受。停止。
  │   │               └─ 否 → 跳至方法 2
  │
  ├─ 方法 2：相同货物成交价格
  │   ├─ 是否有**相同**货物的进口？
  │   │   （所有方面相同：物理特征、质量、声誉、原产国）
  │   ├─ 以相同商业水平和大致相同数量出口销售？
  │   │   （允许对数量差异和运输成本进行调整）
  │   ├─ 与被估价货物大致同时销售？
  │   │   ├─ 全部为是 → 使用这些相同货物的成交价格。停止。
  │   │   └─ 任一为否 → 跳至方法 3
  │
  ├─ 方法 3：类似货物成交价格
  │   ├─ 与方法 2 相同标准，但针对"类似"货物——
  │   │   在特征、部件、材料方面密切相似，
  │   │   能够执行相同功能并可商业互换
  │   │   ├─ 可用 → 使用。停止。
  │   │   └─ 不可用 → 跳至方法 4（在美国应进口商要求跳至方法 5）
  │
  ├─ 方法 4：倒扣价格
  │   ├─ 以在进口国销售最大总量单位的单价开始
  │   │   （在进口日期后 90 天内）
  │   ├─ 扣除：
  │   │   ├─ 进口国相同类别或种类货物销售中通常获得的
  │   │   │   佣金或利润及一般费用
  │   │   ├─ 进口国内的运输和保险成本
  │   │   ├─ 进口应付的关税和其他国家税款
  │   │   └─ 若货物进口后进一步加工（"超级倒扣"）：
  │   │       扣除加工增加的价值
  │   └─ 结果 = 倒扣价格。停止。
  │
  ├─ 方法 5：计算价格
  │   ├─ 从以下**累加**：
  │   │   ├─ 生产国材料和制造/加工的成本或价值
  │   │   ├─ 利润和一般费用金额，等于出口国相同类别或种类
  │   │   │   货物销售中通常反映的金额
  │   │   └─ 到进口国的运输和保险成本（欧盟为 CIF，美国为 FOB+）
  │   ├─ 此方法仅在外国生产商愿意共享成本数据并接受
  │   │   进口国海关当局核查时可用
  │   └─ 结果 = 计算价格。停止。
  │
  └─ 方法 6：后备（"合理手段"）
      ├─ 灵活应用方法 1-5 并进行合理调整
      ├─ 禁止——该价值**不能**基于：
      │   ├─ 进口国生产货物的销售价格
      │   ├─ 提供使用两个替代值中较高者的系统
      │   ├─ 出口国国内市场货物的价格
      │   ├─ 最低海关价值
      │   ├─ 任意或虚构价值
      │   └─ 除相同/类似货物的计算价格外的生产成本
      └─ 结果 = 后备价值。停止。
```

### 3.2 关联方估价——销售情况测试

当买方和卖方关联方（依据 19 USC § 1401a(g) 或 UCC 第 127 条）时，海关当局将审查成交价格。为证明该关系未影响价格：

**支持接受的证据：**

1. 价格以符合行业正常定价实践的方式确定
2. 价格以符合卖方对非关联买方定价方式的方式确定
3. 价格足以确保在代表性期间内销售相同类别或种类货物的所有成本加上等同于公司整体利润的利润得到回收
4. 显示价格按 OECD 指南以公平交易原则确定的转让定价文件

**测试值（销售情况的替代）：**

若成交价格密切接近以下之一，则可接受：
- 进口国销售给非关联方的相同或类似货物的成交价格
- 相同或类似货物的倒扣价格
- 相同或类似货物的计算价格

"密切接近"指在合理范围内——CBP 和欧盟海关在此有自由裁量权，但 5% 以下的差异通常被接受，5-10% 需要解释，超过 10% 可能导致成交价格被拒绝。

### 3.3 协助估价

协助是最常被忽视的成交价格增加项之一。协助是买方为用于生产进口货物而免费或降低成本向卖方提供的任何东西：

**协助类型：**

| 协助类型 | 估价方法 | 分摊 |
|---|---|---|
| 生产中消耗的材料（原材料、部件） | 采购或生产成本 | 全额价值加至第一批货物，或在所有生产单位之间按比例分摊 |
| 工具、模具、模型及类似物品 | 采购或生产成本；如已使用，则仅为当前价值 | 在使用该工具/模具/模型生产的单位数量之间按比例分摊 |
| 工程、开发、艺术品、设计、计划和草图 | 若在进口国进行则为成本；若在别处进行则为价值 | 在预期总生产或第一批货物之间按比例分摊 |
| 由关联方提供的生产中消耗的材料 | 若公平交易则为关联方之间支付的价格；否则为生产成本 | 与非关联方协助相同 |

**关键协助陷阱：** 美国公司免费向中国制造商发送 CAD 图纸。这些图纸的生成价值（工程时间、软件许可）必须加至使用这些图纸的该制造商每次进口的海关价值中。未申报协助是 CBP 重点评估中的主要发现之一，并引发处罚风险。

### 3.4 特许权使用费和许可费

当特许权使用费满足以下条件时，它们是成交价格的应税增加项：
1. 与进口货物相关（与进口后活动无关），且
2. 作为销售条件支付（买方不支付特许权使用费就无法购买货物）

**应税示例：**
- 支付给卖方（或关联方）的特许权使用费，用于使用专利工艺制造的权利——该特许权使用费直接与进口货物的生产相关
- 支付给母公司的商标许可费，制造商将仅为被许可进口商生产带有该商标的货物
- 技术许可费，外国生产商只能通过买方的许可获得该技术

**非应税示例：**
- 在进口国分销或转售权利的特许权使用费（进口后活动）
- 支付给无关第三方的特许权使用费，卖方对特许权使用费安排不知情或不感兴趣，且该特许权使用费不是销售条件

**"销售条件"分析是事实密集型的。** 查看：
- 如果不支付特许权使用费，卖方是否会向买方出售货物？
- 卖方是否参与或控制特许权使用费安排？
- 如果不支付特许权使用费，买方能否从其他来源采购货物？

## 4. 受限方筛查协议

### 4.1 筛查工作流

```
起点：新交易（销售、采购或服务）
  │
  ├─ 步骤 1：识别交易的**所有**方
  │   ├─ 买方 / 进口商备案人
  │   ├─ 卖方 / 出口商
  │   ├─ 最终收货人 / 最终用户
  │   ├─ 中间收货人
  │   ├─ 货运代理和报关行
  │   ├─ 银行（信用证交易中）
  │   ├─ 收货地址（若与买方/收货人不同）
  │   └─ 任何代理人、代表或实益所有人
  │
  ├─ 步骤 2：将所有识别方对照所有适用清单进行筛查
  │   ├─ 美国清单（若存在美国关联——美国原产货物、美国人士、美国金融体系）：
  │   │   ├─ OFAC SDN 清单（特别指定国民和被封锁人员）
  │   │   ├─ OFAC 部门制裁识别（SSI）清单
  │   │   ├─ OFAC 非 SDN 菜单式制裁清单（NS-MBS）
  │   │   ├─ BIS 实体清单（744 部分补充第 4 号）
  │   │   ├─ BIS 被拒绝人员清单
  │   │   ├─ BIS 未核实清单
  │   │   ├─ BIS 军事最终用户（MEU）清单
  │   │   ├─ DDTC 被禁止方（AECA 第 38 条）
  │   │   └─ 防扩散制裁清单
  │   ├─ 欧盟清单（若存在欧盟关联）：
  │   │   ├─ 欧盟综合金融制裁清单
  │   │   └─ 欧盟两用品法规附件 I 受控最终用户
  │   ├─ 英国清单（若存在英国关联）：
  │   │   ├─ 英国 OFSI 综合清单
  │   │   └─ 英国出口管制联合部门
  │   ├─ 其他司法管辖区：澳大利亚 DFAT、加拿大 SEMA、日本 METI 等
  │   └─ 内部拒绝/观察清单（既往合规问题、标记实体）
  │
  ├─ 步骤 3：评估筛查结果
  │   ├─ 无命中 → 记录筛查（日期、工具、筛查方、结果）。
  │   │   继续交易。停止。
  │   ├─ 返回命中 → 进入步骤 4
  │
  ├─ 步骤 4：判定每次命中
  │   ├─ 对每次命中，评估：
  │   │   ├─ 名称匹配质量（精确、近似、部分、语音）
  │   │   ├─ 地址相关性（同一国家？同一城市？同一街道？）
  │   │   ├─ 出生日期 / 成立日期（针对个人/实体）
  │   │   ├─ 别名匹配（匹配是否为已知别名？）
  │   │   ├─ 附加标识符（护照号、税号、邓白氏编号、船舶 IMO 编号）
  │   │   └─ 既往交易历史（10 年的已知良好客户？）
  │   ├─ 误报（高置信度）→ 记录判定理由。
  │   │   记录：判定人、日期、考虑因素、结论。
  │   │   继续交易。停止。
  │   ├─ 可能匹配（模糊）→ 进入步骤 5
  │   └─ 真阳性（已确认）→ 进入步骤 6
  │
  ├─ 步骤 5：升级模糊匹配
  │   ├─ 向客户/对手方请求附加识别信息
  │   ├─ 聘请合规顾问
  │   ├─ 在解决前**不**继续交易
  │   ├─ 记录冻结和所有已采取的步骤
  │   └─ 若无法解决 → 视为真阳性（步骤 6）
  │
  └─ 步骤 6：真阳性——完全停止
      ├─ 立即**阻断**交易
      ├─ SDN/OFAC 匹配：
      │   ├─ 在 10 个工作日内封锁并向 OFAC 报告
      │   ├─ 不放行货物、不处理付款、不向被封锁方
      │   │   透露原因
      │   └─ 确定 OFAC 许可证是否可用且合理
      ├─ 实体清单匹配：
      │   ├─ 确定许可证要求（大多数为"拒绝推定"）
      │   ├─ 若许可证例外可用，记录依据
      │   └─ 若无例外，申请 BIS 许可证（预期 60-90 天处理）
      ├─ 被拒绝人员清单匹配：
      │   ├─ **绝对**禁止——无许可证可用
      │   └─ 不得向被拒绝人员进行出口、再出口或国内转让
      ├─ 通知合规官和法律顾问
      ├─ 记录**一切**——命中、判定、决定、阻断
      └─ DPL 无限期保留记录；其他最少 5 年
```

### 4.2 红旗指标——加强尽职调查触发器

当出现以下任何红旗时，标准筛查不够。在继续前进行加强尽职调查：

| 红旗 | 提示 | 所需行动 |
|---|---|---|
| 客户拒绝说明最终用途 | 转移风险 | 要求提供最终用途证书或拒绝交易 |
| 异常路线（例如电子产品运至拉各斯→迪拜→巴库） | 通过禁运目的地逃避制裁/转移 | 绘制完整供应链；核实最终目的地 |
| 客户愿意为高价值资本货物支付现金 | 洗钱或逃避制裁 | 加强 KYC；核实资金来源 |
| 交付至邮政信箱、住宅地址或自由贸易区 | 隐藏真实最终用户 | 要求实际街道地址，若 > 5 万美元则现场访问 |
| 产品能力超过声明的最终用途 | 军事/大规模杀伤性武器转移 | 核实最终用途声明与产品规格匹配 |
| 客户无网络存在或可核实的业务历史 | 空壳公司或受制裁实体的前台 | 公司注册核查、邓白氏报告、实益所有权分析 |
| 备件订单与装机量不符 | 为禁运目的地囤积 | 要求装机量详情；核实服务历史 |
| 客户要求删除产品标签或标记 | 规避最终用途管制或商标欺诈 | 拒绝请求并上报合规 |
| 客户或转运代理以前被标记 | 重复合规风险 | 任何交易前进行高级合规审查 |
| 来自交易未涉及的第三方或国家的付款 | 通过金融中介逃避制裁 | 核实付款结构的商业理由 |

## 5. 处罚风险评估

### 5.1 美国处罚框架——详细分析

**19 USC § 1592——欺诈、重大疏忽和疏忽的处罚：**

| 要素 | 欺诈 | 重大疏忽 | 疏忽 |
|---|---|---|---|
| **心理状态** | 故意和知情违规，有欺诈意图 | 故意漠视已知义务或严重漠不关心 | 未能尽合理注意 |
| **最高处罚** | 商品的国内价值 | 4 倍损失收入，或应税价值的 40% | 2 倍损失收入，或应税价值的 20% |
| **刑事移送** | 是——每次罪名最高 1 万美元罚款和 2 年监禁（18 USC § 542） | 罕见 | 否 |
| **举证责任** | CBP 必须以明确和有说服力的证据证明欺诈 | CBP 必须以优势证据证明 | CBP 必须以优势证据证明；进口商必须证明合理注意 |
| **减轻潜力** | 有限——需要非凡配合 | 中等——合规项目、披露、配合 | 重大——初犯、纠正措施、小规模 |
| **事先披露效果** | 将处罚限制在 1 倍损失收入（如接受） | 将处罚限制在 1 倍损失收入 | 将处罚限制在未付税款的利息 |
| **诉讼时效** | 自违规之日起 5 年 | 5 年 | 5 年 |

**处罚减轻因素（依据 CBP 减轻指南）：**

1. **CBP 自身过错：** CBP 是否无异议地接受了相同错误的既往入境？
2. **配合：** 全面配合调查，包括提供记录和内部调查结果
3. **立即纠正措施：** 进口商是否在发现后立即修复问题？
4. **既往良好记录：** 5 年以上的清洁合规历史
5. **无力支付：** 经济困难（有文件证明）可减少金钱处罚
6. **违规规模：** 孤立事件与系统性模式
7. **合规项目：** 内部合规项目的存在和有效性

### 5.2 事先披露决策框架

```
起点：你已发现潜在的海关违规行为
  │
  ├─ 步骤 1：评估——CBP 是否已启动调查？
  │   ├─ 是（你已收到 CF-28 信息请求、CF-29 行动通知、
  │   │        预处罚通知或重点评估通知）→
  │   │   事先披露在正式调查启动前仍然可用。
  │   │   仅 CF-28/29 不构成启动。预处罚通知构成启动。
  │   │   ├─ 预处罚通知或正式调查已启动 → 事先披露
  │   │   │   不可用。在法律顾问协助下回应通知。停止。
  │   │   └─ 仅 CF-28/29 → 事先披露仍然可用。继续。
  │   └─ 否 → 事先披露可用。进入步骤 2。
  │
  ├─ 步骤 2：确定违规的性质和范围
  │   ├─ 归类错误 → 受影响入境有多少？计算总税款差额。
  │   ├─ 估价错误 → 量化少报价值和相应的税款影响。
  │   ├─ 原产国错报 → 识别所有受影响入境和正确原产国。
  │   ├─ FTA 过度索赔 → 计算无优惠情况下本应支付的税款。
  │   ├─ 记录保存失败 → 识别哪些记录缺失以及针对哪些入境。
  │   └─ 其他 → 定义具体违规并量化税款影响。
  │
  ├─ 步骤 3：评估事先披露与替代策略
  │   ├─ 事先披露**建议**在以下情况下使用：
  │   │   ├─ 违规明确且税款缺口可量化
  │   │   ├─ 你确信 CBP 尚未开始调查此问题
  │   │   ├─ 处罚风险重大（事先披露将其限制在利息/1 倍）
  │   │   └─ 违规涉及疏忽或重大疏忽（非欺诈）
  │   ├─ 事先披露**不建议**在以下情况下使用：
  │   │   ├─ 归类或估价立场可辩护，且你准备诉讼
  │   │   ├─ 涉案金额极小（披露的行政成本超过收益）
  │   │   ├─ 违规不确定——你可能在披露实际上并无问题的事项
  │   │   └─ 披露将揭示 CBP 不知晓的其他问题（先咨询顾问）
  │   └─ 提交前**始终**咨询法律顾问。事先披露是对违规的承认。
  │
  ├─ 步骤 4：准备事先披露
  │   ├─ 必要要素（19 CFR § 162.74）：
  │   │   ├─ 识别所有受影响入境的入境编号和日期
  │   │   ├─ 描述具体违规（错在哪里）
  │   │   ├─ 提供正确信息（本应申报什么）
  │   │   ├─ 识别导致违规的情况（如何发生）
  │   │   ├─ 计算应付税款和利息
  │   │   └─ 交付未付税款全额（或解释为何无法精确计算
  │   │       并提供善意估计和支付最终金额的承诺）
  │   ├─ 提交至入境港的 FP&F 办公室（或具有管辖权的 CEE）
  │   └─ 保留带有交付证明的副本
  │
  └─ 步骤 5：提交后
      ├─ CBP 将审查披露并可能要求提供附加信息
      ├─ 若接受，你将收到限于利息（疏忽）或
      │   1 倍损失收入（重大疏忽）的处罚通知
      ├─ 若拒绝（披露不完整或调查已启动），
      │   全额处罚风险恢复
      └─ 实施纠正措施以防止再次发生——若发生另一次违规
          CBP 将关注此点
```

### 5.3 出口管制处罚框架

**BIS（EAR）处罚：**
- 民事：每次违规最高 330,198 美元或交易价值的两倍，以较高者为准
  （按年度通胀调整）
- 刑事：每次违规最高 1,000,000 美元和 20 年监禁
- 出口特权拒绝：自动禁止来自美国的所有出口/再出口

**DDTC（ITAR）处罚：**
- 民事：每次违规最高 1,301,256 美元（按年度调整）
- 刑事：每次违规最高 1,000,000 美元和 20 年监禁
- 取消所有 ITAR 受控出口资格

**OFAC 处罚：**
- 因制裁项目而异，但基于 IEEPA 的制裁民事处罚每次违规可超过 330,000 美元
- 刑事：依据 IEEPA 最高 1,000,000 美元和 20 年监禁
- OFAC 违规的严格责任——民事处罚无需意图

## 7. Incoterms 决策矩阵

### 7.1 选择适当的 Incoterm

| 考虑因素 | 推荐的 Incoterm | 理由 |
|---|---|---|
| 买方希望最大程度控制物流 | FCA | 买方选择承运人、路线和保险。卖方处理出口手续。 |
| 卖方有更好的运费率（规模经济） | CIF / CIP | 卖方利用批量合同。买方从第一承运人承担风险。 |
| 买方无法在卖方国家作为出口商行事 | FCA、CPT、CIP、DAP、DDP | 避免 EXW——它使买方成为原产国的出口商备案人。 |
| 买方在目的地缺乏进口能力 | DDP | 卖方处理包括进口清关在内的一切。卖方必须注册为 IOR。 |
| 信用证要求已装船提单 | FCA（带 2020 A6/B6 选项） | 2020 修订版允许买方指示承运人向卖方签发已装船提单。 |
| 高风险运输（盗窃、损坏、海盗走廊） | CIF / CIP | 卖方负责保险。CIP 要求一切险覆盖（ICC A）。 |
| 集装箱海运 | FCA、CPT、CIP | FOB 对集装箱技术上不正确——风险在集装箱堆场转移，而非船舷。 |
| 国内交付（同国） | FCA、DAP | 国内不需要 Incoterms；若使用，FCA 或 DAP 是适当的。 |

### 7.2 Incoterms 对海关估价的影响

Incoterm 影响海关价值，因为不同术语在发票价格中包含或排除运费和保险：

| Incoterm | 美国海关价值调整 | 欧盟海关价值调整 |
|---|---|---|
| EXW | 增加：到港口的内陆运费 + 国际运费 + 到美国港口的保险 | 增加：到港口的内陆运费 + 国际运费 + 到欧盟边境的保险 |
| FCA | 增加：从指定地点到美国港口的国际运费 + 保险 | 增加：从指定地点到欧盟边境的国际运费 + 保险 |
| FOB | 增加：到美国港口的海运费 + 保险（已包括到港口的内陆） | 增加：到欧盟边境的海运费 + 保险 |
| CFR/CPT | 增加：保险（运费已包括） | 增加：保险（运费已包括） |
| CIF/CIP | 无调整（运费和保险已包括） | 无调整（运费和保险已包括） |
| DAP | 扣除：到美国最终目的地的内陆运费（若可识别） | 扣除：从欧盟边境到目的地的内陆运费 |
| DDP | 扣除：内陆运费 + 进口关税（关税永远不是海关价值的一部分） | 扣除：从欧盟边境到目的地的内陆运费 + 进口关税 |
