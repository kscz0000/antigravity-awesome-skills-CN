## 1.1 离婚

#### 协议离婚（非诉讼）（Lei 11.441/2007）

| 要求 | 详情 |
|-----------|---------|
| **协议** | 双方同意离婚及条款 |
| **无未成年/无行为能力子女** | 如有，必须诉讼 |
| **公证书** | 在公证处签署 |
| **律师** | 必需（双方可共用一名） |
| **期限** | 即时（自EC 66/2010起无分居期限） |
| **平均费用** | R$ 1,500 至 R$ 4,000（费用 + 律师费） |
| **财产分割** | 可包含在同一公证书中 |

#### 诉讼离婚（Art. 731-734 CPC）

| 类型 | 描述 |
|------------|-----------|
| **协议** | 双方同意 — 法官批准（Art. 731 CPC） |
| **争议** | 无协议 — 法官裁决（Art. 693 CPC） |
| **管辖** | 子女监护人住所或夫妻最后住所（Art. 53, I CPC） |

#### 财产制度（Art. 1.639-1.688 CC）

| 制度 | 特点 | 配偶份额 |
|--------|---------------|--------|
| **部分共有**（默认） | 婚姻期间获得的财产 = 共同财产 | 获得财产的50% |
| **全部共有** | 一切共有（除Art. 1.668例外） | 全部的50% |
| **完全分离** | 无共同财产 | 无配偶份额 |
| **法定分离** | 法律强制（Art. 1.641 CC） | STF Sumula 377：婚后获得财产平分 |
| **最终参与分配** | 婚姻期间分离 + 解除时共有 | 增值的50% |

#### STF Sumula 377 — 法定财产分离

在法定财产分离制度下，婚姻期间获得的财产为共同财产。

**实际应用：** 70岁以上婚姻（Art. 1.641, II CC） — 即使是法定分离，配偶仍有权获得婚姻期间获得财产的一半。

## 1.2 抚养费

#### 法律依据

- **Art. 1.694-1.710 CC** — 亲属、配偶和伴侣之间的抚养费
- **Lei 5.478/1968** — 抚养费法（特别程序）
- **Art. 528-533 CPC** — 抚养费执行（监禁、查封、工资扣除）

#### 抚养费类型

| 类型 | 描述 |
|------|-----------|
| **临时抚养费** | 在抚养费诉讼中初步确定（Art. 4 Lei 5.478） |
| **紧急抚养费** | 在紧急保护令中确定（Art. 300 CPC） |
| **最终抚养费** | 在判决中确定 |
| **补偿性抚养费** | 用于平衡财产失衡（STJ — REsp 1.954.279） |
| **孕期抚养费** | 给孕妇的抚养费（Lei 11.804/2008） |
| **过渡性抚养费** | 给前配偶的临时抚养费，助其重新自立 |

#### 抚养费执行（Art. 528-533 CPC）

| 途径 | 程序 | 期限 |
|-----|-------------|-------|
| **民事监禁** | Art. 528, par. 3 — 封闭式1-3个月 | 3期（STJ Sumula 309） |
| **查封** | 按确定金额执行 | 每期2年时效 |
| **工资扣除** | Art. 529 CPC — 向雇主发出命令 | 最高净收入的50% |
| **SISBAJUD** | 账户冻结 | 即时 |
| **抗议** | Art. 528, par. 1 — 票据抗议 | 无限制 |

#### 需求与能力原则（Art. 1.694, par. 1 CC）

- **被抚养人的需求：** 生活费用、医疗、教育、住房
- **抚养人的能力：** 收入、资产、生活水平
- **比例原则：** 法官平衡两者

#### 确定标准（判例）

| 情况 | 常见标准 |
|----------|----------------|
| 1个子女（CLT） | 净收入的30% |
| 2个子女（CLT） | 33-40% |
| 3个及以上子女（CLT） | 40-50% |
| 自雇/非正式就业 | 最低工资的百分比（1-3 SM） |
| 前配偶抚养费 | 收入的20-33%（临时） |

## 1.3 事实婚姻（Art. 1.723-1.727 CC）

| 方面 | 详情 |
|---------|---------|
| **要件** | 公开、持续、持久的共同生活，组建家庭的目的 |
| **财产制度** | 部分共有（除非另有约定 — Art. 1.725 CC） |
| **认定** | 可通过司法、非诉讼（公证书）或死后认定 |
| **继承权** | 伴侣与直系卑亲属和直系尊亲属竞争（Art. 1.790 CC — 被STF RE 878.694宣告违宪） |
| **居住权** | 有（类比婚姻 — STJ） |
| **解除** | 与离婚相同（Art. 7, par. 2 Lei 9.278/96） |

## 1.4 父亲身份确认

| 方面 | 详情 |
|---------|---------|
| **法律依据** | Lei 8.560/1992 + Art. 1.606-1.617 CC |
| **诉讼** | 父亲身份确认附带抚养费 |
| **管辖** | 未成年人住所（Art. 53, II CPC） |
| **DNA** | 最佳鉴定证据 — 但拒绝产生推定（STJ Sumula 301） |
| **无时效限制** | Art. 27 ECA — 诉讼不受时效限制 |
| **否认之诉** | Art. 1.601 CC — 丈夫可质疑父亲身份 |
| **社会情感亲子关系** | STF Tema 622 — 社会情感亲子关系不排除生物学亲子关系 |

## 4.1 离婚财产分割

#### 共同财产与个人财产（部分共有制）

| 共同财产（50%份额） | 个人财产 |
|------------------------|----------------------------------|
| 婚姻期间购买的房产 | 婚前财产（Art. 1.659, I CC） |
| 婚姻期间获得的车辆 | 继承和赠与（Art. 1.659, I CC） |
| 用工作收入进行的投资 | 个人财产的替代财产（Art. 1.659, II CC） |
| 联名账户余额 | 附不可转让标记的财产（Art. 1.659, III CC） |
| 婚姻期间累积的FGTS（STJ） | 个人使用的物品、书籍、职业工具（Art. 1.659, V CC） |
| 私人养老金（STJ — 有争议） | 个人工作收入（Art. 1.659, VI CC） — 有争议 |

#### 财产评估

| 方法 | 适用情况 |
|--------|-------------|
| 司法鉴定评估（Art. 464 CPC） | 房产、公司、高价值财产 |
| 双方协议 | 协议离婚 — 双方确定价值 |
| 市场评估（中介/评估师） | 住宅房产、车辆 |
| 资产负债表 | 股权、企业投资 |

## 4.2 死亡继承与财产分割

#### 非诉讼继承（Art. 610, par. 1 CPC + Lei 11.441/2007）

| 要求 | 详情 |
|-----------|---------|
| 所有继承人成年且有行为能力 | 必须 |
| 对分割达成一致 | 全部同意 |
| 无遗嘱 | 一般规则（例外：CNJ Resolucao 35/2007, Art. 12-A允许已确认遗嘱） |
| 公证书 | 公证处 — 任何市镇 |
| 律师 | 必须 |
| 期限 | 继承开始后60天（Art. 611 CPC） — 超期罚款ITCMD |

#### 诉讼继承（Art. 610-673 CPC）

| 类型 | 适用情况 |
|------------|--------|
| **简易清单**（Art. 659 CPC） | 继承人有行为能力 + 协议 |
| **普通清单**（Art. 664 CPC） | 财产不超过1,000 SM |
| **传统继承**（Art. 610 CPC） | 继承人无行为能力、分歧、遗嘱 |

#### 继承顺序（Art. 1.829 CC）

| 顺序 | 继承人 | 备注 |
|-------|-----------|-----------|
| 第1顺序 | 直系卑亲属 + 配偶 | 配偶与直系卑亲属竞争（Art. 1.832 CC） |
| 第2顺序 | 直系尊亲属 + 配偶 | 配偶与父母竞争得1/3（Art. 1.837 CC） |
| 第3顺序 | 生存配偶（单独） | 获得全部 |
| 第4顺序 | 四亲等内旁系亲属 | 兄弟姐妹、侄子女、叔伯姑舅、表兄弟姐妹 |

#### 生存配偶的权利

| 财产制度 | 与直系卑亲属竞争？ | 依据 |
|----------------|--------------------------|------|
| 部分共有 | 是，针对死者**个人财产** | Art. 1.829, I CC |
| 全部共有 | 不竞争 | Art. 1.829, I CC |
| 法定分离 | 不竞争（有争议 — STF Sumula 377） | Art. 1.829, I CC |
| 约定分离 | 是，针对全部财产竞争 | STJ — REsp 1.382.170 |

#### 伴侣（事实婚姻）

- **STF RE 878.694（Tema 498）：** 将伴侣等同于配偶用于继承目的
- Art. 1.790 CC被宣告违宪 — 适用Art. 1.829 CC

## 4.3 遗嘱

| 类型 | 法律依据 | 要求 |
|------|-----------|-----------|
| **公证遗嘱** | Art. 1.864 CC | 公证人 + 2名见证人 |
| **密封遗嘱** | Art. 1.868 CC | 立遗嘱人书写，公证人批准 |

## 5.1 监护类型（Art. 1.583-1.590 CC + Lei 13.058/2014）

| 类型 | 描述 | 法律依据 |
|------|-----------|-----------|
| **共同监护** | 原则 — 双方行使监护权，即使无协议 | Art. 1.584, par. 2 CC |
| **单方监护** | 一方行使，另一方有探视权 | Art. 1.583, par. 1 CC |
| **轮流监护** | 儿童定期更换住所 | 判例（法律未规定） |
| **巢式监护** | 儿童固定，父母轮流 | 巴西罕见 |

## 5.2 共同监护（Lei 13.058/2014）

| 方面 | 详情 |
|---------|---------|
| **一般原则** | 即使无协议也是**原则**（Art. 1.584, par. 2 CC） |
| **例外** | 仅当一方声明不想要监护或无能力时不适用 |
| **居住基地** | 儿童有居住基地，但与双方共同生活 |
| **共处时间** | 平衡 — 不必是50/50 |
| **决策** | 双方共同决定健康、教育、娱乐 |
| **抚养费** | 共同监护**不排除**抚养费（STJ — REsp 1.629.994） |

## 5.3 探视安排（Art. 1.589 CC）

| 方面 | 详情 |
|---------|---------|
| **权利主体** | 父母和儿童（儿童利益优先） |
| **祖父母** | 有探视权（Art. 1.589, par. unico CC — Lei 12.398/2011） |
| **确定方式** | 司法或协议 |
| **违反** | 儿童搜查令（Art. 461 CPC） + 罚款 |
| **监督** | 有风险时监督探视 |

## 5.4 亲职异化（Lei 12.318/2010）

#### 定义（Art. 2）

干扰儿童心理形成，由一方父母（或祖父母/监护人）实施，旨在损害儿童与另一方父母的纽带。

#### 异化形式（Art. 2, paragrafo unico）

| # | 形式 |
|---|-------|
| I | 诋毁父母的运动 |
| II | 阻碍行使亲权 |
| III | 阻碍儿童与父母接触 |
| IV | 阻碍共处权的行使 |
| V | 隐瞒重要个人信息（学校、健康） |
| VI | 提出虚假举报以阻碍共处 |
| VII | 更换住所以阻碍共处 |

#### 制裁（Art. 6）

| 制裁 | 严重程度 |
|--------|-----------|
| 警告 | 轻微 |
| 扩大共处制度 | 中等 |
| 罚款 | 中等 |

## 完整案件分析工作流程（12步）

对于任何复杂法律案件，遵循以下12步：

## 第1步 — 法律定性

- 法律领域（哪个模块？）
- 主要法律依据（哪部法律、哪条、哪款？）
- 管辖（哪个法院/法庭？）

## 第2步 — 当事人

- 确定当事人（原告/被告/第三人）
- 当事人关系（亲属、合同、非合同）
- 脆弱性（未成年人、老人、消费者、弱势方）

## 第3步 — 相关事实

- 事件时间线
- 现有/必要文件
- 证人和可用证据

## 第4步 — 法律依据

- 适用法律条文
- 相关判例（STJ、STF、TJs）
- 适用学说（相关时）

## 第5步 — 实体分析

- 客户的权利有依据吗？
- 有可行的法律抗辩吗？
- 现有证据力度？

## 第6步 — 诉讼风险

- 时效/除斥期间
- 当事人适格和利益
- 地域/事物管辖
- 期限届满

## 第7步 — 结果预估

- 乐观情景
- 基本情景（最可能）
- 悲观情景

## 第8步 — 预估成本

- 诉讼费
- 律师费（约定和败诉方承担）
- 鉴定和附加费用
- 司法援助（如适用 — Art. 98 CPC）

## 第9步 — 诉讼策略

- 非诉讼途径（调解、和解、仲裁）
- 诉讼途径（程序、请求、紧急保护）
- 可用上诉

## 第10步 — 相关期限

- 权利时效
- 诉讼期限
- 上诉期限
- 判决履行期限

## 第11步 — 紧急措施

- 预防性紧急保护（Art. 300 CPC）
- 证据性保护（Art. 311 CPC）

## 治理

此技能实施以下政策：

- **action_log**：每次法律分析都记录以便追溯
- **rate_limit**：通过生态系统的check_rate集成控制
- **requires_confirmation**：高风险警报向用户发出确认请求
- **warning_threshold**：诉讼风险高时自动警报
- **负责人：** 法律技能生态系统
- **范围：** 巴西法律所有领域
- **限制：** 不替代现场律师。分析基于提供的数据。
- **审计：** 由skill-sentinel验证
- **敏感数据：** 不存储用户的个人或诉讼数据
