## 1.2 暴力形式（11.340/2006号法律第7条）

| 形式 | 定义 | 示例 |
|-------|-----------|----------|
| **身体暴力**（I） | 侵犯身体完整性或健康 | 耳光、拳打、推搡、烧伤、勒颈 |
| **心理暴力**（II） | 情感伤害、自尊降低、控制 | 羞辱、威胁、孤立、煤气灯效应、操控 |
| **性暴力**（III） | 强迫观看/维持/参与性行为 | 婚内强奸、阻止使用避孕措施、强迫堕胎 |
| **财产暴力**（IV） | 扣留、取走、毁坏物品/工作工具 | 扣留证件、毁坏手机、控制金钱 |
| **道德暴力**（V） | 诽谤、诋毁或侮辱 | 公开指控出轨、暴露隐私、辱骂 |

## 1.3 紧急保护措施

#### 针对施暴者（Art. 22）

| 措施 | 描述 |
|--------|-----------|
| **I** | 暂停武器持有/携带许可 |
| **II** | 驱逐出住所/住宅 |
| **III-a** | 禁止接近（由法官确定最小距离） |
| **III-b** | 禁止任何形式的接触 |
| **III-c** | 禁止出入特定场所 |
| **IV** | 限制/暂停对未成年子女的探视 |
| **V** | 临时抚养费 |
| **par. 5** | **电子监控**（脚镣）— 15.125/2025号法律 |

#### 有利于被害人（Art. 23）

| 措施 | 描述 |
|--------|-----------|
| **I** | 转介至保护计划 |
| **II** | 施暴者被驱逐后返回住所 |
| **III** | 被害人离开且不丧失权利 |
| **IV** | 分居 |

#### 财产保护（Art. 24）

| 措施 | 描述 |
|--------|-----------|
| **I** | 返还被施暴者取走的财产 |
| **II** | 禁止出售/出租共有财产 |
| **III** | 暂停授权书 |
| **IV** | 临时担保 |

## 1.4 接待流程 — 家庭暴力被害人

```
处于暴力处境的被害人
│
├─→ 紧急情况（生命直接危险）
│   └─→ 拨打190（宪兵）或180（妇女中心）
│       └─→ 现行犯 + 立即驱逐施暴者
│           └─→ 警察局（B.O.）+ DEAM（如有）
│               └─→ 48小时内签发保护措施（Art. 12-C）
│
├─→ 紧急（暴力反复发生）
│   └─→ 警察局（B.O.）或DEAM
│       └─→ 申请保护措施
│           └─→ 法官48小时内决定（未经听证）
│               └─→ 必要时电子监控
│
├─→ 咨询（想了解权利）
│   └─→ CRAM（妇女援助中心）
│       └─→ 公设辩护人办公室 / OAB / 私人律师
│           └─→ 案件评估 + 法律策略
│
└─→ 红色信号（14.188/2021号法律）
    └─→ 在手上画红色X
        └─→ 在参与的药店/医院出示
            └─→ 机构通知当局
```

## 5.1 程序欺诈（CP第347条）

```
Art. 347. 在民事或行政诉讼进行期间，
以欺诈手段改变地点、物品或人的状态，
意图误导法官或鉴定人。

刑罚 — 3个月至2年拘留，并处罚金。

唯一款。如果变更是为了在刑事诉讼中产生效力，
即使尚未开始，刑罚加倍。
```

## 5.2 犯罪要素

| 要素 | 描述 |
|----------|-----------|
| **行为** | 以欺诈手段改变地点、物品或人的状态 |
| **特定故意** | 误导法官或鉴定人的意图 |
| **时间** | 程序进行期间（或之前，如刑事） |
| **形式犯** | 变更即既遂，不问结果 |
| **未遂** | 可成立 |
| **诉讼类型** | 公诉（无条件） |

## 5.4 欺诈作为加重情节

欺诈可作为：
- **一般加重情节**（Art. 61, II, "c" CP）— 背叛、埋伏、**伪装**或其他妨碍被害人防御的手段犯罪
- **杀人加重情节**（Art. 121, par. 2, IV）— 背叛、埋伏、**伪装**或其他妨碍被害人防御的手段
- **诈骗加重情节**（Art. 171, par. 1）— 针对老年人

## 6.1 三阶段制度（CP第68条）

```
阶段1：基础刑（Art. 59 CP — 司法情节）
  → 罪责、前科、社会行为、人格、
    动机、情节、后果、被害人行为
  → 结果：法定最低与最高之间

阶段2：加重和减轻情节
  → 加重情节（Arts. 61-62）和减轻情节（Arts. 65-66）
  → 不得超过法定限度（摘要231 STJ）

阶段3：增减原因
  → 加重和减轻事由（分则和总则）
  → 可超过法定限度
```

## 模块10 — 完整刑事案件分析工作流程

收到刑事案件进行分析时，**始终**遵循这10个步骤：

## 步骤1 — 事实定性

- 犯罪类型（什么犯罪？）
- 法律依据（CP哪一条/特别立法？）
- 分类：故意/过失、未遂/既遂、普通/特别/恶性犯罪

## 步骤2 — 主体

- 主动主体（谁实施的？）
- 被动主体（谁受害？）
- 他们之间的关系（家庭、职业、陌生人）
- 被害人脆弱性

## 步骤3 — 客观要件与主体认定

- 客观要件证据（鉴定报告、B.O.、照片、医疗记录）
- 主体证据（证人、监控、供述、数字证据）
- 是否有足够迹象起诉？

## 步骤4 — 情节

- 适用加重和减轻情节
- 增减原因
- 罪数（实质竞合、想象竞合、连续犯）

## 步骤5 — 预估量刑

- 预估基础刑（阶段1）
- 加重/减轻情节（阶段2）
- 加重/减轻事由（阶段3）
- 可能的初始服刑方式

## 步骤6 — 程序问题

- 管辖权（刑事法庭、陪审团、轻微犯罪法院、家庭暴力法庭）
- 诉讼类型（公诉条件、无条件、自诉）
- 现行犯羁押？预防性？临时性？
- 适用保全措施

## 步骤7 — 可用论点

- 辩护：哪些论点可行？
- 控诉：哪些是强项？
- 相关判例

## 步骤8 — 风险与情景

- 乐观情景（无罪、降格）
- 基本情景（有罪但有减轻情节）
- 悲观情景（法定最高刑）

## 步骤9 — 推荐策略

- 和解（如适用ANPP — Art. 28-A CPP）
- 审判辩护
- 与MP谈判
- 可能的上诉

## 14.3 Maria da Penha背景下的诬告

**敏感情况**：当所谓被害人伪造侵害以获取利益（监护权、房产、抚养费）。

**注意要点：**
- 家庭暴力中被害人陈述有特殊分量（隐秘犯罪）
- 主张虚假需要**有力证据**（仅否认不足）
- 无根据指控有二次伤害风险
- 如证明虚假：Art. 339 CP（诬告）— 2-8年监禁

**可证明虚假的证据：**
- IML鉴定报告阴性 / 与指控不符
- 矛盾信息（WhatsApp、SMS）
- 安全监控
- 目击证人
- 已证不在场证明（地理定位、银行卡、监控）
- 双方之间财产诉讼历史

| **赦免** | 总统令 | 根据年度令 |

## 15.2 折抵与减免

- **折抵**（Art. 42 CP）：临时羁押和住院时间从最终刑期中扣除
- **工作减免**（Art. 126 LEP）：3天工作 = 1天刑期
- **学习减免**（Art. 126, par. 1, I LEP）：12小时学习 = 1天刑期
- **阅读减免**（CNJ 44/2013建议）：1本书/30天 = 4天刑期（最多12天/年）
