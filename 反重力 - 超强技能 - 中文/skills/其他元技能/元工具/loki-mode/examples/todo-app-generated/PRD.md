# PRD：简单待办应用

## 概述
一个最小化的待办应用，用于测试 Loki Mode 的简单、明确定义范围的功能。

## 目标用户
想要简单方式跟踪任务的个人用户。

## 功能

### MVP 功能
1. **添加待办** - 用户可以添加带标题的新待办项
2. **查看待办** - 显示所有待办列表
3. **完成待办** - 将待办标记为已完成
4. **删除待办** - 从列表中移除待办

### 技术栈（建议）
- 前端：React + TypeScript
- 后端：Node.js + Express
- 数据库：SQLite（本地文件）
- 无部署（仅本地测试）

## 验收标准

### 添加待办
- [ ] 待办标题输入框
- [ ] 提交按钮
- [ ] 新待办出现在列表中
- [ ] 提交后清空输入框

### 查看待办
- [ ] 在列表中显示所有待办
- [ ] 显示完成状态
- [ ] 无待办时显示空状态

### 完成待办
- [ ] 复选框或按钮标记完成
- [ ] 已完成项目的视觉指示器
- [ ] 刷新后持久化

### 删除待办
- [ ] 每个待办上的删除按钮
- [ ] 删除前确认
- [ ] 从列表和数据库中移除

## 不在范围内
- 用户认证
- 截止日期
- 分类/标签
- 移动应用
- 云部署

## 成功指标
- 所有功能正常运行
- 测试通过
- 无控制台错误

---

**目的：** 此 PRD 故意保持简单，以便快速测试 Loki Mode 的核心功能，无需等待复杂的构建或部署。
