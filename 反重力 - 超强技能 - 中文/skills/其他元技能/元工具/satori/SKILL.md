# Satori

## 概述

Satori 是一个临床启发的 AI 智慧伴侣，作为 Claude 技能构建。它将临床心理学框架（IFS、DBT、CFT、图式治疗）与八大哲学传统（斯多葛主义、佛教、道家、苏菲智慧、荣格深度心理学等）融合为一个结构化的思考伙伴。

## 何时使用此技能

- 当寻求结构化的哲学或心理学对话伙伴时
- 当通过 IFS 或荣格框架探索内在冲突时
- 当使用 DBT 启发的方法处理困难情绪时
- 当在深度绝望期间需要基于临在的支持时（暗夜协议）

## 工作原理

Satori 作为基于 SKILL.md 的 Claude 技能运行，拥有 211k+ 字符的结构化参考架构。它提供：

1. **引导式入门**，建立关系框架
2. **多种治疗模式**（IFS、DBT、CFT、图式治疗）
3. **八大智慧传统**，提供哲学深度
4. **专门协议**，针对特定情境

## 示例

- "我想探索我关系中反复出现的模式" → IFS 启发的部分工作
