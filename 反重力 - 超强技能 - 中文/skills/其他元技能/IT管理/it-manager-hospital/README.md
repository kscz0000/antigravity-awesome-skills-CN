# 医院IT经理（医疗数字化领导者）

医疗行业技术领导者的权威顾问伙伴。

## 🏥 概述

**it-manager-hospital** 是一项精英级技能，旨在支持复杂医院环境中的IT经理和CIO。它弥合了传统IT治理与临床安全、互操作性和数字化成熟度这一使命关键领域之间的鸿沟。

### 核心支柱
- **临床安全：** 通过"零浪费"和"条码"技术减少医疗差错。
- **认证卓越：** 掌握ONA Nível 3、HIMSS Stage 7和JCI认证的路径。
- **HIS/PEP专长：** 深入了解MV-SOUL、MV-PEP和Tasy系统。
- **法律与合规：** 全面符合LGPD（巴西）、RNDS标准和远程医疗法规。

## 👤 虚拟专家委员会（10位专家）

本技能激活由10位专家角色组成的集体，分析每一个管理场景，从**临床战略师**到**患者安全守护者**。

## 🛠 使用方式

使用特定短语触发技能：
- `it manager hospital: we are starting our HIMSS EMRAM journey, where do we begin?`
- `ti hospitalar: como integrar o MV PEP com a farmácia para medicação beira-leito?`
- `ona accreditation it: what are the security requirements for Nível 1?`

### 💡 协议
技能将首先提供核心战略答案，然后**询问**您是否希望深入了解临床适用性或真实案例。
