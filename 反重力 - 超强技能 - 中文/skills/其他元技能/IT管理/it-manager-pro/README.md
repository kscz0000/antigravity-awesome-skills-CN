# IT Manager Pro（精英顾问）

面向IT经理、CTO和产品领导者的终极数字化领导力伙伴。

## 🚀 概览

**IT Manager Pro** 是为2026数字化环境设计的高保真咨询技能。它从多专家视角提供现代技术领导力最关键支柱的洞察：
- **战略对齐：** 使用ITIL 5（DPSM）实现业务-IT融合。
- **现代FinOps：** 管理从云到AI推理的技术总价值。
- **以人为本的领导力：** VUCA环境中的情商与绩效。
- **数据驱动治理：** 算法伦理、可持续发展（ESG）和持续改进。

## 👥 七专家顾问团

在单次对话中咨询虚拟专家团队：
1.  **战略家：** 价值共创与ITIL 5。
2.  **FinOps审计师：** 预算与GPU/AI价值。
3.  **人才教练：** 领导力与文化。
4.  **风险守卫：** 安全与AI伦理。
5.  **ESG官：** 可持续发展。
6.  **持续改进工程师：** 数据驱动优化。
7.  **沟通桥梁：** C-level叙事。

## 🛠 使用方法

使用触发短语激活技能：
- `it manager pro: we have an outage, help me draft the executive update.`
- `ti management: what are the key KPIs for ai-native service delivery?`
- `finops strategy: how do I self-fund our new agentic ai project?`

### 💡 基于审批的洞察（特性）
为保持回复简洁高效，技能会先提供核心答案，然后**询问**您是否需要深入洞察或真实适用性案例。
