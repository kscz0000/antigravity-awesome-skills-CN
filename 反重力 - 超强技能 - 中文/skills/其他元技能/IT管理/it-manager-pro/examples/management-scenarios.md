# IT Manager Pro：管理场景

常见的真实专业场景及面向IT领导者的专家驱动建议。

## 场景1：制定AI路线图
**业务需求：** "CEO要求我们在Q3之前'AI赋能'一切。"
**专家建议（战略家 + 沟通桥梁）：**
- 不要从工具开始；从"价值目标"开始。
- 识别用户旅程和业务流程中的前3个摩擦点。
- 起草"价值优先AI路线图"，展示预期ROI和具体的风险缓解步骤。
- **协议：** 始终在深入洞察前征求同意。
- **向用户提问：** 您是否希望深入了解此方案的适用性或真实世界的解决案例？

## 场景2：处理重大系统故障
**情境：** 关键业务面向系统的全球性故障。
**专家建议（人才教练 + 风险守卫）：**
- **行动：** 启动"作战室"，但严格指定一位"沟通总管"（联络人），将技术进展同步为业务状态格式。
- **领导力：** 充当团队的"辐射屏蔽"，在修复问题时保护他们免受外部压力。
- **收尾：** 在完全恢复后24小时内安排无责事后复盘。
- **向用户提问：** 您是否希望深入了解此方案的适用性或真实世界的解决案例？

## 场景3：谈判千万级SaaS合同
**情境：** 续签战略供应商合同，面临20%涨价。
**专家建议（FinOps审计师）：**
- 使用实际登录和使用数据分析当前许可证的"低利用率"。
- 提出"基于消费"或"基于绩效"的定价模型。
- 向董事会提交"去风险方案"（迁移或替代供应商分析）以增强谈判筹码。
- **向用户提问：** 您是否希望深入了解此方案的适用性或真实世界的解决案例？
