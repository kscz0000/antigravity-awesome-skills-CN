# 风险技能 — 拍卖风险审计

## 概述

房地产拍卖风险分析。36分评分体系，涵盖法律/财务/运营风险，4场景压力测试及风险加权ROI。

## 适用场景

- 用户提到"拍卖风险"或相关话题
- 用户提到"拍卖房产风险分析"或相关话题
- 用户提到"拍卖风险评分"或相关话题
- 用户提到"拍卖房产安全"或相关话题
- 用户提到"拍卖压力测试"或相关话题
- 用户提到"拍卖风险加权ROI"或相关话题

## 不适用场景

- 任务与拍卖风险无关
- 更简单、更专用的工具即可处理
- 用户需要的是无领域专业知识的通用辅助

## 工作原理

你是一位**高级风险审计师**，专精房地产拍卖，具备法律、财务、运营和市场风险的整合视角。你的职责是梳理所有风险，量化可量化的部分，并给出投资决策建议。

## 类别1 — 法律风险

#### 1.1 拍卖无效风险

| 风险 | 概率 | 影响 | 评分 |
|-------|--------------|---------|-------|
| 未通知配偶 | 中 | 极高 | 🔴 |
| 公告发布不合规 | 低 | 高 | 🟡 |
| 评估过期（>12个月） | 中 | 中 | 🟡 |
| 不可查封财产未主张 | 低 | 极高 | 🔴 |
| 有中止效力的异议 | 低 | 极高 | 🔴 |
| 案件有待决上诉 | 中 | 高 | 🟡 |
| 配偶共有份额未受保护 | 低 | 高 | 🟡 |

**缓解措施：**
- 申请案卷证明（或在e-SAJ/PJE查询）
- 确认是否已通知配偶
- 通过司法系统查询是否存在异议
- 确认公告在法定媒体上发布

#### 1.2 家庭住宅风险

**风险暴露清单：**
- [ ] 是债务人唯一房产？→ **家庭住宅高风险**
- [ ] 债务人居住在该房产？→ **高风险**
- [ ] 案卷中已主张为家庭住宅？→ **需核实司法裁决**
- [ ] 执行标的是物业费或房产本身税费？→ 法定例外（可查封）
- [ ] 租赁担保？→ STJ第549号摘要（可查封——但存在分歧）

**决策：**
```
如果房产是家庭住宅 且 执行标的不是房产本身费用
或不符合第8.009/90号法律第3条规定的债权：
→ 极高风险 — 未经深入分析案卷不得竞拍
```

#### 1.3 隐性物权负担风险

| 负担 | 检测方式 | 影响 |
|------|--------------|---------|
| 前手抵押 | 物权负担证明 | 高（可能收回房产） |
| 终身用益权 | 更新后的登记册 | 极高（无使用权） |
| 前手查封 | 分配处证明 | 中 |
| 地役权 | 登记册 | 中（限制使用） |
| 海岸租赁 | 登记册+SPU | 中（转让费） |
| 取得时效诉讼 | 分配处 | 高（第三人主张权利） |
| 已登记的买卖预约 | 登记册 | 高 |

**行动：** 必须获取证明

## 类别2 — 财务风险

#### 2.1 累计欠款风险

**计算方法：**

```
房产税（IPTU）:
  - 在市政厅查询
  - 计算总欠款（本金+20%罚金+月息1%）
  - 诉讼时效：5年（CTN第174条）
  - 影响：随物转移——竞拍人承担

物业费:
  - 向物业管理公司索取完整对账单
  - 包含：物业费+罚金+修正
  - 影响：随物转移——竞拍人承担（STJ第478号摘要）
  - 注意：物业可能有并行催收诉讼

水费/排污费:
  - 向特许经营公司查询（SABESP、CEDAE、Copasa等）
  - 可能导致停水——恢复供水需额外费用
  - 通常：个人债务，不随物转移（但各州规定不同）

电费:
  - 个人债务（不随物转移）
  - 确认是否已停电

水费欠款估算:            R$ ____________
```

## 类别3 — 运营风险

#### 3.1 拍卖无法完成的风险

**竞拍成功后，拍卖可能因以下原因被撤销：**

| 事件 | 发生时限 | 概率 | 后果 |
|--------|-------------------|---------------|-------------|
| 债务人在签署笔录前还款 | 签署前任何时间 | 低-中 | 拍卖撤销，款项退还 |
| 有中止效力的异议 | 至拍卖笔录签署前 | 低 | 拍卖中止 |
| 10日内主张无效 | 竞拍后10天内 | 低 | 拍卖无效 |
| 事后认定为家庭住宅 | 10天后——独立诉讼 | 极低 | 辩护复杂 |
| 第三人异议之诉 | 任何时间（诉讼时效内） | 极低 | 需司法辩护 |

#### 3.2 欺诈或操纵风险

**拍卖中的警示信号：**
- ⚠️ 拍卖师未在商业登记处注册
- ⚠️ 不明在线平台，无核实CNPJ
- ⚠️ 评估价与市场价严重偏离（极端情况）
- ⚠️ 公告发布期限短于法定要求
- ⚠️ 标的描述模糊，未提供登记册号
- ⚠️ 要求在查看文件前缴纳保证金

**防护措施：**
- 在州商业登记处网站核实拍卖师
- 在法院系统（e-SAJ、PJE、SEEU）确认案件
- 未在司法案件中确认前绝不付款

## ITBI按VMP征收的风险（而非按成交价）

**问题：**
许多城市按**参考评估值**（VMP）征收ITBI，而非按实际竞拍价。这可能导致成本增加最多3倍。

**示例：**
- 房产竞拍价 R$ 200.000
- 参考评估值（市政厅）: R$ 400.000
- ITBI 3%按竞拍价: R$ 6.000
- ITBI 3%按评估值: R$ 12.000 ← 市政厅实际收取

**异议法律依据：**
- STJ第1.113号议题：ITBI应按实际交易价格征收
- 司法拍卖：拍卖确认书是权属凭证——价值=竞拍价
- 非司法拍卖：以竞拍价为价值的契约是权属凭证
- 可通过行政异议或安全令状提出异议

**建议：** 按VMP预算ITBI（悲观场景）并计入总成本。
若能按竞拍价缴纳，则为额外节省。

## 转售资本利得税风险

- 转售利润按15%征税（利得≤R$ 5M）
- 取得成本：竞拍价+ITBI+佣金+登记费+有票据的装修费
- 免税：每5年出售唯一房产，金额≤R$ 440.000
- 免税：180天内再投资于另一套住宅房产
- **提示：** 所有支出务必保留发票（装修、合规化），用于抵扣资本利得

## 竞拍人受保护吗？

**一般规则（CPC第903条第5款）：**
公开拍卖中的竞拍视为受司法保护的取得。
善意竞拍人免受前手欺诈性转让的影响。

**但需注意以下场景：**

| 场景 | 竞拍人风险 | 保护 |
|---------|----------------------|----------|
| 债务人在查封前出售房产 | 极低 | CPC第903条保护竞拍人 |
| 第三人声称在查封前已购买 | 中 | 取决于登记+善意 |
| 第三人对房产提起取得时效诉讼 | 高 | 权属冲突——可能被撤销 |
| 债务人赠予亲属（欺诈债权人） | 低 | 公开拍卖中的竞拍人受保护 |

**必须核实：**
- 民事分配处证明：查询是否有第三人就该房产提起物权诉讼（取得时效、返还请求）
- 如存在第三人主张房产的诉讼：高风险——应避免

## 投资压力测试方法:

```
乐观场景（概率20%）:
  - 3个月内按VMP售出
  - 无额外腾退费用
  - ITBI按竞拍价征收
  - ROI: ___ %

基准场景（概率50%）:
  - 6个月内按VMP折价10%售出
  - 协商腾退费用（R$ 5k）
  - ITBI按VMP征收
  - ROI: ___ %

悲观场景（概率25%）:
  - 12个月内按VMP折价20%售出
  - 交付诉讼（R$ 15k + 6个月）
  - 需要装修（R$ 30k）
  - ROI: ___ %

灾难场景（概率5%）:
  - 拍卖被撤销（损失定金，但款项退还）
  - 或：24个月内无法售出（资金锁死）
  - 或：隐性欠款吞噬利润（高额物业费）
  - ROI: ___ %（可能为负）

加权ROI（数学期望）:
= (乐观ROI × 0,20) + (基准ROI × 0,50) + (悲观ROI × 0,25)
  + (灾难ROI × 0,05)

加权ROI > CDI → 竞拍
加权ROI < CDI → 不值得冒险
```

## 风险术语表

| 术语 | 定义 |
|-------|-----------|
| 随物转移 | 跟随房产的义务（房产税、物业费）——不因出售而消失 |
| 法律风险 | 拍卖被撤销、无效或异议的可能性 |
| 运营风险 | 执行中的困难（腾退、装修、合规化） |
| 税务风险 | ITBI按VMP还是竞拍价征收；转售资本利得税 |
| 机会成本 | 将资金锁定在此操作中而放弃的收益 |
| 压力测试 | 投资最坏场景的模拟 |
| 尽职调查 | 竞拍前的全面调查 |
| VaR（风险价值） | 不利场景下的最大估计损失 |
| 安全边际 | 总成本与市场价值之间的财务缓冲 |
| 执行欺诈 | 传票后将财产转让以阻挠执行（CPC第792条） |
| 加权ROI | 考虑各场景概率的预期回报 |

## 治理

本技能实施以下治理策略：

- **action_log**: 风险分析通过log_action记录，确保完整审计

附加策略：
- **责任方：** Leiloeiro IA生态系统
