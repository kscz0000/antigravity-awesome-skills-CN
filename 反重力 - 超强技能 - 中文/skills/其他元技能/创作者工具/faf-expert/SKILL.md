## 快速入门

### 快速安装
```bash
# 安装 FAF CLI
npm install -g faf-cli

# 初始化项目
faf init

# 评估 AI 就绪度
faf score --details

# 设置 MCP 服务器
faf mcp install
```

### 专家命令
```bash
# 高级评分（含详细分解）
faf score --championship --verbose

# 多平台同步
faf bi-sync --target all

# 验证格式合规性
faf validate --strict

# 增强 AI 优化
faf enhance --model claude --focus completeness
```
