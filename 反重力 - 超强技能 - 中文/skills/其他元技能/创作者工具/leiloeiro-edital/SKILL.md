# 拍卖公告技能 — 拍卖公告专家分析

## 概述

司法与非司法拍卖公告的分析与审计。隐藏风险、危险条款、欠款、占用人及机会评级。

## 何时使用此技能

- 用户提到"拍卖公告"或相关话题时
- 用户提到"公告分析"或相关话题时
- 用户提到"公告风险"或相关话题时
- 用户提到"公告条款"或相关话题时
- 用户提到"拍卖房产欠款"或相关话题时
- 用户提到"解读公告"或相关话题时

## 何时不使用此技能

- 任务与拍卖公告无关
- 更简单、更专用的工具即可处理请求
- 用户需要的是无领域专业知识的通用协助

## 工作原理

你是一位**拍卖公告鉴定专家**，能够提取并分析任何司法或非司法拍卖公告中的每一条关键条款。

---

## 公告分析规程

收到公告（或其中信息）后，始终执行以下8个模块：

---

## 模块1 — 识别与定性

**从公告中提取：**
- 案件编号（司法拍卖时）
- 拍卖师姓名及资质（CRC/商业登记）
- 拍卖平台（现场/线上 — 哪个门户）
- 首场拍卖的日期、时间和地点
- 第二场拍卖的日期、时间和地点
- 委托方（谁委托拍卖）：银行、执行方、公证处
- 类型：司法拍卖（CPC）或非司法拍卖（Lei 9.514/97）

**初步分类：**
```
Tipo: [ ] Judicial  [ ] Extrajudicial - Alienação Fiduciária  [ ] Venda Direta
Modalidade: [ ] 1º Leilão  [ ] 2º Leilão  [ ] Único
Plataforma: ___________
Data/Hora: ___________
```

---

## 模块2 — 房产描述与位置

**核查：**
- 完整精确的地址（CEP、门牌号、补充信息）
- 类型：住宅、公寓、地块、商业用房、仓库、农地
- 总面积和建筑面积（与产权登记比对）
- 产权登记号及登记处
- IPTU编号/市政编码
- 公告中描述的建筑标准
- 声明的保存状况
- 是否含车位（若有，是独立产权还是附属产权？）

**警示：**
- ⚠️ 公告面积 ≠ 产权登记面积 → 可能存在不规范
- ⚠️ 无产权登记号 → 竞拍前必须调查
- ⚠️ 描述模糊（"位于以下地址的房产..."） → 要求提供评估报告

---

## 模块3 — 评估价值与最低出价

**提取并计算：**
```
Valor de Avaliação (VAN):          R$ _____________
Lance Mínimo 1º Leilão:            R$ _____________  (= VAN em judicial / VAN em extraJ)
Lance Mínimo 2º Leilão:            R$ _____________  (50% VAN em judicial / dívida em extraJ)
Data da Avaliação:                 _______________
Avaliador responsável:             _______________
```

**折价分析：**
- 首场最低出价相对VAN的折价：____%
- 第二场最低出价相对VAN的折价：____%
- 实际折价（相对估算市场价值）：____%

**警示：**
- ⚠️ 评估超过12个月 → 过期风险 — 可申请重新评估（CPC第873条）
- ⚠️ VAN远低于市场 → 调查评估报告或是否存在利益输送
- ⚠️ VAN远高于市场 → 首场不会成交；等待第二场
- ⚠️ 非司法拍卖第二场：最低出价=债务额 → 可能远低于市场价值（绝佳机会）

---

## 模块4 — 房产状况（占有与占用）

**在公告中核查：**
- [ ] 房产空置（可直接使用）
- [ ] 房产被被执行人/债务人占用
- [ ] 房产被第三方占用（租户或侵占人）
- [ ] 公告未说明占用情况（⚠️ 风险）

**占用影响：**

| 情况 | 风险 | 预估费用 | 周期 |
|------|------|----------|------|
| 空置 | 低 | 零 | 即时 |
| 债务人配合 | 中低 | 协商 | 30-90天 |
| 债务人抗拒 | 高 | R$ 5-15k（诉讼） | 6-18个月 |
| 有合同租户 | 中 | 赔偿 | 3-6个月 |
| 第三方侵占 | 高 | 返还占有诉讼 | 6-24个月 |

**若被占用，核查：**
- 公告是否规定了谁负责腾退？
- 是否已签发占有移交禁令？
- 竞得人是否有银行/债权人的法律协助？
- 租赁是否登记在产权上？（有效期内的租赁可能必须被尊重）

---

## 5.1 欠款责任 — 公告怎么说？

**逐一核查：**
- [ ] IPTU — 欠款金额及责任方
- [ ] 物业管理费 — 欠款金额及责任方
- [ ] 垃圾费、公共照明费
- [ ] 水/排污欠费（SABESP、CEDAE等）
- [ ] 市政改善费和工程费

**条款解读：**

| 公告原文 | 解读 | 风险 |
|----------|------|------|
| "房产按现状出售" | 欠款可能随房产转移 | 高 |
| "无负担" | 竞得人不承担责任 | 低 |
| "欠款由竞得人承担" | 你支付全部 | 高 — 需量化 |
| "公告对欠款未提及" | 适用propter rem规则 | 中 |
| "欠款从拍卖所得中支付" | 法官预留资金 | 低 |

**务必量化：**
竞拍前，获取：
1. IPTU欠款证明（市政厅）
2. 物业管理费欠款明细（业委会/管理公司）
3. 水/燃气欠费声明

## 5.2 产权登记中的物权负担

**在公告和产权登记中核查：**
- [ ] 抵押（哪个银行、金额、日期）
- [ ] 此前的信托让与担保（查封之前）
- [ ] 登记的用益权（用益权人是谁？预计存续期？）
- [ ] 地役权（通行权、公共设施权）
- [ ] 不可转让条款（含此条款的继承）
- [ ] 海岸土地批租 — 海岸地块（laudêmio：每次转让5%）
- [ ] 此前的查封（另案 — 优先权如何？）

**特别注意：**
- 终身用益权 → 用益权人在世期间竞得人无使用权
- 海岸批租 → 向SPU支付laudêmio + 年度foro
- 查封前的抵押 → 核查是否在执行程序中被传唤（代位）

---

## 模块6 — 付款条件

**从公告中提取：**
- 接受的付款方式（现金、TED、支票、信用证）
- 一次性付款期限
- 分期付款可能性 — CPC第895条：
  - 竞拍时支付25%
  - 余款30天内付清（或按裁定）
- 是否接受银行贷款？哪家银行？
- 拍卖师佣金：____%（标准：5%）
- 佣金按出价计算还是另计？
- ITBI（市政转让税）：___%（各市不同 — 平均2-3%）
  - 圣保罗：3%
  - 里约热内卢：3%
  - 贝洛奥里藏特：3%
- 登记和公证费用：_____（登记处价目表）

**总成本估算：**
```
Lance arrematado:                  R$ _____________
(+) Comissão leiloeiro (5%):       R$ _____________
(+) ITBI (2-3%):                   R$ _____________
(+) Registro cartório:             R$ _____________
(+) Advogado (imissão, se necessário): R$ ________
(+) Débitos IPTU acumulados:       R$ _____________
(+) Débitos condomínio:            R$ _____________
(+) Obras/adequações estimadas:    R$ _____________
= CUSTO TOTAL REAL:                R$ _____________
```

## 模块7 — 文件与法律合规性

**核查公告合规事项：**

**a) 公告发布（CPC第887条 / Lei 9.514第27条）：**
- [ ] 是否在官方公报发布？
- [ ] 是否在大型流通报纸发布？
- [ ] 是否在法院门户发布（司法拍卖时）？
- [ ] 是否满足至少5天的提前期？

**b) 必要传唤（CPC第889条）：**
- [ ] 债务人/信托人是否被传唤？
- [ ] 配偶/伴侣是否被传唤？
- [ ] 抵押权人是否被传唤（若有）？
- [ ] 用益权人是否被传唤（若有）？
- [ ] 优先权人是否被传唤？

**c) 合格拍卖师：**
- [ ] 姓名及商业登记注册号
- [ ] 法院认证（司法拍卖时）
- [ ] 非司法拍卖：由信托债权人指定的拍卖师

**d) 公告完整性（CPC第887条§1º）：**
- [ ] 财产描述
- [ ] 评估价值
- [ ] 现有负担
- [ ] 付款条件
- [ ] 拍卖地点、日期和时间

## 附加模块 — 直接出售公告（CEF、BB、Santander）

银行直接出售公告的格式与司法拍卖不同。要点如下：

## Caixa线上出售（caixavbr.com.br）

**CEF公告结构：**
```
1. Identificação do lote (número, endereço, matrícula)
2. Valor mínimo de venda (VMAV — Valor Mínimo de Aquisição e Venda)
3. Forma de pagamento aceita:
   - À vista (desconto de 5-10%)
   - Financiamento pela própria CEF (até 80% do VMAV)
   - FGTS: pode ser usado para parte do pagamento
4. Estado do imóvel: "no estado em que se encontra"
5. Responsabilidade por débitos: geralmente a cargo do arrematante
6. Comissão do leiloeiro/intermediário: 5%
7. Prazo para desocupação (se ocupado): responsabilidade do comprador
```

**CEF特色：**
- 可使用FGTS（需满足SFH要求）
- 本行贷款最长360个月
- 一次性付款额外折扣
- PMCMV/MCMV房产：价格亲民，需求旺盛
- 公告无需遵守CPC（非司法拍卖）

## BB / Santander / Itaú直接出售

**共同特征：**
- 简化公告（不遵循CPC）
- 售价由银行确定（内部评估）
- 中介佣金：5-6%
- 可能提供本行贷款
- 房产"按现状及含负担出售"
- **注意：** "含负担"= 竞得人承担一切（IPTU、物业费、修缮、占用）

## 直接出售专项清单

- [ ] VMAV相对市场是否合理？（查询ZAP/VivaReal）
- [ ] 是否接受贷款？比例多少？
- [ ] 是否接受FGTS？
- [ ] 报价和付款期限
- [ ] 中介佣金（内含还是另计）
- [ ] 是否明确IPTU/物业费欠款责任
- [ ] 房产列为占用还是空置
- [ ] 是否有可用的勘察报告（银行照片/评估）

## 治理

本技能实现以下治理策略：

- **action_log**：每次公告分析通过log_action记录，确保可追溯
- **rate_limit**：通过集成的check_rate控制频率
- **requires_confirmation**：裁定"不建议竞拍"时向用户发出确认请求
- **warning_threshold**：风险评分>10/14时触发warning_threshold自动警报

附加策略：
- **负责人：** Leiloeiro IA生态系统
- **范围：** 司法与非司法拍卖公告的专家分析
- **限制：** 分析基于提供的信息，无法访问司法案件
- **审计：** 由skill-sentinel验证
- **敏感数据：** 不存储已分析的公告数据

## 公告常见陷阱 — 十大排行

| # | 陷阱 | 如何识别 | 影响 |
|---|------|----------|------|
| 1 | "按现状及含负担出售" | 仔细阅读责任条款 | 意外欠款 |
| 2 | 公告对占用情况沉默 | 未提及是否占用/空置 | 腾退费用 |
| 3 | 评估超过3年 | 公告中的评估报告日期 | 价值失真 |
| 4 | 未告知高额物业费 | 未提及费用标准 | 固定支出偏高 |
| 5 | 海岸带内房产 | 描述中出现"批租"或"海岸地块" | laudêmio 5% |
| 6 | 车位产权独立分离 | 公告写"不含车位"或"车位不包含" | 丢失车位 |
| 7 | 建筑面积未登记 | 产权面积小于实际面积 | 合规化费用 |
| 8 | 第二场=债务金额（非市场价） | 非司法拍卖 — 最低价可能仅为VMP的20% | 看似划算，但需核查欠款 |
| 9 | 佣金未计入出价 | "佣金由竞得人额外承担" | 额外5% |
| 10 | 分期利息极高 | 阅读分期条款（IGP-M、IPCA、月息1%） | 隐性融资成本 |
