# 决策框架 — 能源采购

本参考提供详细的决策树、评估矩阵、财务模型和战略框架，覆盖电力与天然气采购、费率优化、需量电费管理、PPA 评估、对冲策略设计和多设施组合优化。代理在需要做出或建议细致入微的能源采购决策时按需加载。

所有阈值、价格假设和市场基准反映美国商业和工业电力与天然气市场。请根据地区市场、当前远期曲线和设施特定的费率结构进行调整。

## 1. 采购策略选择

### 1.1 采购前情报收集

在任何采购决策之前——合同续签、新设施接入或中期重组——汇总完整的数据包。

#### 数据汇编清单

| 数据项 | 来源 | 用途 |
|-----------|--------|---------|
| 36 个月的 15 分钟间隔数据（kWh 与 kW） | 公用事业电表数据 / MDM 系统 | 负荷曲线分析、峰值识别 |
| 当前费率表及所有适用附加费 | 公用事业费率手册 / 州 PUC | 基准成本结构 |
| 当前供应合同条款、到期日与自动续签条款 | 合同档案 | 时间表与约束 |
| 相关枢纽 12、24、36 个月远期能源曲线 | ICE、CME、经纪人报价 | 定价评估的市场基准 |
| 容量市场拍卖结果（PJM RPM、ISO-NE FCA） | ISO 公告 | 未来容量电费预测 |
| 设施峰值负荷贡献（PLC）或装机容量（ICAP）标签 | 公用事业 / ISO 结算数据 | 容量电费敞口 |
| 设施所在地的历史天气数据（HDD/CDD） | NOAA / 气象服务 | 消耗的天气归一化 |
| 州 PUC 待定的公用事业费率案例 | 州 PUC 案卷检索 | 监管风险评估 |
| 公司可持续发展目标与时间表 | 可持续发展团队 | 可再生能源采购要求 |
| 需求侧投资的资本预算可用性 | 财务团队 | 需量电费缓解的投资约束 |

### 1.2 固定价格 vs. 指数 vs. 分块指数 决策树

对每个设施或组合段独立使用该决策树——没有一种策略适合所有站点。

```
START: 组织对能源成本波动的承受能力如何？

├── 预算波动 >10% 触发高管升级
│   ├── 合同期限 ≤ 24 个月？
│   │   └── 是 → 固定价格全需求
│   │       - 接受风险溢价（高于远期曲线 5-12%）
│   │       - 谈判数量容差带（±10-15%）
│   │       - 确保合同包含用途变更条款
│   │   └── 否（>24 个月）→ 固定价格 + 年度价格重置
│   │       - 锁定第 1 年为固定，第 2-3 年按公式（远期 + 加价）
│   │       - 这限制了供应商的长期风险溢价
│
├── 预算波动 5-10% 可承受
│   ├── 设施负荷率 > 0.70？
│   │   └── 是 → 分块指数
│   │       - 购买固定区块 = 基本负荷的 70-80%
│   │       - 剩余 20-30% 按指数浮动（日前或实时）
│   │       - 区块形状匹配基本负荷模式（ATC vs. 仅高峰）
│   │   └── 否（负荷率 < 0.70）→ 成型分块指数
│   │       - 仅购买高峰区块（匹配生产计划）
│   │       - 谷段与肩段按指数浮动
│   │       - 用 TOU 指数产品补充谷段
│
├── 组织可承受 >15% 波动（能源占 COGS <5%）
│   ├── 具备内部监控批发市场的能力？
│   │   └── 是 → 指数定价 + 金融对冲
│   │       - 基础产品：实时或日前指数 + 供应商加价
│   │       - 分层金融对冲：为高峰月份购买看涨期权
│   │       - 通过期权设置价格上限（$X/MWh 上限）
│   │   └── 否 → 指数 + 价格上限产品
│   │       - 供应商提供带合同上限的指数定价
│   │       - 上限溢价通常为高于远期曲线 $3-7/MWh
│   │       - 比单独管理金融对冲简单
```

### 1.3 分层采购方法论

分层消除了单点市场时机风险。方法论：

**步骤 1：确定对冲区间。**
大多数工商业买家在交付期前 18-36 个月分层。对于 2028 年 1 月的起始日，从 2026 年 7 月开始购买批次。

**步骤 2：设定批次数。**
标准做法：

| 批次数 | 购买频率 | 每批电量 | 适用场景 |
|----------|-----------------|-------------------|----------|
| 4 | 季度 | 25% | 默认做法，平衡性好 |
| 6 | 双月 | ~17% | 大型组合，更高粒度 |
| 8 | 月度（最后 8 个月） | 12.5% | 积极的美元成本平均 |
| 12 | 月度 | ~8% | 配备专职采购人员的超大型组合 |

**步骤 3：执行规则。**
- 在预定日期按当时市场价格执行每批——不要试图在批次窗口内择时。
- 例外：若远期曲线跌入 5 年区间的底部 20 分位，立即加速购买 2 批（"逢低买入"规则）。
- 例外：若远期曲线冲入顶部 20 分位，将本批延后 30 天（跳过并在之后追赶）。
- 切勿连续延后超过 2 批——滚动延后会让你毫无对冲。

**步骤 4：记录与报告。**
维护采购日志，记录：批次日期、采购电量、锁定价格、执行时远期曲线价格、累计加权平均价格、剩余未对冲头寸。每季度向财务报告。

**示例——10 MW 峰值负荷，年消耗 60M kWh：**

```
交付年：2028
对冲起始：2026 年 7 月
批次数：6（双月，每批约 10M kWh）

批次 1（2026 年 7 月）：10M kWh @ $44.50/MWh  — 远期为 $45.20
批次 2（2026 年 9 月）：10M kWh @ $42.80/MWh  — 远期为 $43.10
批次 3（2026 年 11 月）：10M kWh @ $46.30/MWh  — 远期为 $46.30
批次 4（2027 年 1 月）：10M kWh @ $41.20/MWh  — 远期为 $41.50（逢低买入规则：
                                                 同时提前执行批次 5）
批次 5（2027 年 1 月）：10M kWh @ $41.40/MWh  — 从 3 月加速执行
批次 6（2027 年 5 月）：10M kWh @ $43.80/MWh  — 远期为 $44.00

加权平均：$43.33/MWh
执行价格区间：$41.20 - $46.30（$5.10 价差）
若一次性在 2026 年 7 月锁定：$44.50/MWh → 分层节省 $1.17/MWh = $70,200
```

### 1.4 解除管制市场的 RFP 流程

#### 时间表与阶段

| 阶段 | 持续时间 | 关键活动 |
|-------|----------|---------------|
| RFP 前分析 | 2-3 周 | 负荷数据汇编、费率分析、市场基准、可持续要求定义 |
| RFP 设计 | 1-2 周 | 模板创建、供应商长名单开发、评标标准权重 |
| RFP 分发 | 1 周 | 发给 5-8 家合格的 REP，回应澄清问题 |
| 投标窗口 | 2-3 周 | 供应商基于你的间隔数据和要求开发报价 |
| 投标评估 | 1-2 周 | 总成本建模、信用评估、合同审查 |
| 谈判 | 1-2 周 | 短名单至 2-3 家，谈判条款，最终定价 |
| 授标与签署 | 1 周 | 签署合同，通知公用事业供应商切换（可能需要 30-60 天准备期） |
| **总计** | **9-14 周** | |

#### 供应商评估评分矩阵

| 标准 | 权重 | 评分指引 |
|-----------|--------|---------------|
| 总成本（能源 + 加价 + 成型溢价） | 35-45% | 最低总成本 = 100 分。每高于最低 1% = -5 分。跨 3 个价格情景建模。 |
| 信用质量 | 15-20% | 投资级（S&P BBB- 或以上）= 100 分。低于投资级 = 50 分。无评级 / 私人 = 有母公司担保 70 分，无 30 分。 |
| 合同灵活性 | 10-15% | 数量容差 ±15% = 100。数量容差 ±5% = 50。无容差 = 0。可提前终止 = +20 分。用途变更条款 = +15 分。 |
| 可持续服务 | 10-15% | 来自具名项目的捆绑 REC = 100。可提供非捆绑 REC = 60。无 REC 选项 = 0。碳报告支持 = +20 分。 |
| 市场情报与咨询 | 5-10% | 专属客户经理 + 定期市场更新 = 100。仅客户经理 = 50。呼叫中心支持 = 0。 |
| 运营能力 | 5-10% | EDI/API 计费集成 = 100。仅电子发票 = 60。纸质账单 = 0。多站点合并计费 = +20 分。 |

#### 投标对比模板

对每个站点，建模每个供应商方案的年度成本：

```
年度成本 = Σ(每小时电量 × 每小时电价) + 固定费用 + REC 成本 + 加价费用

其中每小时电价取决于产品结构：
  固定价格：所有时段均为合同电价
  分块指数：区块电量按区块价，超出部分按指数价
  指数：(负荷区日前或实时 LMP) + 供应商加价
```

始终在三种远期价格情景下建模：基准情景（当前远期曲线）、低情景（远期 - 20%）、高情景（远期 + 30%）。在基准下看起来最便宜的指数产品供应商，在高情景下可能最贵。

## 2. PPA 评估框架

### 2.1 实物 PPA 评估

实物 PPA 涉及直接电力交付，适用于以下情况：
- 你的负荷与项目位于同一 ISO
- 你希望获得来自特定具名设施的电力与 REC
- 你能够管理调度与平衡的运营复杂性

#### 财务建模框架

**步骤 1：建立基准（无 PPA 情景）。**
使用前 1-5 年的远期曲线和长期价格递增假设（通常 2-3%/年）预测 PPA 期内能源成本，第 6 年起。

**步骤 2：建模 PPA 现金流。**

```
第 N 年 PPA 净价值 = (枢纽市场价格 - PPA 执行价格) × 预期发电量
                     - 基差成本（枢纽到负荷区）
                     - 削减成本（预期削减 MWh × 执行价格）
                     - 平衡成本（覆盖 PPA 未覆盖的残余负荷）
                     + REC 价值（若 REC 原本需单独购买）
```

**步骤 3：敏感性分析——至少运行以下情景：**

| 情景 | 市场价格假设 | 发电量假设 | 基差假设 |
|----------|----------------------|----------------------|-----------------|
| 基准 | 当前远期曲线 + 2.5%/年 递增 | 开发商 P50 估计 | 5 年历史平均基差 |
| 牛市 | 远期 + 4%/年 递增 | P50 发电量 | 基差收窄 20% |
| 熊市 | 远期 + 1%/年 递增 | P75 发电量（更低） | 基差扩大 30% |
| 压力 | 5 年内价格持平，然后 2%/年 | P90 发电量（远低） | 基差扩大 50% |

**步骤 4：在各情景下计算 NPV、IRR 和平准化度电成本（LCOE）。**
如果 PPA 在基准和牛市情景下 NPV 为正，熊市情景下的损失可承受（通常为中型工商业买家 PPA 期内累计 <$2M），则经济上可行。

### 2.2 虚拟 PPA（VPPA）评估

VPPA 是金融工具——无实物电力交付。关键风险不同：

#### 基差风险分析

基差风险是 VPPA 的主要金融风险。源于发电商在其节点结算而你的负荷在负荷区价格结算。

**量化方法：**

1. 从 ISO 获取发电商节点与你的负荷区 3-5 年的小时 LMP 数据。
2. 计算小时基差：负荷区 LMP - 发电商节点 LMP。
3. 筛选到发电商正在发电的小时（光伏：日照时段；风电：使用历史发电曲线）。
4. 计算发电量加权平均基差。
5. 建模基差对 PPA 结算的影响：

```
年度基差成本 = Σ(每小时基差 × 每小时预期发电量)

若发电量加权平均基差 = $5/MWh，年发电量 = 200,000 MWh：
年度基差成本 = $1,000,000/年
15 年 PPA 期内：$15M 基差成本（未折现）
```

**基差风险的红旗：**
- 基差价差 > $8/MWh 发电量加权平均 → 高风险，谈判基差对冲或拒绝
- 基差波动性（标准差）> $15/MWh → 难以预测，难以预算
- 基差趋势在历史区间内持续扩大 → 结构性阻塞，可能恶化
- 发电商位于已知输电约束后方 → 随着该区域新增发电，阻塞将加剧

#### 削减风险分析

当 ISO 因输电约束或供过于求下令发电商降低出力时，发生削减。

| ISO | 技术 | 典型削减 % | 趋势 |
|-----|-----------|----------------------|-------|
| ERCOT | 风电（西德州） | 3-8% | 随新增风电持续增加 |
| ERCOT | 光伏 | 1-3% | 较低但持续增加 |
| CAISO | 光伏 | 5-12%（春季） | 因鸭形曲线持续增加 |
| CAISO | 风电 | 1-3% | 稳定 |
| PJM | 风电 | <1% | 极小 |
| PJM | 光伏 | <1% | 极小 |
| MISO | 风电 | 2-5% | 中等，取决于区域 |
| SPP | 风电 | 3-7% | 西部区域持续增加 |

**合同保护：** 谈判一个削减阈值（例如，前 5% 由开发商承担风险）和超出削减的补偿机制（开发商提供替代 REC 或价格调整）。在 VPPA 上永远不要接受"买方承担全部削减风险"——这将一种买方无法管理或影响的风险转嫁给买方。

#### 信用与会计要求

| 要求 | 详情 |
|-------------|---------|
| ISDA 主协议 | VPPA 必需。谈判信用阈值、追加保证金条款和终止价值。 |
| 信用支持 | 投资级：通常前 $5-10M 名义价值无需抵押。低于投资级：2-3 年潜在负结算的信用证或母公司担保。 |
| 会计处理 | 若 VPPA 满足有效性测试要求，可适用套期会计（ASC 815）。若无套期会计，盯市损益通过 P&L 流转，造成盈利波动。请尽早咨询财务和会计部门。 |
| 董事会 / CFO 批准 | VPPA 是多年期金融承诺。大多数组织对 >$10M 名义价值或 >10 年期的承诺要求董事会批准。呈现为能源成本管理工具，而非投机头寸。 |

### 2.3 实物 vs. 虚拟 PPA 决策矩阵

| 因素 | 倾向实物 PPA | 倾向虚拟 PPA |
|--------|-------------------|-------------------|
| 负荷位置 | 与可用项目同一 ISO | 负荷位于管制市场或附近无项目 |
| 能源供应 | 需要实物电力（替代公用事业供应） | 已有零售供应合同 |
| 可持续目标 | 希望从特定设施获得捆绑电力 + REC | 仅需 REC 用于范围 2 报告 |
| 运营能力 | 具备能源调度和平衡资源 | 无能源交易或调度人员 |
| 资产负债表 | 希望避免金融衍生品分类 | 可接受 ISDA 和盯市 |
| 信用画像 | 低于投资级（实物可能需要更少信用支持） | 投资级（可高效提供抵押） |
| 监管环境 | 解除管制市场有零售选择 | 管制市场（VPPA 可能是唯一增加性的选项） |

## 3. 需量电费优化

### 3.1 负荷分析方法论

**步骤 1：下载 15 分钟间隔数据。**
从公用事业或你的电表数据管理系统获取至少 12 个月的 15 分钟 kW 需量数据。对于具备分项计量的设施，除主电表外，还需获取系统级（HVAC、生产、压缩空气）的间隔数据。

**步骤 2：识别峰值需量时段。**
按 kW 降序排序所有 15 分钟时段。聚焦前 50 个时段（全年所有时段的顶部 0.15%）。这些时段决定你的需量电费。

**步骤 3：刻画峰值驱动因素。**
对前 50 个时段中的每个时段，识别：
- 日期和一天中的时间
- 周几
- 室外温度（HVAC 负荷的代理指标）
- 生产计划（生产线是否在运行？）
- 任何异常事件（设备启动、测试、维护）

**制造设施的典型发现：**

| 峰值驱动因素 | 前 50 中出现频率 | 根本原因 |
|------------|---------------------|------------|
| 早晨爬坡（6-9 点） | 30-50% | HVAC、压缩机和生产线同时启动 |
| 炎热的下午（2-5 点） | 20-35% | HVAC 满载与生产峰值重合 |
| 维护后设备启动 | 10-20% | 大型电机同时启动的浪涌电流 |
| 测试 / 调试 | 5-10% | 在高峰时段测试新设备 |

**步骤 4：计算峰值时段的需量电费成本。**

```
月度需量电费 = 峰值 kW × 需量电费费率（$/kW）

若正常运行峰值为 4,000 kW，实际峰值为 4,800 kW：
超量峰值成本 = (4,800 - 4,000) × $15/kW = $12,000/月

按 80% 棘轮：
未来 11 个月的最低计费需量 = 4,800 × 0.80 = 3,840 kW
若下月正常峰值降至 3,500 kW，仍按 3,840 kW 计费
年度棘轮成本 = (3,840 - 3,500) × $15/kW × 11 个月 = $56,100
```

### 3.2 削峰投资回报率框架

#### 电池储能系统（BESS）

**容量确定方法：**
1. 确定目标削峰量（kW）。
2. 计算所需能量容量：目标 kW × 峰值事件持续时间。
   对于需量电费管理，1-2 小时的持续时间通常足够。
3. 应用往返效率（锂离子 88-92%）：将电池容量放大 10% 以容纳效率损失。

**示例——制造工厂 500 kW 削峰：**

```
目标削减：500 kW
峰值事件持续时间：2 小时（基于间隔数据分析）
电池规模：500 kW / 1,000 kWh（含 10% 效率缓冲：500 kW / 1,100 kWh）

安装成本（2025 年）：工商业 BESS $800-$1,200/kWh
总资本：$880,000-$1,320,000（按 1,100 kWh × 中位 $1,000/kWh = $1,100,000）

年度价值叠加：
  需量电费节省：500 kW × $15/kW × 12 个月 = $90,000
  容量标签削减：500 kW × $60/kW-年（PJM 示例）= $30,000
  TOU 电量套利：谷段充电（$0.04/kWh），高峰放电（$0.08/kWh）
    1,100 kWh × $0.04/kWh 价差 × 250 天 × 90% 效率 = $9,900
  需求响应收入：500 kW × $40/kW-年（PJM 经济型 DR）= $20,000

年度总价值：$149,900
简单回本：$1,100,000 / $149,900 = 7.3 年
含 ITC（IRA 独立储能 30%）：回本 = $770,000 / $149,900 = 5.1 年
```

**决策阈值：**
- 回本 < 5 年（含叠加价值 + 激励）：强经济性案例，推进
- 回本 5-7 年：若与可持续目标一致或需量电费持续上涨，则可行
- 回本 7-10 年：边缘，需额外战略理由
- 回本 > 10 年：无监管强制要求时经济性不成立

#### 需求响应项目评估

并非所有 DR 项目都同等。评估以下维度：

| 维度 | 需回答的问题 |
|-----------|-------------------|
| 收入确定性 | 是按容量支付（保证的 $/kW-年）还是按绩效支付（按削减事件付费）？ |
| 调度频率 | 每年多少次事件？最长持续时间？能否在完整持续时间内保持削减？ |
| 基线方法 | 如何计量你的削减？使用 10-of-10 或调整方法计算客户基线负荷（CBL）？计算不当的基线可能低估你的削减并降低支付。 |
| 未能履约的处罚 | 若事件中无法削减怎么办？一些项目按容量支付的 2-3 倍施加处罚。 |
| 与其他项目的交互 | DR 入网是否影响你的容量标签计算？是否与表后发电冲突？ |
| 运营影响 | 你的设施能否在不损害生产质量、安全或客户承诺的情况下削减承诺的 kW？ |

### 3.3 错峰启动协议

单位成本最低的需量电费削减策略——无需资本：

**问题：** 早晨启动期间，HVAC、压缩机、照明和生产设备在 5:30-6:30 之间的同时通电产生需量尖峰。

**解决方案：** 在 60-90 分钟窗口内错峰启动设备：

```
5:00 — 照明（50-100 kW）
5:15 — HVAC 预冷/预热（500-800 kW，30 分钟爬坡）
5:45 — 压缩空气系统（200-400 kW，压缩机分批启动）
6:00 — 生产线 1（300-500 kW）
6:15 — 生产线 2（300-500 kW）
6:30 — 辅助系统、电池充电器、水加热

结果：启动期间峰值从 2,200 kW（同时启动）降至 1,600 kW（错峰）
节省：600 kW × $15/kW × 12 个月 = $108,000/年，零资本成本
```

**实施：** 编程楼宇自控系统（BAS）以强制启动顺序。设置硬性联锁，防止下一系统在前一系统达到稳态前启动。

## 4. 市场分析框架

### 4.1 管制 vs. 解除管制 策略图

| 你的情况 | 主策略 | 次策略 |
|---------------|-----------------|-----------------|
| 管制市场，单一费率表 | 需量电费管理、现场发电、费率表优化 | 游说公用事业绿色费率，评估社区光伏 |
| 管制市场，多个费率选项 | 费率分析以选择最优费率表（TOU vs. 统一 vs. 基于需量） | 负荷转移以利用 TOU 价差 |
| 解除管制，单一站点 | 竞争性供应采购（向 5-8 家 REP 发 RFP） | 分层采购以管理时机风险 |
| 解除管制，多站点同一 ISO | 聚合站点进行组合采购（数量杠杆） | 谈判组合级产品（单一供应商、混合费率） |
| 解除管制，多站点多 ISO | 按 ISO 分别采购（市场结构不同） | 即使合同分开，在供应商谈判中利用总电量 |
| 混合管制/解除管制组合 | 解除管制站点竞争性采购；管制站点需求管理 | 在管制区域寻求监管试点项目 |

### 4.2 远期曲线分析

**远期曲线告诉你的：**
- 未来能源价格的市场共识（已调整风险溢价）
- 季节性价格模式（夏/冬价差）
- 同比价格轨迹（递增或下降）

**远期曲线无法告诉你的：**
- 实际未来现货价格（远期不是预测——含风险溢价）
- 短期价格尖峰（远期是平均值，不是尾部）
- 尚未定价的监管变化、工厂退役或输电新增

**将远期曲线用于采购决策：**

| 远期曲线位置 | 采购行动 |
|-----------------------|-------------------|
| 5 年区间底部 20% | 加速购买——以有利价格锁定更多电量 |
| 第 20-40 分位 | 按计划分层——价格合理 |
| 第 40-60 分位 | 维持默认分层计划 |
| 第 60-80 分位 | 减缓购买——将非关键批次延后 30 天 |
| 5 年区间顶部 20% | 尽可能延后，增加指数敞口，评估金融对冲而非实物锁定 |

### 4.3 容量市场敞口

在有组织的容量市场（PJM、ISO-NE、NYISO），容量电费是重要成本组成部分——$30–$120/kW-年，取决于区域和拍卖结果。

**PJM 可靠性定价模型（RPM）：**
- 拍卖在交付年前 3 年举行（基础剩余拍卖）
- 增量拍卖在临近交付时调整数量
- 你的容量义务基于你的 PLC（峰值负荷贡献）
- PLC 由上一交付年 5 个系统同时峰值小时（5CP）期间你的电表负荷决定

**管理容量敞口：**

1. **跟踪 PJM 系统峰值预警。** PJM 在系统峰值预期时发布"高温预警"和"紧急预警"。在这些时段削减可自由支配负荷，以降低次年的 PLC。
2. **安装峰值通知系统。** 订阅 PJM 需求响应预警。部署负荷削减控制，可在峰值预警后 30 分钟内削减 10-20% 的设施负荷。
3. **表后发电。** 在同时峰值小时期间运行备用发电机会减少你的电表负荷，从而降低 PLC。确保发电机的非紧急运行合规排放许可。
4. **容量标签交易。** 在一些市场，容量义务可通过金融工具交易或抵消。你的供应商可能提供容量标签管理服务。

**示例——容量电费影响：**

```
设施峰值：5,000 kW
PLC（测量自上年 5CP 小时）：4,200 kW
你所在区域的 PJM BRA 清算价格：$85/MW-天

年度容量电费：4,200 kW × $85/MW-天 × 365 / 1,000 = $130,305/年

若你在 5CP 小时削减 500 kW：
降低 PLC：3,700 kW
年度容量电费：3,700 kW × $85/MW-天 × 365 / 1,000 = $114,793/年
节省：$15,512/年，来自 5 小时的负荷削减
```

## 5. 对冲策略设计

### 5.1 工商业买家可用的对冲工具

| 工具 | 复杂度 | 所需资本 | 保护 |
|-----------|-----------|-----------------|------------|
| 固定价格合同（通过 REP） | 低 | 无（嵌入价格中） | 合同电量的完全价格确定性 |
| 区块购买（通过 REP） | 低-中 | 无 | 基本负荷价格确定性；可变负荷暴露 |
| 金融互换（通过经纪商/银行） | 中 | ISDA + 可能的保证金 | 在指定电量上将浮动价格转为固定 |
| 看涨期权（通过经纪商/银行） | 中-高 | 权利金（$/MWh 预付） | 执行价 + 权利金的价格上限；保留无限下行收益 |
| 热率看涨期权 | 高 | 权利金 | 防范气电价格尖峰（当天然气驱动边际电价时有用） |
| 领口期权（卖看跌、买看涨） | 中-高 | 降低权利金（看跌收益抵消看涨成本） | 上下限——限制上行与下行 |

### 5.2 按风险画像的对冲策略

| 风险画像 | 对冲比率 | 工具 | 监控 |
|-------------|-------------|-------------|-------------|
| 保守型（预算确定性至上） | 80-95% 对冲 | 固定价格合同、金融互换 | 每月盯市审查 |
| 稳健型（平衡成本/风险） | 60-80% 对冲 | 分块指数、分层采购 | 每月远期曲线审查，季度对冲调整 |
| 激进型（成本最小化导向） | 30-60% 对冲 | 指数 + 看涨期权防范尾部风险 | 每周市场监控，波动期间每日监控 |
| 投机型（工商业买家从不推荐） | <30% 对冲 | 无保护的指数 | 实时监控（对大多数工商业买家不切实际） |

### 5.3 期权定价与评估

购买看涨期权以封顶指数定价敞口时，评估：

```
期权价值 = Max(0, 现货价 - 执行价) × 电量

成本：每 MWh 权利金 × 合同电量
日前定价 $50/MWh 上限的年度权利金：$2-5/MWh（因市场波动而异）

示例——保护 50,000 MWh 年度指数电量：
  看涨期权执行价：$50/MWh
  权利金：$3/MWh
  权利金总成本：$150,000/年

  若现货平均 $42/MWh：期权到期无价值，总成本 = $42 + $3 = $45/MWh
  若现货平均 $65/MWh：期权支付 $15/MWh，有效成本 = $65 - $15 + $3 = $53/MWh
  若现货尖峰至 $200/MWh（天气事件）：期权支付 $150/MWh，有效上限 = $53/MWh

  最高有效费率：执行价 + 权利金 = $53/MWh，无论市场价格如何
```

**何时使用期权 vs. 固定合同：**
- 期权：当你希望参与下行但防范尖峰时
- 固定合同：当期权权利金超过锁定固定价格的总成本时（这发生在波动率高、期权昂贵时）

## 6. 可持续采购对齐

### 6.1 将采购映射到 RE100 和 SBTi

**RE100 进展计算：**

```
RE% = (采购的可再生 MWh) / (总用电 MWh) × 100

可接受的可再生 MWh 来源（按额外性排序）：
1. 现场发电（最强主张）
2. 新项目的实物 PPA（强额外性）
3. 新项目 REC 的虚拟 PPA（良好额外性）
4. 公用事业绿色费率（因项目设计而异）
5. 非捆绑 REC（最弱主张——RE100 正在收紧要求）
```

**SBTi 路径对齐：**
- SBTi 要求基于定义路径的绝对范围 2 排放减少（通常 1.5°C 对齐为 4.2%/年）。
- 锁定长期可再生采购（PPA），逐年交付减排。
- 避免增加化石依赖的采购策略（无 REC 组件的化石密集型电网的长期固定合同）。

### 6.2 经济高效的可再生采购路径

| 目标 RE% | 最低成本策略 |
|-----------|-------------------|
| 0-25% | 非捆绑国家风电 REC（$1-3/MWh）。最便宜的切入点。 |
| 25-50% | 公用事业绿色费率 + 非捆绑 REC。绿色费率通常溢价 $0.005-$0.015/kWh。 |
| 50-75% | 与新风电/光伏项目的 VPPA。固定成本、长期 REC 供应、额外性。 |
| 75-90% | 实物 PPA 或额外 VPPA 覆盖剩余缺口。条件允许时现场光伏。 |
| 90-100% | 用项目特定 REC 或小型现场装置匹配剩余未对冲负荷。最后 10% 是每 MWh 最昂贵的。 |

## 7. 多设施组合优化

### 7.1 组合聚合策略

**何时聚合：**
- 同一 ISO/公用事业辖区内 3+ 站点
- 总电量 > 20 GWh/年（吸引竞争性供应商关注）
- 站点具有互补负荷曲线（一些夏季峰值，另一些冬季峰值）

**聚合收益：**
- 数量杠杆：比单独站点采购低 5-15% 的供应价
- 负荷多样性：合并组合的负荷率高于单独站点，降低供应商风险溢价
- 行政效率：单一合同、单一发票、单一关系

**何时不要聚合：**
- 站点位于具有不同市场结构的不同 ISO（PJM 和 ERCOT 应分别采购）
- 一个站点有独特要求（例如，需求响应策略所需的实时定价），会约束整个组合
- 站点合同到期日差异巨大（错开到约，避免一次性重新签约风险）

### 7.2 组合级风险指标

在组合级跟踪，而非仅逐站点：

| 指标 | 公式 | 目标 |
|--------|---------|--------|
| 组合对冲比率 | (已对冲 MWh / 总预期 MWh) × 100 | 60-80% |
| 加权平均采购价格 | Σ(站点 MWh × 站点 $/MWh) / 总 MWh | 在组合基准 5% 以内 |
| 供应商集中度 | 最大供应商 MWh / 总 MWh | <50%（避免单一供应商依赖） |
| 合同到期聚类度 | 任何 12 个月内到期的组合 MWh % | <40%（错开到约） |
| 可再生覆盖率 | 可再生 MWh / 总 MWh | 按目标推进 |
| 组合负荷率 | 总 kWh / (站点峰值 kW 总和 × 小时) | 跟踪趋势，越高越好 |

### 7.3 需求侧投资的站点优先级

对于有限的需量电费缓解资本，使用此评分模型对站点排序：

| 因素 | 权重 | 评分 |
|--------|--------|---------|
| 需量电费占总账单比例 | 30% | >35% = 100，25-35% = 70，15-25% = 40，<15% = 10 |
| 峰均比 | 25% | >2.5 = 100，2.0-2.5 = 70，1.5-2.0 = 40，<1.5 = 10 |
| 可用需量削减（kW） | 20% | >1000 kW = 100，500-1000 = 70，200-500 = 40，<200 = 10 |
| 公用事业需量费率（$/kW） | 15% | >$20 = 100，$15-$20 = 70，$10-$15 = 40，<$10 = 10 |
| 容量市场敞口 | 10% | PJM/ISO-NE（高）= 100，NYISO = 70，MISO = 40，无 = 0 |

**投资优先级：最高综合分优先。** 综合分 >80 的站点是电池储能或需求响应的强候选。综合分 <40 的站点需量电费优化潜力有限——应聚焦于供应侧采购。

## 8. 天然气采购

### 8.1 天然气采购结构

工商业消费者（锅炉、CHP、工艺热、备用发电）的天然气采购遵循与电力类似的原则，但市场机制不同。

| 结构 | 描述 | 适用场景 |
|-----------|-------------|-------------|
| 固定价格确定合同 | 合同期内锁定 $/therm 或 $/MMBtu | 预算确定性，大型供热负荷 |
| 指数（月初） | 月度 NYMEX Henry Hub 结算 + 基差 + 加价 | 成本优化，风险承受型买家 |
| 指数（每日） | 每日 Gas Daily 中位价 + 基差 + 加价 | 高灵活性负荷，可中断工艺 |
| 基本负荷区块 + 指数 | 固定区块覆盖基本供热/工艺负荷，指数覆盖可变部分 | 同时具备基本工艺热和天气可变 HVAC 的设施 |
| 摆动合同 | 数量灵活性（提名量的 50-130%） | 天然气消耗高度可变的设施 |

### 8.2 天然气的基差差

天然气价格因交付点而异。Henry Hub（路易斯安那）是基准，但交付成本取决于 Henry Hub 与你的本地城市门站或公用事业交付点之间的基差差。

**常见基差差（近似）：**

| 交付点 | 较 Henry Hub 的典型基差 | 驱动因素 |
|---------------|--------------------------|--------|
| 芝加哥（NGPL 中大陆） | -$0.10 至 +$0.15/MMBtu | 墨西哥湾到中西部管道容量 |
| 纽约（Transco Zone 6 NY） | +$0.50 至 +$3.00/MMBtu | 纽约市入城管道冬季约束 |
| 新英格兰（Algonquin） | +$1.00 至 +$8.00/MMBtu（冬季） | 严重管道约束，与 LNG 竞争 |
| 加利福尼亚（SoCal 边境） | -$0.50 至 +$1.50/MMBtu | 随西海岸供需变化 |
| 阿巴拉契亚（Dominion South） | -$1.50 至 -$0.30/MMBtu | Marcellus 页岩气超供 |
| 德州（HSC） | -$0.05 至 +$0.20/MMBtu | 接近生产，基差极小 |

**关键洞察：** 新英格兰的设施在指数定价下面临的冬季风险与德州的设施截然不同。寒潮期间新英格兰的基差可超过 $15/MMBtu，将交付天然气成本翻三倍。新英格兰的天然气采购需要通过确定管道容量或 LNG 备用进行冬季对冲——无保护的指数定价在该市场是不计后果的。

### 8.3 天然气-电力相互依赖

对于同时具备电力和天然气负荷的设施，识别其耦合：

- **当天然气价格飙升时，电力价格飙升。** 天然气是美国大多数 ISO 的边际发电燃料。Henry Hub 上涨 $2/MMBtu 转化为批发电力价格约上涨 $10-$15/MWh（取决于边际燃气机组的平均热率，通常为 7,000-8,000 BTU/kWh）。

- **CHP 经济性取决于天然气价格。** 以 6,500 BTU/kWh 热率发电的 CHP 系统的燃料成本为 $6.50 × 天然气价（每 MWh）。天然气 $3/MMBtu 时，发电成本为 $19.50/MWh。天然气 $8/MMBtu 时，发电成本为 $52/MWh。若你的电网电力成本超过 CHP 发电成本，则运行 CHP。若电网电力降至 CHP 成本以下（例如，在温和天气和低电网需求的春季肩月），考虑关闭 CHP 并从电网购电。

- **双燃料对冲：** 同时对冲天然气和电力时，认识到锁定天然气成本而让电力保持指数（或反之）会创造跨商品基差风险。若天然气价格下跌但电力保持高位（由于输电约束或非天然气发电紧张），你的天然气对冲表现不佳而你的电费仍高。考虑基于相关性对冲两种商品——许多能源供应商提供合并的天然气+电力组合管理。

## 9. 管制市场的费率优化

### 9.1 费率表选择

在管制市场，可用的费率选项看似有限，但在费率表之间切换可在不改变消耗的情况下节省 5-15% 的总账单。

**步骤 1：识别你的需量水平和电压可用的费率表。**
大多数公用事业为大型工商业客户提供 2-4 个费率选项：
- 标准需量费率（统一电量 + 需量电费）
- 分时电价（低段电量低，高段电量和需量高）
- 实时定价试点（如可用）
- 可中断服务费率（成本较低，公用事业可在紧急情况下削减）

**步骤 2：将 12 个月的实际间隔数据针对每个可用费率表建模。**

```
对每个费率表：
  月度成本 = Σ(电量费用组成部分) + 需量电费 + 客户费 + 附加费

其中：
  电量费用组成部分 = Σ(每间隔 kWh × 适用每 kWh 费率)
  需量电费 = 月内最大 15 分钟 kW 间隔 × 需量费率
  对 TOU 费率：可能适用单独的高峰需量电费
```

**步骤 3：比较年度总计。**

| 费率表 | 年度电量 | 年度需量 | 年度固定 | 年度合计 | 较当前 |
|--------------|--------------|---------------|-------------|-------------|-------------|
| 当前（GS-3） | $580,000 | $312,000 | $24,000 | $916,000 | baseline |
| TOU（GS-3-TOU） | $545,000 | $298,000 | $24,000 | $867,000 | -$49,000 (-5.3%) |
| RTP 试点 | $510,000 | $312,000 | $36,000 | $858,000 | -$58,000 (-6.3%) |
| 可中断 | $565,000 | $250,000 | $24,000 | $839,000 | -$77,000 (-8.4%) |

**步骤 4：评估非财务因素。**
- TOU：需要转移负荷的能力或接受更高的高峰成本
- RTP：需要市场监控和对价格波动的承受能力
- 可中断：需要在短通知（通常 30-60 分钟）内削减负荷的能力

### 9.2 费率案例监控与响应

**何时干预费率案例：**

| 影响级别 | 年度成本增加 | 推荐行动 |
|-------------|---------------------|-------------------|
| <$50K | 对大型工商业可忽略 | 仅监控——通过和解跟踪备案 |
| $50K-$200K | 重要但不关键 | 加入现有干预方团体（OIEC 等） |
| $200K-$500K | 显著 | 单独干预，聘请监管顾问 |
| >$500K | 关键 | 全面干预，聘请专家证人，提交费率设计证词 |

**费率案例时间表（典型）：**

```
第 0 月：公用事业向州 PUC 提交费率案例
第 1-2 月：干预方申请参与
第 3-4 月：证据开示（质询、对公用事业的数据请求）
第 5-7 月：干预方证词提交
第 8-9 月：听证
第 10-12 月：PUC 发布命令
第 13-15 月：新费率生效（可追溯至提交日）
```

**在费率案例中挑战什么：**
1. **股权回报率（ROE）：** 公用事业通常请求 10-11% 的 ROE。当前授权 ROE 趋势向下（9-10%）。挑战过高的 ROE 请求。
2. **费率基础新增：** 公用事业在其费率基础（投资资本）上赚取 ROE。挑战费率基础中包含的过度或轻率的资本投资。
3. **费率类别间的成本分摊：** 公用事业在住宅、商业和工业费率类别间分摊总营收需求。确保你的费率类别未在成本成因之上补贴住宅或其他类别。
4. **费率设计：** 即使总营收获批，也要争取基于需量的费率设计（奖励负荷率管理）而非纯按量费率（无论负荷形状如何都惩罚高消耗客户）。

## 10. 紧急采购协议

### 10.1 供应商违约 / 破产

若你的零售能源供应商申请破产或未能履约：

**立即行动（24-48 小时）：**
1. 与公用事业核实你的账户状态。若供应商违约，你的账户将恢复为公用事业的最后承运商（POLR）服务或标准报价服务。你不会失去电力——电网始终持续供电。
2. 确定 POLR 费率。在大多数州，POLR 费率按季度设定，基于批发市场价格加溢价（高于竞争性供应 10-20%）。这可能高于或低于你当前的合同费率。
3. 立即联系 2-3 家替代供应商。说明情况——他们将提供加急入网（5-10 个工作日 vs. 正常 30-60 天的切换流程）。
4. 审查你的合同中的供应商违约条款，包括可能在破产财产中受损的任何定金或预付款。

**中期（2-4 周）：**
1. 与最佳替代供应商执行新的供应合同。
2. 在破产程序中对任何预付款、定金或损害提出索赔。
3. 审查你的供应商资质标准——考虑对未来合同增加财务契约（最低信用评级、有形净资产要求）。

### 10.2 不可抗力事件

当不可抗力事件（自然灾害、电网紧急情况、流行病）扰乱你的能源供应或运营时：

**评估框架：**

| 事件类型 | 能源影响 | 采购响应 |
|-----------|--------------|---------------------|
| 飓风/恶劣天气 | 发电/T&D 物理损坏，价格尖峰 | 启动备用发电，削减非必要负荷，记录以供保险 |
| 电网紧急情况（EEA3） | 轮流停电，极端价格 | 最大负荷削减，DR 激活，发电机部署 |
| 供应商不可抗力索赔 | 供应商试图暂停合同 | 严格审查 FM 条款——"市场价格上涨"不是不可抗力；"实际无法交付"可能是 |
| 流行病/运营停工 | 设施关闭，消耗骤降 | 援引数量容差条款，谈判合同暂停，评估提前终止 |

### 10.3 合同终止决策矩阵

评估是否提前终止供应合同时：

```
提前终止费（ETF）= Σ(剩余月数 × 月电量 × |合同价 - 当前市场价|)

若合同价 > 当前市场：
  你欠供应商（你支付高于市场）
  ETF = 剩余月数 × 电量 ×（合同价 - 市场价）× 折现因子

若合同价 < 当前市场：
  供应商欠你（你拥有有利的合同）
  你不会终止——合同处于价内

决策：若 ETF < 替代合同的累计节省 + 风险减少价值，则终止
```

**示例——中期退出评估：**

```
当前合同：$0.062/kWh，剩余 18 个月，剩余 50 GWh
当前市场：$0.055/kWh（自合同签订以来市场已下跌）
ETF：50,000 MWh × ($0.062 - $0.055) = $350,000

替代合同：18 个月 $0.054/kWh
替代节省：50,000 MWh × ($0.062 - $0.054) = $400,000

终止的净收益：$400,000 节省 - $350,000 ETF = $50,000

决策：边缘。考虑：
  - 重新谈判风险（能否在市场变动前锁定 $0.054？）
  - 切换供应商的行政成本
  - 与当前供应商的关系成本
  - 若净收益 < $100K，通常不值得这种中断
```
