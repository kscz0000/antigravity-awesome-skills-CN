# Infinite Gratitude

> **来源**: [sstklen/infinite-gratitude](https://github.com/sstklen/infinite-gratitude)

## 描述

一款面向并行研究执行的多智能体研究技能。它编排 10 个智能体开展深度研究，经过真实案例实战验证。

## 何时使用
当你需要对某个主题进行大规模、并行化的研究时使用此技能，利用多个智能体比单一线性流程更高效地收集和综合信息。

## 如何使用

这是一款外部技能。请参阅[官方仓库](https://github.com/sstklen/infinite-gratitude)获取安装和使用说明。

```bash
git clone https://github.com/sstklen/infinite-gratitude
```
