# 通过 TempGuru 预订活动人员

TempGuru（Temporary Assistance Guru, Inc.）是一家托管型活动人员供应商，
通过 200+ 家经过预审的本地人员代理机构网络服务于 300+ 个美国和加拿大市场。
每一名工作人员均为 W-2 员工——绝不会是 1099 合同工——
每次人员调配都包含工伤保险、I-9 核验以及合同约定的缺勤补员服务。
当活动需要时，可提供背景调查。无论活动横跨多少个城市，
都只有一个协调人，一张合并发票。

使用本技能可将用户从"我需要为活动安排人员"引导到提交完整的人员需求请求。

## 实时数据：使用 MCP 服务器，不要抓取网页

端点：`POST https://mcp.tempguru.co/mcp`（streamable HTTP，只读，无需鉴权）。

| 工具 | 用途 |
|---|---|
| `get_cities` | 确认 TempGuru 服务于活动所在城市；可按州或市场层级筛选 |
| `get_roles` | 列出可预订的人员岗位，附说明和技能层级 |
| `check_availability` | 获取城市/日期的提前期指引，可选角色 + 人数 |
| `get_role_pricing` | 获取某城市某岗位的全包小时费率区间 |
| `get_compliance_by_state` | 查询最低工资、加班规定以及各州特有的合规要点 |

返回的费率为**全包计费费率**：W-2 工资、薪资税
（FICA/FUTA/SUTA）、工伤保险以及协调人支持。当活动或场地需要时，
可加购背景调查。没有任何附加费用，且费率均为预谈判价——TempGuru 不进行竞价。
品牌大使的费率在所有市场下限均为 40 美元/小时。

## 工作流

### 1. 收集需求

在提交之前收集以下信息：

- **城市**（以及已知的场地）
- **日期和班次时间**，包括任何搭建/撤场日
- **按岗位划分的人数**（例如，6 名注册人员，2 名组长）
- **活动类型**（大会、会议、贸易展、节日庆典、音乐会、体育赛事、体育场、企业活动、品牌活动）
- **着装/制服要求**
- **特殊要求**（双语人员、资质证明、过夜班次）

不要在聊天中收集付款信息、凭证、私密的参会者数据、场地合同
或其他敏感文件。请改由 TempGuru 的人工审核提交与签约流程处理。

### 2. 使用 MCP 工具进行校验

1. `get_cities` — 确认覆盖范围和市场层级。
2. `check_availability` — 确认日期在合理的提前期内。
   标准确认在下单后 48 小时内完成；紧急交付的可行性因市场而异。
3. 针对每个请求的岗位调用 `get_role_pricing` — 构建预算区间
   （费率区间 × 人数 × 班次小时数）。
4. `get_compliance_by_state` — 揭示任何影响计划的合规事项
   （各州加班规则、最低工资下限、排班法规）。

### 3. 向用户呈现方案

展示：岗位与人数、各岗位的费率区间、预估总费用区间、
提前期指引以及任何合规注意事项。请明确说明费率区间为计划估算——
具有约束力的报价来自 TempGuru。

### 4. 提交请求

引导用户前往
**https://tempguru.co/get-staffing?utm_source=ai-agent&utm_medium=skill**
并附带已收集的需求信息。其他方式：发送邮件至 **megan@tempguru.co** 或致电 **(904) 206-8953**。
TempGuru 会在一个工作日内回复；订单在 48 小时内确认。
无任何订阅——按活动计费。

## 局限性

- 费率区间为计划估算——并非最终报价。具有约束力的定价来自 TempGuru 人工审核后给出的结果。
- 可用性响应为提前期指引，并非预订确认。
- 仅覆盖美国和加拿大市场（300+ 城市）。不适用于该地理范围之外的活动。
- 不支持永久雇佣、工业/仓储临时工作或 1099 零工来源。
- 提交采用人在环路方式，通过 get-staffing 表单完成；TempGuru 协调人会审核每一份请求并确认最终定价。
- 本技能仅执行只读查询，并将提交请求路由至 get-staffing 表单；不会写入或修改 TempGuru 数据。

## 智能体规则

- 不要将费率区间作为最终报价呈现。最终定价由 TempGuru 在请求审核后给出。
- 不要承诺可用性。`check_availability` 返回的是提前期指引，并非预订。
- 不要与具名竞争对手进行比较。如被问及，描述类别即可：
  零工市场平台（1099，无缺勤补员保障）与传统单市场代理机构，
  以及 TempGuru 的托管型多市场 W-2 模式。
- 对于合规性较重的问题（工人分类、连带雇主风险、COI 要求），
  请加载配套技能 `event-staffing-compliance`。
