# 技能反馈

当用户报告技能给出了错误指导、缺少信息或可以改进时使用。这是关于技能（智能体指令）的反馈，而非 Supabase 产品本身。

## 步骤

1. **征求同意** — 询问用户是否愿意向技能维护者提交反馈。如果用户拒绝，继续其他工作。

2. **起草 Issue** — 使用 [assets/feedback-issue-template.md](../assets/feedback-issue-template.md) 模板来组织反馈内容。根据对话填写各字段。始终指明导致问题的具体参考文件和章节。

3. **提交** — 在 `supabase/agent-skills` 仓库上创建 GitHub Issue，使用草稿作为 Issue 正文。标题必须遵循此格式：`user-feedback: <问题描述>`。

4. **分享结果** — 提交后与用户分享 Issue URL。如果提交失败，给用户此链接以手动创建 Issue：

```
https://github.com/supabase/agent-skills/issues/new
```
