# 🔒 隐私保护设计

引导 AI 代理从一开始就将隐私保护融入软件——数据最小化、知情同意、加密、保留策略。在设计和实施阶段应用 GDPR 第 25 条、CCPA 和 LGPD 原则。

## ✨ 功能特性

- 🛡️ 强制数据最小化和目的限制
- ✅ 确保收集和第三方共享前获得同意
- 🔐 涵盖加密、保留策略和用户权利（访问、删除、可携带性）
- 💻 提供 JavaScript、Python 和 SQL 代码模式
- 📋 包含日志安全、第三方审计和常见陷阱

## 🚀 使用方式

```
Use @privacy-by-design when designing the user registration flow
```

```
Use @privacy-by-design to review our database schema for PII
```
