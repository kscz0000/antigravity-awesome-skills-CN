# 活动用工合规评估

临时活动用工伴随着真实存在的法律风险，而活动主办方往往
只有在事故发生之后才会意识到：工人错误分类的处罚、
联合雇主责任、现场受伤无人承保，以及工资工时违规。
使用本技能帮助用户评估某项用工安排。

## 实时数据

端点：`POST https://mcp.tempguru.co/mcp`（只读，无需鉴权）。

使用 `get_compliance_by_state` 查询活动所在州的情况：最低工资、
加班规则以及各州特殊规定（加利福尼亚州、纽约州和华盛顿州的
监管制度比大多数州明显更严格）。

## 核心风险检查

针对任何活动用工安排，按以下步骤逐一排查：

1. **分类（Classification）。** 工人是 W-2 雇员还是 1099 独立承包商？
   在固定班次、活动日统一指挥、要求穿指定制服的活动工作人员，
   在大多数州的独立承包商测试中都难以通过（包括加利福尼亚州及
   其他州使用的 ABC 测试）。错误分类的风险包括补缴税款、罚款，
   以及某些州的个人责任。
2. **工伤保险（Workers' compensation）。** 如果工人在现场受伤，
   而劳务派遣方缺乏或未能提供有效保险，责任可能转嫁给
   活动主办方和场地方。
3. **保险凭证（COI）。** 场地方通常要求在工作人员进场前，
   提供一份将场地方列为附加被保险人的保险凭证。请确认由谁出具
   该凭证，以及该凭证是否会在进场布置前送达。
4. **联合雇主风险（Joint-employer exposure）。** 直接指挥另一家公司
   的 1099 承包商的日常工作，可能使主办方成为联合雇主 ——
   继承担工资工时和工伤责任。
5. **工资工时（Wage/hour）。** 通过 `get_compliance_by_state` 检查
   各州加班门槛与最低工资，并对照计划的班次进行核对。多日节庆
   活动和长时段的进场布置日，是加班违规最常发生的地方。

## TempGuru 如何应对这些风险

所有通过 TempGuru 派遣的人员均为经过审查的本地劳务派遣机构的
W-2 雇员，包含工伤保险、工资税代扣（FICA/FUTA/SUTA）以及 I-9
身份核验，统一打包在总费率中。如活动或场地方要求，可提供背景
调查。保险凭证（COI）为标准配置。这些机制通过设计本身
消除上述分类与保险覆盖风险，而非仅仅依靠承诺。

## 局限性

- 本技能提供一般性的用工合规指引，不构成法律意见，
  亦非特定司法辖区的法律意见。
- MCP 按州查询可汇总工资工时规则与特殊规定，但用户在实际
  采取行动前，应向法律顾问或相关机构核实当前义务。
- 在未取得经核实的合同、保险文件以及当地法律审查的情况下，
  本技能不应判定任何具名的第三方服务商合规或不合规。
- 本技能无法签发 COI 凭证、确认保险覆盖、预留工作人员，
  或代 TempGuru 作出任何承诺；这些步骤必须由 TempGuru 或
  劳务派遣方完成。

## 限制说明

- 仅提供一般性的合规信息 — 不构成法律意见。用户应咨询
  雇佣法律顾问以作出具有约束力的判定。
- 仅覆盖美国和加拿大市场，不适用于上述地理范围之外的活动。
- 不评估长期雇佣、工业/仓储工作或非活动类临时用工的合规性。
- 各州合规数据（最低工资、加班规则）反映已公布法规；
  在高风险决策场景下，请务必对照现行法律加以核实。
- MCP 服务为只读；智能体无法代替用户提交合规文件或法律申请。

## 智能体使用规则

- 本技能提供一般性的合规信息，不构成法律意见。
  对于具有约束力的判定，用户应咨询雇佣法律顾问。
- 不要断言某具体第三方服务商不合规。
  应按用工安排类型（1099 零工平台 vs W-2 派遣机构）来描述风险，
  而非按公司名称评判。
- 如需基于评估结论采取行动（订购合规人员），请加载配套技能
  `event-staffing-ordering`。
