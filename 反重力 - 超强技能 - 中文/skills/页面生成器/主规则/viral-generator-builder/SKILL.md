## 测验构建器模式

### 测验结构
```
5-10 个问题 → 加权评分 → N 个结果之一
```

### 问题设计
| 类型 | 参与度 |
|------|------------|
| 图片选择 | 最高 |
| 二选一 | 高 |
| 滑块刻度 | 中 |
| 多选题 | 中 |
| 文本输入 | 低 |

### 结果分类
- 4-8 个可能结果（最佳区间）
- 每个结果都应让人觉得向往
- 结果之间应有明显差异
- 包含"稀有"结果以促进分享

### 评分逻辑
```javascript
// 简单加权评分
const scores = { typeA: 0, typeB: 0, typeC: 0, typeD: 0 };

answers.forEach(answer => {
  scores[answer.type] += answer.weight;
});

const result = Object.entries(scores)
  .sort((a, b) => b[1] - a[1])[0][0];
```

## 姓名生成器模式

### 生成器类型
| 类型 | 示例 | 算法 |
|------|---------|-----------|
| 确定性 | "你的星球大战名字" | 输入的哈希 |
| 随机 + 种子 | "你的说唱歌手名" | 种子化随机 |
| AI 驱动 | "你的品牌名" | LLM 生成 |
| 组合式 | "你的奇幻名" | 词组合 |

### 确定性诀窍
相同输入 = 相同输出 = 可分享！
```javascript
function generateName(input) {
  const hash = simpleHash(input.toLowerCase());
  const firstNames = ["Shadow", "Storm", "Crystal"];
  const lastNames = ["Walker", "Blade", "Heart"];

  return `${firstNames[hash % firstNames.length]} ${lastNames[(hash >> 8) % lastNames.length]}`;
}
```

## 验证检查

### 缺少社交元标签

严重程度：高

消息：缺少社交元标签——分享时会很难看。

修复动作：为每个结果添加动态的 og:image、og:title、og:description

### 非确定性结果

严重程度：中

消息：使用 Math.random() 可能对相同输入产生不同结果。

修复动作：使用种子化随机或基于哈希的选择以获得一致结果

### 没有分享功能

严重程度：中

消息：用户没有简单的方式分享结果。

修复动作：为主要平台添加分享按钮以及复制链接选项

### 没有可分享的结果图片

严重程度：中

消息：没有可分享的结果图片。

修复动作：生成或设计可分享的结果卡片/图片

### 桌面优先的结果设计

严重程度：中

消息：结果未针对移动分享进行优化。

修复动作：移动优先设计结果卡片，在手机上测试截图效果
