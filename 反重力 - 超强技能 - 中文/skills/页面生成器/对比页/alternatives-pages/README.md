# 备选方案页面

为开发者工具创建"[竞品]替代方案"与对比页面，在竞争关键词下获得排名，并诚实地打动开发者转化。

## 本技能涵盖内容

- 替代与对比关键词的 SEO 研究
- 不耍滑头的高转化页面结构
- 打造诚实、可核实的对比表
- 迁移内容与回应切换顾虑
- 何时点名竞品，何时保持泛化
- 竞争内容的法律考量
- 持续的内容维护

## 推荐工具

- **社交聆听工具** — 监控竞品讨论与开发者痛点
- **SEO 工具**（Ahrefs、Semrush）— 关键词研究与搜索量

## 快速上手

1. 使用社交聆听调研竞品痛点与对比模式
2. 按搜索量与竞争重叠度排定页面优先级
3. 创作诚实、详尽的对比内容
4. 提供迁移指引与支持
5. 设置内容新鲜度的监控触发
6. 建立季度回顾节奏

## 页面类型

**备选方案页面：**"[竞品] 替代方案"——面向寻找切换出路的开发者
**对比页面：**"[你] vs [他]"——面向正在主动评估的开发者
**品类页面：**"最佳 [品类] 工具"——面向品类调研者

## 核心原则

- 保持诚实——开发者会逐项核对
- 保持有用——即便他们最终没有选你
- 保持具体——含糊的对比浪费时间
- 保持新鲜——标注日期并持续更新
