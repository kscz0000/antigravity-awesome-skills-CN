# 实操示例 — 改造前后对比

来自真实项目的匿名示例。每个展示弱版、诊断和强版。

---

## 示例1：分时CMO服务

### 改造前

**报价（原样）：**
> 分时CMO服务。$15K/月。我们会帮你增长。

**诊断：**
- 理想结果: 4 (模糊 — "增长")
- 感知实现概率: 3 (没有方法论，没有案例)
- 时间延迟: 4 (没有时间线，无限期项目)
- 努力与牺牲: 5 (不清楚买家要做什么)
- 解剖：只有核心存在。没有赠品、没有保障、没有稀缺性、没有命名。

**最低瓶颈约束：** 感知实现概率。买家不相信一个未命名的服务会交付。

### 改造后

| 组件 | 添加了什么 |
|------|-----------|
| **核心** | "90天营销重置" — 8周审计 + 12周执行计划，由在3+家类似阶段公司做过营销的CMO交付 |
| **赠品** | (1) 12周每周1:1（~$12K价值）；(2) 预筛选执行合作伙伴引荐（无价）；(3) 董事会营销战略部分模板 |
| **保障** | "8周审计后，如果你没有一个你会自己跑的清晰90天计划，不收审计费。" |
| **稀缺性** | "我们每季度接2个项目——下一个空位在[日期]开放" |
| **命名** | "90天营销重置" |
| **价格** | $15K → $5K启动，$5K第8周，$5K第16周 |

同样交付，同样的人，约3倍成交率，更长项目（因为买家对范围更清晰）。

**教训：** 价格没动。结构动了。

## 示例2：$1,997 基于班期的文案课

### 改造前

**报价：**
> 学文案写作。$1,997。包含6个模块和Slack访问。

**诊断：**
- 理想结果: 5 ("学文案写作" — 表面需求，不是理想结果)
- 感知实现概率: 3 (没有案例，没有命名方法论)
- 时间延迟: 4 (6个月课程，没有首次胜利)
- 努力与牺牲: 4 (大量作业、每周通话、大承诺)

**最低瓶颈约束：** 感知实现概率。买家不相信*他们自己*能做到。

### 改造后

| 组件 | 变更了什么 |
|------|-----------|
| **核心** | "12周写出客户付$5K+的销售页" — 结果框架化 |
| **赠品** | (1) 30个优秀销售页模板(最后更新Q2 2026) — $297价值；(2) 6个行业9个具名案例 — 证据，不是推销；(3) 60天Slack含每周答疑时间 — $497价值；(4) 工具栈 + 折扣码 — $1,200价值 |
| **保障** | "完成全部6个模块，提交最终练习，如果你12个月内没写出获得$5K+客户的销售页，全额退款。" |
| **稀缺性** | 班期稀缺性 — 周五关门，下一期3个月后 |
| **命名** | "$5K销售页训练营" |
| **价格** | $1,997全额 或 $797 × 3 |

同样模块。同样讲师。约4倍转化。更低退款率（有条件保障筛选）。

**教训：** 重命名结果，添加证据，安装真正的稀缺性机制。

## 示例3：$97 Notion 模板包

### 改造前

**报价：**
> 营销人 Notion 模板。$97。包含20个模板。

**诊断：**
- 理想结果: 6 (清楚得到什么，不太清楚用它达成什么)
- 感知实现概率: 6 (模板对有些人有效，对其他人不太行 — 没有证据)
- 时间延迟: 8 (即时访问)
- 努力与牺牲: 5 (需要设置工作来定制每个模板)

**最低瓶颈约束：** 感知实现概率 + 理想结果。"这些真的能为*我的*设置节省时间吗？"

### 改造后

| 组件 | 变更了什么 |
|------|-----------|
| **核心** | "营销运营套件 — 20个Notion模板，一个周六把你散乱的文档变成可运行的营销OS" — 结果框架化 |
| **赠品** | (1) 10分钟"先做这个"Loom — 速度赠品；(2) "叠加模板"流程图(可视化设置图)；(3) 模板添加时终身更新 |
| **保障** | "30天无理由退款" — 无条件，适合这个价位 |
| **稀缺性** | 创始买家定价 — 前200名$97，然后$147 |
| **命名** | "营销运营套件" |
| **价格** | $97全额 |

同样模板。同样流量约2倍成交率。差异化在于"一个周六"结果锚点和证明速度声明的Loom。

**教训：** 对于低价信息产品，理想结果和快速首次胜利是杠杆。不要过度设计保障。

## 示例4：$50K B2B SaaS 年度合同

### 改造前

**报价：**
> 企业计划：$50K/年。包含无限用户、全部功能、专属支持。

**诊断：**
- 理想结果: 5 (列功能，非结果框架化)
- 感知实现概率: 5 (没有上线计划，没有价值时间)
- 时间延迟: 3 (不清楚价值何时开始；销售说"实施因人而异")
- 努力与牺牲: 4 (采购 + 安全审查 + IT集成 + 变革管理)

**最低瓶颈约束：** 时间延迟。企业买家无法容忍"实施因人而异"。

### 改造后

| 组件 | 变更了什么 |
|------|-----------|
| **核心** | "30天生产就绪，季度末见ROI" — 时间锚定 |
| **赠品** | (1) 30天专属实施工程师；(2) 前5个平台预构建集成包；(3) 买家团队定制培训；(4) 与买家CSM的季度业务评审 |
| **保障** | "30天未上线？你不用付钱直到上线。" 基于SLA。 |
| **稀缺性** | 基于产能："我们每季度上线4个企业账户。下一个空位[日期]开始。" |
| **命名** | 层级名保持"企业"但添加项目名"战略入驻" |
| **价格** | $50K年度 → $50K年度季付 + 30天付费试点 |

同样产品。约30%更高成交率，50%更短销售周期。试点 + SLA组合消除了采购异议。

**教训：** 对于企业B2B，价值时间就是报价。显式解决它。

## 示例5：$4K 小组教练大师班

### 改造前

**报价：**
> 创始人小组教练。$4K/季度。包含12次通话和Slack。

**诊断：**
- 理想结果: 5 (模糊 — "成为更好的创始人")
- 感知实现概率: 4 (一个校友证言，没有方法论)
- 时间延迟: 6 (季度节奏合理)
- 努力与牺牲: 7 (12次通话是真实时间)

**最低瓶颈约束：** 理想结果 + 感知实现概率。

### 改造后

| 组件 | 变更了什么 |
|------|-----------|
| **核心** | "12位创始人，12周，每人一个具体目标——以及一个见过的房间" — 同伴房间定位 |
| **赠品** | (1) 1:1入驻通话设定个人目标；(2) 创始人资源库 — 过去成员90个框架；(3) 1:1季度中检查；(4) 之后1年校友访问 |
| **保障** | "前两周——如果不是你想要的房间，全额退款。之后，你就在里面了。" |
| **稀缺性** | 班期规模限12人 — 满后下季度候补 |
| **命名** | "创始人季度" |
| **价格** | $4K/季度全额 或 $1,500 × 3 |

同样教练，同样节奏。更高成交率。值得注意的是：成员续费率约70%（之前约35%），因为"1年校友访问"赠品将购买决策框架从"季度"改为"年度"。

**教训：** 对于教练，房间就是报价。定位房间，不是课程。有利于续费的赠品锁定长期LTV。

## 示例6：代理保留服务 — 内容营销

### 改造前

**报价：**
> 内容营销保留服务。$8K/月。每月4篇文章 + SEO策略。

**诊断：**
- 理想结果: 4 (描述产出，非结果框架化)
- 感知实现概率: 5 (没有将内容与收入关联的案例)
- 时间延迟: 3 (SEO慢；客户预期不匹配)
- 努力与牺牲: 6 (访谈、评审、审批全在客户方)

**最低瓶颈约束：** 理想结果(模糊)和时间延迟(预期不匹配)。

### 改造后

| 组件 | 变更了什么 |
|------|-----------|
| **核心** | "我们拥有内容引擎。你在第9个月获得有机驱动的销售会议，有可衡量的收入归因。" — 结果 + 时间线 |
| **赠品** | (1) 人物研究启动(一次性)；(2) 季度内容审计 + 重发列表；(3) 预筛选自由撰稿人 + QA层；(4) 季度高管简报 |
| **保障** | "前30天是付费试点 — 4篇发布 + 3个关键词路线图。如果结束时你没看到清晰的12个月路径，我们终止项目，不欠余额。" |
| **稀缺性** | 基于产能："我们每季度接3个保留客户。下一个空位[日期]。" |
| **命名** | 层级名："增长保留服务"；项目名："90天内容重置 → 9个月增长引擎" |
| **价格** | $8K/月，6个月最低，或12月承诺$7K/月 |

同样撰稿人。同样SEO方法论。约2倍成交率。60%的试点转化为12月承诺。

**教训：** 对于慢周期服务(SEO、品牌、内容)，报价必须显式应对时间线。"相信我们，6个月出结果"卖不动；"付费试点 → 第30天里程碑 → 起飞"才卖得动。

## 六个示例的共同模式

并排看变更：

| 示例 | 核心变更 | 最重要的其他变更 |
|------|---------|---------------|
| 分时CMO | 命名了、添加了范围 | 首里程碑保障 |
| 文案课 | 结果框架化、添加了证据 | 案例赠品 |
| Notion 模板 | "一个周六"锚点 | 第一步Loom赠品 |
| B2B SaaS | 价值时间承诺 | SLA保障 + 试点 |
| 教练大师班 | 定位了房间，不是教练 | 1年校友访问赠品 |
| 代理保留服务 | 结果 + 时间线框架化 | 付费试点保障 |

**模式：** 在每个案例中，价格几乎没动（或完全没动）。移动的是报价的*结构*——命名、框架化、保障、排序。

价格是比较。价值是报价。
