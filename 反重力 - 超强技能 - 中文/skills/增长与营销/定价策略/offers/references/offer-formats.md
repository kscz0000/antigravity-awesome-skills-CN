# 按商业类型的报价格式

正确的报价格式取决于你卖什么。同样的六个组件(核心、赠品、保障、稀缺性、命名、价格)按商业类型以不同方式组装。

本参考按商业类型组织。找到你的类型，然后将格式作为起点——不是固定配方。

---

## 服务 / 自由职业

你出售时间和技能。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 一个有明确交付物和时间线的范围化项目 |
| **赠品** | 模板、框架、项目后支持、工具栈 |
| **保障** | 首里程碑门槛(付费试点或首交付物退款) |
| **稀缺性** | 基于产能(下一空位在[日期]开放) |
| **命名** | 方法论命名或结果命名(如"30天激活冲刺") |
| **价格** | 项目制首付，或月度保留服务 |

### 注意事项

- **命名的重要性超乎比例** — 没有命名报价的服务在价格上竞争；命名服务在定位上竞争
- **范围蔓延是报价杀手** — 在项目开始前，*书面*定义包含、不包含和可选的内容
- **赠品应该强化交付物** — 让买家在项目结束后而非期间变得自立的模板和框架

### 产品化报价

从"我卖咨询"到"我卖8周营销重置"。同样的交付，不同的报价。

产品化版本：
- 有名称
- 有固定的范围和时间线
- 有固定的价格
- 每次有相同的赠品
- 有明确的门槛(第4周检查、付费试点、里程碑评审)

产品化提升感知价值、简化销售、并创建可重复的案例工厂。

## 课程(异步、基于班期、直播)

你出售结构化学习。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 课程内容 + 交付格式 |
| **赠品** | 模板、模板包、案例、社区访问 |
| **保障** | 有条件退款(以完成为门槛) |
| **稀缺性** | 班期稀缺性(报名在[日期]关闭) |
| **命名** | 结果命名或方法论命名 |
| **价格** | 全额付款或2–4期分期 |

### 异步 vs 基于班期

| 决策 | 异步 | 班期 |
|------|------|------|
| **定价** | 较低($297–$1,997) | 较高($1,497–$5,000+) |
| **稀缺性** | 赠品到期、涨价 | 班期开课日期 |
| **保障** | 宽松无条件 | 以完成为条件 |
| **转化机制** | 邮件漏斗、常青研讨会 | 发布窗口 + 班期截止 |
| **默认赠品** | 模板、模板包 | Slack + 答疑时间 + 1:1评审 |

### 注意事项

- **完成率是营销资产** — 每个完成者都是案例。在第1周工程化实现首次胜利。
- **班期稀缺性必须真实** — 如果"周五关门"之后是"周一因为延期重新开门"，这把戏会被发现
- **退款设计比退款率更重要** — 听起来宽松但有聪明条件的保障转化好且很少退款

## 教练(一对一、小组、大师班)

你出售专业知识应用于他们具体情况的机会。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 辅导会 + 异步访问 + 具体结果聚焦 |
| **赠品** | 你的资源库资源、引荐网络、项目后跟进 |
| **保障** | 结果或延期，或前两次辅导退出 |
| **稀缺性** | 基于产能(N名额/季度) |
| **命名** | 身份命名或结果命名(如"创始人营销大师班") |
| **价格** | 月度保留服务或3/6/12个月项目 |

### 一对一 vs 小组

| 决策 | 一对一 | 小组 / 大师班 |
|------|--------|---------------|
| **定价** | $1,500–$10,000+/月 | $497–$2,500+/月 |
| **稀缺性** | 产能(4–10个1:1客户) | 班期规模(8–30人) |
| **保障** | 前两次辅导退出 | 试用期，之后不退 |
| **赠品** | 定制资源、引荐 | 小组访问、同伴问责、资源库访问 |
| **默认交付** | 每周或双周辅导 | 月度小组通话 + 社区 |

### 注意事项

- **身份就是报价** — 小组教练往往更多是关于和同伴在同一个房间里，而不是教练的指导。命名房间，不命名教练。
- **入驻是报价的一部分** — 粗糙的 intake 破坏感知实现概率
- **续费才是真正的转化事件** — 为6个月的决策设计，不是第一个月的决策

## 信息产品(指南、模板包、模板包、社区)

你出售打包的知识或资产。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 资产 + 终身访问 |
| **赠品** | 相邻资产、操作指南、模板 |
| **保障** | 宽松无条件(30天无理由) |
| **稀缺性** | 赠品到期、涨价时间表 |
| **命名** | 结果命名，通常干脆而具体 |
| **价格** | $29–$497，全额付款 |

### 注意事项

- **第一印象的重要性超乎比例** — 买家打开一次。如果前5分钟不感觉高端，他们不会继续使用剩余内容
- **快速启动是赠品** — 配合一个10分钟"先做这个"的操作指南
- **终身访问是隐性定价** — 澄清"终身"意味着什么(你的、产品的、直到你停运)

## 高客单 B2B ($5K+ ACV，销售主导)

你通过销售对话向企业出售。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 多月项目或年度合同 |
| **赠品** | 入驻、培训、集成、专属CSM |
| **保障** | SLA、绩效制、或试点门槛 |
| **稀缺性** | 季末定价、产能(N个入驻/季度)、层级限制 |
| **命名** | 内部稳定(如"企业计划") + 命名的项目类型(如"战略入驻") |
| **价格** | 年度合同季付，通常定制 |

### 注意事项

- **采购委员会，不是单个买家** — 报价必须同时打动推动者、经济买家和影响者
- **采购流程就是报价** — 你的条款(付款、NET 60、安全审查、MSA)是报价的一部分；僵化的条款丢单
- **试点报价转化老练买家** — "30天付费试点，到期决定是否继续"降低决策风险
- **CSM 是报价的一部分** — 买家一致将售后关系评为报价感知价值的一部分

## 代理保留服务

你出售持续的服务交付。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 月度交付物 + 专属团队 + 汇报节奏 |
| **赠品** | 战略会议、工具访问、审计额度、资源库访问 |
| **保障** | 第1月付费试点或90天退出条款 |
| **稀缺性** | 产能(N客户/垂直/季度) |
| **命名** | 层级命名("增长""扩展""企业") + 服务线 |
| **价格** | 月度保留服务，年度承诺享折扣 |

### 注意事项

- **入驻速度就是报价** — 花6周才开始交付的代理输给第1周就交付东西的代理
- **汇报是报价的一部分** — 清晰的月度行动导向汇报比额外交付物更能降低流失
- **层级升级是最容易的收入** — 设计有清晰价值阶梯的层级，让升级对话显而易见

## 自助 SaaS

你出售带分层订阅的工具。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | 分层订阅，功能区分清晰 |
| **赠品** | 免费入驻、模板、集成、合作伙伴折扣 |
| **保障** | 免费试用 或 年付30天退款 |
| **稀缺性** | 前N名客户创始定价，或季节性发布 |
| **命名** | 层级命名("入门""专业""团队""企业") |
| **价格** | 月付或年付享折扣，基于价值指标 |

### 注意事项

- **定价层级 > 报价构建** — 对于自助SaaS，打包和价值指标比保障和赠品做更多工作。使用 `pricing` 技能。
- **免费试用设计就是报价设计** — 长度、功能门控、是否需要信用卡、自动转化。每个都是报价决策。
- **年付预付是报价杠杆** — 同样的产品，不同的承诺，通常20–40%折扣。许多SaaS转化提升来自改进年付报价，不是月付。

对于SaaS，本技能是补充性的。先读 [`pricing`](../../pricing/SKILL.md)。

## 直复营销 / 付费流量

你从销售页或VSL向冷流量出售。

### 默认格式

| 组件 | 默认 |
|------|------|
| **核心** | "东西" + 明确回报 |
| **赠品** | 重度赠品叠加(5–7个赠品，分层价值) |
| **保障** | 激进风险逆转(超值退款、双重保障) |
| **稀缺性** | 真实时间限制(发布窗口、常青硬截止) |
| **命名** | 抓人眼球的，通常是模式打断 |
| **价格** | 通常单次付款，可提供分期 |

### 注意事项

- **直复营销买家期望侵略性** — 安静、高端感的报价在冷付费流量上转化差。页面的审美和报价一样重要。
- **退款率可能10–20%** — 算进数学里。如果利润率承受不了15%退款，重新结构化。
- **前7秒决定一切** — 先钩住，再报价

这个格式需要高技能。如果你没有直复营销背景，请雇佣或合作有经验的人。

## 选择你的格式

如果不确定哪个格式适用，选最接近的然后调整。最大的错误是借用不同商业类型的格式(例如把直复营销的赠品叠加用在高端B2B服务上——错误的受众，错误的审美)。

两个诊断问题：

1. **谁买它，他们有多老练？** 高端B2B和直复营销冷流量都买，但他们需要不同的报价。
2. **主要约束是什么？** 服务业务受产能约束，SaaS受定价层级约束，课程受班期/季节约束。将稀缺性格式匹配到真实约束。

各商业类型的实操示例，参见 [examples.md](examples.md)。
