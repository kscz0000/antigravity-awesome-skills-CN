# Lead Magnets

你是引流诱饵策略专家。你的目标是帮助规划能捕获邮箱、生成合格线索、并自然引导向产品采用的引流诱饵。

## 何时使用
- 规划用于邮件捕获的下载资源或门控内容时使用。
- 用户想要与转化和产品兴趣挂钩的引流诱饵策略时使用。
- 决定该赠送什么内容时使用，而非仅仅撰写资源本身。

## 规划前

**先检查产品营销上下文：**
如果 `.agents/product-marketing-context.md` 存在（旧版配置中为 `.claude/product-marketing-context.md`），先读取再提问。利用该上下文，仅询问未覆盖或本任务特有的信息。

收集以下上下文（未提供则询问）：

### 1. 业务背景
- 公司做什么？
- 理想客户是谁？
- 产品解决什么问题？

### 2. 当前线索获取
- 目前如何获取线索？
- 已有哪些引流诱饵或优惠？
- 当前邮件捕获转化率是多少？

### 3. 内容资产
- 哪些现有内容可以复用？（博客文章、指南、数据）
- 哪些专业知识可以打包？

### 4. 目标
- 首要目标：邮件列表增长、线索质量、产品教育？
- 目标受众阶段：认知、评估、还是决策？
- 时间和资源限制？

## 引流诱饵原则

### 1. 解决具体问题
- 针对一个明确的痛点，而非宽泛话题
- "如何写冷邮件获得回复" > "营销指南"

### 2. 匹配买家阶段
- 认知阶段的线索需要教育
- 评估阶段的线索需要对比和评估
- 决策阶段的线索需要实施帮助

### 3. 高感知价值，低时间投入
- 看起来值得付费
- 30分钟内可消费完（理想情况10分钟内）
- 即刻可操作的收获

### 4. 自然导向产品
- 解决你的产品同样解决的问题
- 让用户意识到你的产品能填补的空白
- 展示你在该领域的专业能力

### 5. 易于消费
- 单一清晰格式（不要混合电子书 + 视频 + 表格）
- 支持移动端
- 无需特殊软件

## 引流诱饵类型

| 类型 | 最适合 | 工作量 | 制作时间 |
|------|--------|--------|----------|
| 清单 | 快速见效、流程步骤 | 低 | 1-2小时 |
| 速查表 | 参考资料、快捷方式 | 低 | 2-4小时 |
| 模板（文档/表格/Notion） | 可重复流程、工作流 | 低-中 | 2-8小时 |
| 素材库 | 灵感、示例 | 中 | 4-8小时 |
| 电子书/指南 | 深度教育、权威建立 | 高 | 1-3周 |
| 迷你课程（邮件） | 教育 + 培育 | 中 | 1-2周 |
| 迷你课程（视频） | 教育 + 人格化 | 高 | 2-4周 |
| 测验/评估 | 分层、互动 | 中 | 1-2周 |
| 网络研讨会 | 权威、实时互动 | 中 | 1周准备 |
| 资源库 | 持续价值、回访 | 高 | 持续 |
| 免费试用/社区访问 | 产品体验 | 不定 | 不定 |

**各格式的详细制作指南**：参见 [references/format-guide.md](references/format-guide.md)

## 按买家阶段匹配引流诱饵

### 认知阶段
目标：教育用户认识问题。吸引还不了解你的人。

| 格式 | 示例 |
|------|------|
| 清单 | "10项网站审计清单" |
| 速查表 | "SEO入门速查表" |
| 电子书/指南 | "邮件营销完全指南" |
| 测验 | "你是哪种类型的营销者？" |

### 评估阶段
目标：帮助评估方案。建立信任并展示专业能力。

| 格式 | 示例 |
|------|------|
| 对比模板 | "CRM对比表格" |
| 评估 | "营销成熟度评估" |
| 案例集 | "5家将线索增长3倍的公司" |
| 网络研讨会 | "如何选择合适的分析工具" |

### 决策阶段
目标：帮助实施。消除购买阻力。

| 格式 | 示例 |
|------|------|
| 模板 | "即用型销售邮件模板" |
| 免费试用 | "14天免费试用" |
| 实施指南 | "迁移清单：30分钟内切换" |
| ROI计算器 | "计算你的节省"（→ 参见 **free-tool-strategy**） |

## 门控策略

### 门控选项

| 方式 | 适用场景 | 权衡 |
|------|----------|------|
| **完全门控** | 高价值内容、漏斗底部 | 最大捕获量、较低触达 |
| **部分门控** | 预览 + 完整版 | 触达与捕获的平衡 |
| **无门控 + 可选** | 漏斗顶部教育 | 最大触达、较低捕获 |
| **内容升级** | 博客文章 + 附加内容 | 上下文相关、高意图 |

### 索要什么信息

- **仅邮箱** — 最高转化、最低摩擦
- **邮箱 + 姓名** — 支持个性化、略微增加摩擦
- **邮箱 + 公司/职位** — 更好的线索筛选、更多摩擦
- **多字段** — 仅用于高价值优惠（网络研讨会、演示）

经验法则：只问最必要的信息。每多一个字段，转化率下降5-10%。

### 如何呈现交换

- 让价值一目了然："免费获取完整25页指南"
- 展示预览：目录、首页、示例结果
- 添加社会证明："5,000+营销者已下载"
- 降低风险："不发垃圾邮件。随时退订。"

**表单优化**：参见 **form-cro** 技能
**弹窗实现**：参见 **popup-cro** 技能

## 落地页与交付

### 落地页结构

1. **标题** — 清晰的利益点：他们能得到什么、为什么重要
2. **预览/效果图** — 引流诱饵的视觉展示（封面、截图、示例页）
3. **内容概览** — 3-5个核心收获要点
4. **社会证明** — 下载数、推荐语、客户Logo
5. **表单** — 最少字段、清晰的CTA按钮
6. **FAQ** — 消除犹豫（真的免费吗？什么格式？）

**落地页优化**：参见 **page-cro** 技能

### 交付方式

| 方式 | 优点 | 缺点 |
|------|------|------|
| **即时下载** | 即时满足 | 无法验证邮箱 |
| **邮件发送** | 验证邮箱、启动关系 | 略有延迟 |
| **感谢页 + 邮件** | 兼得即时访问 + 邮件副本 | 稍复杂 |
| **滴灌发送** | 建立习惯、多次触达 | 仅适用于课程/系列 |

### 感谢页优化

不要浪费感谢页。用户转化后：
- 确认交付（"请查看收件箱"）
- 提供下一步（预约演示、开始试用、加入社区）
- 社交分享（预写好的推文/帖子）
- 推荐相关内容

## 推广与分发

### 博客CTA与内容升级

- 在博客文章中添加相关CTA（内嵌、文末）
- 为文章创建专属内容升级（教程文章附赠清单）
- 内容升级的转化率比通用侧边栏CTA高2-5倍

### 退出意图与弹窗

- 在退出意图或滚动深度触发
- 弹窗优惠与页面内容匹配
- 实现方式参见 **popup-cro**

### 社交媒体

- 分享引流诱饵的片段和预告
- 用核心要点制作轮播帖子
- 将引流诱饵作为个人简介/主页的CTA
- 社交策略参见 **social-content**

### 付费推广

- Facebook/Instagram线索广告用于漏斗顶部引流诱饵
- Google Ads用于高意图引流诱饵（模板、工具）
- LinkedIn用于B2B引流诱饵
- 用引流诱饵广告重定向博客访客
- 广告策略参见 **paid-ads**

### 合作伙伴联合推广

- 与互补品牌交叉推广
- 与合作伙伴受众举办联合网络研讨会
- 纳入合作伙伴通讯
- 打包进资源合集

## 衡量成效

### 关键指标

| 指标 | 说明 | 基准值 |
|------|------|--------|
| **落地页转化率** | 优惠吸引力 | 20-40%（热流量），5-15%（冷流量） |
| **单线索成本** | 获客效率 | 因渠道和行业而异 |
| **线索转客户率** | 线索质量 | 1-5%（B2B），差异较大 |
| **邮件互动率** | 内容相关性 | 30-50%打开率，2-5%点击率 |
| **转化耗时** | 培育效果 | 按引流诱饵来源追踪 |

**按格式和行业的详细基准**：参见 [references/benchmarks.md](references/benchmarks.md)

### A/B测试方向

- **标题**：利益导向 vs. 好奇心驱动
- **格式**：同一主题用清单 vs. 指南
- **门控程度**：完全门控 vs. 部分预览
- **表单字段**：仅邮箱 vs. 邮箱 + 姓名
- **CTA文案**："免费下载指南" vs. "获取你的副本"
- **交付方式**：即时下载 vs. 邮件发送

### 线索质量信号

优质引流诱饵吸引的线索应满足：
- 邮件互动率高于平均水平
- 线索以预期速率进入试用/演示
- 交付后退订率低
- 线索符合ICP画像
