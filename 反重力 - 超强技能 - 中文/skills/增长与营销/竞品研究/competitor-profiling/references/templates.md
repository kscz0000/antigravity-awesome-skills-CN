# 档案模板

可直接复用的模板，用于竞品档案各章节以及汇总文档。

## 目录
- 快速扫描模板
- 汇总对比表
- 定位图
- 竞争 SWOT
- 档案更新变更日志

---

## 快速扫描模板

当速度优先于深度时使用的精简档案。

```markdown
# [竞品名称] — 快速档案

**URL**: [网站]
**生成日期**: [日期]

## 一览

| 指标 | 数值 |
|------|------|
| 一句话标语 | [来自首页] |
| 目标受众 | [从文案推断] |
| 起售价 | [最低付费档位] |
| 免费档位/试用 | [是/否 + 详情] |
| 域名权重 | [来自 DataForSEO] |
| 预估自然流量 | [每月] |
| 自然搜索关键词（前 10 名） | [数量] |
| 引荐域名数 | [数量] |

## 定位

**主标题**: "[首页原文标题]"
**副标题**: "[原文副标题]"
**定位角度**: [1-2 句话概括其定位方式]

## 定价概览

| 档位 | 价格 | 主要包含 |
|------|------|----------|
| [档位] | [价格] | [关键项] |
| [档位] | [价格] | [关键项] |

## 关键结论

[2-3 句话：竞品亮点、强项与弱项]
```

| **一句话标语** | [你的] | [其] | [其] | [其] |
| **目标受众** | [你的] | [其] | [其] | [其] |

## 定位图

以两个关键维度展示竞品所处位置的直观表达。选择与你所在市场最相关的两个轴。

### 常见轴组合

| 市场类型 | X 轴 | Y 轴 |
|----------|------|------|
| SaaS 工具 | 简单 → 复杂 | 便宜 → 昂贵 |
| 开发者工具 | 低代码 → 代码优先 | 个人 → 团队 |
| B2B 平台 | 面向 SMB → 面向企业 | 单点方案 → 平台 |
| 内容工具 | 模板驱动 → 自定义 | 自助服务 → 代运营 |

### 格式

```markdown
## 定位图

**坐标轴**: [X 轴标签] vs. [Y 轴标签]

                    [Y 轴高值标签]
                           │
                           │
          [竞品 A]         │    [竞品 B]
                           │
    ───────────────────────┼───────────────────────
    [X 轴低值]             │           [X 轴高值]
                           │
          [你的产品]       │    [竞品 C]
                           │
                    [Y 轴低值标签]

### 解读
- [1-2 句话说明地图揭示的内容]
- [空白点或机会在哪里]
```

## 竞争 SWOT

针对每个竞品相对于你产品的 SWOT。

```markdown
## SWOT：[竞品] vs. [你的产品]

### 优势（对方相对我们）
- [他们真正胜过我们的地方——实事求是]

### 劣势（对方相对我们）
- [他们与我们相比不足之处——附证据]

### 机会（对我们而言）
- [他们产品中的可利用缺口]
- [他们忽略的细分群体]
- [他们未覆盖的传播角度]

### 威胁（来自对方）
- [他们正在快速改进的领域]
- [正在构建与我们重叠的功能]
- [可能改变市场认知的市场动作]
```

## 档案更新变更日志

更新档案时追加到任何档案的末尾。

```markdown
---

## 变更日志

| 日期 | 变更内容 | 来源 |
|------|----------|------|
| [日期] | 定价由 $X 上调至 $Y | 重新抓取定价页 |
| [日期] | 新发布 [功能] | 抓取更新日志 |
| [日期] | 域名权重由 X 变为 Y | 重新拉取 DataForSEO |
| [日期] | 新增 [集成] | 重新抓取集成页 |
```
