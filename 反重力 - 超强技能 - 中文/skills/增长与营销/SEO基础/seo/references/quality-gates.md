# 内容质量门控

## 按页面类型的最低字数

| 页面类型 | 最低字数 | 唯一内容 % | 备注 |
|-----------|-----------|-----------------|-------|
| 首页 | 500 | 100% | 必须清晰传达价值主张 |
| 服务/功能页面 | 800 | 100% | 产品的详细说明 |
| 位置页面（主要） | 600 | 60%+ | 城市总部或主要服务区域 |
| 位置页面（次要） | 500 | 40%+ | 卫星位置 |
| 博客文章 | 1,500 | 100% | 深入、有价值的内容 |
| 产品页面 | 400 | 80%+ | 独特描述、规格 |
| 分类页面 | 400 | 100% | 独特介绍，不仅是产品列表 |
| 关于页面 | 400 | 100% | 公司故事、团队、价值观 |
| 落地页 | 600 | 100% | 聚焦转化的内容 |
| FAQ 页面 | 800 | 100% | 全面的问答 |

## 位置页面阈值

### 警告级别（30+ 页面）
- ⚠️ 30+ 位置页面时**警告**
- 强制每页 60%+ 唯一内容
- 内容必须包含：
  - 独特的本地信息（地标、社区）
  - 特定位置的服务或产品
  - 本地团队或员工信息
  - 来自该地区的真实客户推荐

### 硬性停止（50+ 页面）
- 🛑 50+ 位置页面时**硬性停止**
- 需要用户明确说明理由
- 必须证明：
  - 在每个位置的合法业务存在
  - 每个页面的独特内容策略
  - 本地信号（Google Business Profile、本地评论）

### 为什么这很重要
Google 的门页算法会惩罚具有薄内容/重复内容的程序化位置页面。门页的迹象：
- 页面之间仅更改城市/州名称
- 无独特的本地信息
- 无本地业务信号
- 关键词堆砌的 URL

---

## 安全与有风险的程序化页面

### 大规模安全 ✅
| 页面类型 | 安全原因 |
|-----------|---------------|
| 集成页面 | 真实的设置文档，独特的技术内容 |
| 模板/工具页面 | 可下载资产，独特功能 |
| 词汇表页面 | 200+ 字的独特定义 |
| 产品页面 | 独特规格、图片、评论 |
| 用户资料页面 | 用户生成的独特内容 |

### 惩罚风险 ❌
| 页面类型 | 风险原因 |
|-----------|---------------|
| 仅交换城市的位置页面 | 重复内容，门页 |
| "[行业]最佳[工具]" | 通常内容薄弱，无行业特定价值 |
| "[竞争对手]替代品" | 需要真正的对比数据 |
| AI 批量生成内容 | 无独特价值，E-E-A-T 失败 |

## 标题标签要求

| 方面 | 要求 |
|--------|-------------|
| 最小长度 | 30 个字符 |
| 最大长度 | 60 个字符（Google 在约 60 个字符处截断） |
| 主要关键词 | 靠近开头 |
| 品牌名称 | 在末尾（如果包含） |
| 唯一性 | 每个页面必须有唯一标题 |

### 良好示例
- "奥斯汀紧急管道服务 | ABC Plumbing"
- "如何修复漏水水龙头：分步指南"
- "企业 SEO 软件 | 综合平台"

### 不良示例
- "首页"（太短，无描述性）
- "奥斯汀德克萨斯及周边地区所有管道需求的最佳管道服务"（太长）
- "ABC Plumbing - 管道 - 管道工 - 管道服务"（关键词堆砌）

---

## 元描述要求

| 方面 | 要求 |
|--------|-------------|
| 最小长度 | 120 个字符 |
| 最大长度 | 160 个字符（Google 在约 155-160 个字符处截断） |
| 行动号召 | 包含引人注目的 CTA |
| 主要关键词 | 自然包含 |
| 唯一性 | 每个页面必须有唯一描述 |

---

## 图片 Alt 文本要求

| 方面 | 要求 |
|--------|-------------|
| 适用范围 | 所有非装饰性图片 |
| 长度 | 10-125 个字符 |
| 内容 | 描述图片内容，而非"图片"或文件名 |
| 关键词 | 在相关处自然包含 |
| 装饰性图片 | 使用 `alt=""` 或 `role="presentation"` |

### 良好示例
- "专业管道工修理厨房水槽水龙头"
- "红色 2024 丰田凯美瑞轿车前视图"
- "现代办公室会议室团队会议"

### 不良示例
- "image.jpg"（文件名，非描述）
- "管道工 管道 管道工 服务"（关键词堆砌）
- "点击这里"（无描述性）

## 内部链接指南

| 页面类型 | 内部链接目标 |
|-----------|----------------------|
| 博客文章（1,500+ 字） | 5-10 个内部链接 |
| 服务页面 | 3-5 个内部链接 |
| 分类页面 | 链接到所有子页面 |
| 产品页面 | 2-4 个内部链接 |

### 锚文本规则
- 使用描述性锚文本（而非"点击这里"）
- 变化锚文本（不要总是使用完全匹配关键词）
- 链接到相关、相关的内容
- 确保无孤立页面（每个页面至少从另一个页面链接）

## 内容新鲜度信号

| 内容类型 | 更新频率 |
|--------------|------------------|
| 新闻/时事 | 数小时/数天内 |
| 博客文章（常青） | 每年审查 |
| 产品页面 | 规格变更时 |
| 服务页面 | 每季度审查 |
| 公司信息 | 变更发生时 |

### 必需元素
- 可见的发布日期（文章/博客）
- 最后更新日期（如有重大修订）
- 主要更新的更新日志（可选但推荐）
