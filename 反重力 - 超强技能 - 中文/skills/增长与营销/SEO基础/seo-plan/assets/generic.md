# 通用业务SEO策略模板

## 概述

本模板适用于不属于SaaS、本地服务、电商、出版商或代理机构类别的业务。请根据您的具体业务模式进行定制。

## 推荐网站架构

```
/
├── Home
├── /products (or /services)
│   ├── /product-1
│   ├── /product-2
│   └── ...
├── /solutions (if applicable)
│   ├── /solution-1
│   └── ...
├── /about
│   ├── /team
│   ├── /history
│   └── /values
├── /resources
│   ├── /blog
│   ├── /guides
│   ├── /faq
│   └── /glossary
├── /contact
├── /support
└── /legal
    ├── /privacy
    └── /terms
```

## 通用SEO原则

### 每个页面应具备
- 独特的标题标签（30-60字符）
- 独特的元描述（120-160字符）
- 与页面意图匹配的唯一H1
- 合理的标题层级（H1→H2→H3）
- 指向相关内容的内链
- 清晰的行动号召

### 所有网站的Schema
| 页面类型 | Schema类型 |
|----------|-----------|
| 首页 | Organization, WebSite |
| 关于 | Organization, AboutPage |
| 联系 | ContactPage |
| 博客 | Article, BlogPosting |
| FAQ | (FAQPage仅适用于政府/健康类) |
| 产品/服务 | Product 或 Service |

## 内容质量标准

### 最低字数要求
| 页面类型 | 最低字数 |
|----------|---------|
| 首页 | 500 |
| 产品/服务 | 800 |
| 博客文章 | 1,500 |
| 关于页面 | 400 |
| 落地页 | 600 |

### E-E-A-T要点
1. **经验（Experience）**：分享真实案例和案例研究
2. **专业（Expertise）**：展示资质和资格
3. **权威（Authoritativeness）**：获得提及和引用
4. **信任（Trustworthiness）**：完整联系信息，政策可见

## 技术基础

### 必备项
- [ ] 启用HTTPS
- [ ] 移动端响应式设计
- [ ] 配置robots.txt
- [ ] 提交XML站点地图
- [ ] 验证Google Search Console
- [ ] Core Web Vitals达标（LCP <2.5s, INP <200ms, CLS <0.1）

### 推荐项
- [ ] 关键页面的结构化数据
- [ ] 内链策略
- [ ] 优化404错误页面
- [ ] 消除重定向链
- [ ] 图片优化（WebP、懒加载）

## 内容优先级

### 第1阶段：基础（第1-4周）
1. 首页优化
2. 核心产品/服务页面
3. 关于和联系页面
4. 基础Schema实施

### 第2阶段：扩展（第5-12周）
1. 博客上线（2-4篇/月）
2. FAQ页面
3. 额外产品/服务页面
4. 内链审计

### 第3阶段：增长（第13-24周）
1. 持续内容发布
2. 链接建设外联
3. GEO优化
4. 性能优化

### 第4阶段：权威（第7-12个月）
1. 思想领导力内容
2. 原创研究
3. PR和媒体报道
4. 高级Schema

## 关键追踪指标

- 自然流量（整体及按版块）
- 关键词排名（品牌词和非品牌词）
- 自然流量转化率
- 已索引页面数
- Core Web Vitals得分
- 获得的外链数

## 定制要点

根据以下因素调整本模板：

1. **业务模式**：B2B vs B2C vs D2C
2. **地理范围**：本地、全国或国际
3. **内容类型**：产品导向 vs 内容密集
4. **竞争程度**：利基市场 vs 竞争激烈市场
5. **资源**：预算和团队能力

## 生成式引擎优化（GEO）检查清单

- [ ] 包含清晰、可引用的事实和统计数据，便于AI系统提取和引用
- [ ] 使用结构化数据（Schema.org）帮助AI系统理解内容
- [ ] 通过全面的内容集群建立主题权威
- [ ] 提供AI无法从其他地方获取的原创数据、研究或独特观点
- [ ] 在全网保持一致的实体信息（品牌、人物、产品）
- [ ] 使用清晰的标题、定义和分步格式组织内容
- [ ] 考虑在站点根目录添加 `llms.txt` 文件（AI爬虫的新兴约定：Google将其视为普通文本文件）
- [ ] 监控Google AI Overviews、ChatGPT、Perplexity和Bing Copilot中的AI引用
