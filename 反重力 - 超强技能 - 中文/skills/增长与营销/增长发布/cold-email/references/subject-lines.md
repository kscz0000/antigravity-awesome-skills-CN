# 主题行优化

主题行决定邮件是否被阅读。数据反直觉：**短、无聊、像内部邮件的主题行完胜。**

## 长度：2–4 个词

- 2 词主题行比 5 词多 **60% 打开率**（Lavender）。
- 从 2 词到 4 词，回复率下降 **17.5%**。
- 2–4 词产生 **46% 打开率**，而 10 词仅 34%（Belkins，550 万封邮件）。
- 移动端截断在 30–35 字符——简洁是实际必需。

## 内部伪装原则

看起来像来自同事而非供应商的主题行，打开率翻倍（Gong）。买家在打开前就在心理分类——如果看起来像销售，就被过滤。

**高表现示例：** "回复率" · "试用延迟" · "招聘机会" · "员工流失" · "Q2 预测" · "新患者" · "个性化问题" · "第二页"

## 大小写：小写胜出

全小写打开率最高（Gong，8500 万+ 封邮件）。小写看起来更个人/内部。对于冷外联，小写胜过首字母大写。

## 个性化：语境胜过名字

个性化主题行提升打开率 **26–50%**，但类型很重要：

- **主题行里的名字 → 回复率低 12%。** 暗示自动化。
- **语境个性化有效：** 痛点、竞争对手、触发事件、行业挑战。
- 用 {{痛点}}、{{竞争对手}}、{{共同点}}——而非 {{名字}}。

## 问题：仅当高度具体时

数据冲突：Belkins 说问题表现好（46% 打开率）。Lavender 说问题降低打开率 **56%**。解决方案：**具体痛点问题有效**（"需要 {{挑战}} 帮助吗？"），**通用问题失败**（"有个问题？" / "有 15 分钟吗？"）。默认用陈述句。

## 避免什么

| 反模式                                   | 影响                      |
| ---------------------------------------------- | --------------------------- |
| 销售语言（"增加"、"提升"、"ROI"）   | -17.9% 打开率                |
| 紧迫词（"ASAP"、"紧急"）               | 低于 36% 打开率             |
| 过多标点（"!!!" 或 "??"）          | -36% 打开率                  |
| 数字和百分比                        | -46% 打开率                  |
| 表情符号                                         | 损害 B2B 专业度    |
| 主题行推销产品                    | -57% 回复率                |
| 空/无主题行                          | +30% 打开率但 -12% 回复率 |
| 垃圾触发词（"免费"、"保证"、"立即行动"） | 投递风险         |

## C 级高管主题行

高管每天收到 300–400 封邮件，几秒内决定。邮件通过筛选后，他们比非 C 级 **多回复 23%**（6.4% 回复率）。

有效：超简洁、人性化、低调。"{{公司项目}}" · "谢谢" · "一个更新" · "一个问题" · 引用具体项目或触发事件。

任何"销售味"都会立即被拒。
