# 跟进序列

55% 的回复来自跟进，而非首封邮件。然而 48% 的销售人员从未跟进过一次。

## 数量：3–5 封邮件总计

- 单封邮件最高回复率：**8.4%**（Belkins）。
- 4–7 封邮件的活动达到 **27% 回复率**，而 1–3 封仅 9%（Woodpecker，2000 万封邮件）。
- 到第 4 次跟进，回复率下降 **55%**，垃圾投诉增加两倍。
- 解决方案：更长的序列可以覆盖不同的时间窗口。上限 4 次跟进（共 5 封邮件）。每封必须增加真正的新价值。

## 最佳节奏

每次触达间隔逐渐增加：

| 触达         | 天数   | 说明                                          |
| ------------- | ----- | ---------------------------------------------- |
| 首封邮件 | 0     | 个性化投入最大             |
| 跟进 1   | 3     | 等待 3 天回复率提升高达 31% |
| 跟进 2   | 7–8   | 不同角度                                |
| 跟进 3   | 14    | 新价值内容                                |
| 跟进 4   | 21–28 | 分手邮件                                  |

**最佳日期：** 周二至周四（周四峰值回复率 6.87%）。
**最佳时间：** 潜在客户当地时间的上午 9–11 点或下午 1–3 点。
**避免：** 周一上午（收件箱爆满）、周五下午（心不在焉）。

## 角度轮换

每次跟进必须独立成篇，同时朝目标推进。绝不能只是"顶一下"。

| 邮件       | 角度                                                      | 目的                    |
| ----------- | ---------------------------------------------------------- | -------------------------- |
| 首封     | 个性化钩子 + 核心价值主张 + 软性 CTA             | 引入问题/解决方案 |
| 跟进 1 | 不同角度、新价值内容（数据、洞察、资源） | 展示额外收益    |
| 跟进 2 | 社会认同 / 类似公司的案例研究             | 建立可信度          |
| 跟进 3 | 新洞察、行业趋势或相关资源          | 展示专业度      |
| 跟进 4 | 分手——承认沉默，留有余地             | 触发损失厌恶      |

每封邮件只增加 **一个新价值主张**（SalesBread）。这自然迫使角度不同。

## 分手邮件

利用损失厌恶——在撤回时创造稀缺性。Close.com 报告分手邮件对冷潜在客户有 **10–15% 回复率**。

**结构：**

1. 承认已多次联系
2. 确认他们可能不感兴趣
3. 说明这是最后一封邮件
4. 留有余地

**示例：**

> 我没收到回复，所以我假设现在不是合适时机。在我结束之前：[一句话洞察或资源]。如果情况有变，随时回复。否则，无妨——祝你[他们的目标]顺利。

**1-2-3 格式**（将摩擦降至近乎零）：

> 既然没收到回复，我简单说。回复一个数字：
>
> 1 — 感兴趣，聊聊
> 2 — 现在不行，3 个月后再联系
> 3 — 不感兴趣，请停止

**关键规则：** 发送分手邮件后，必须遵守。不要再联系该潜在客户。

## 扼杀回复率的短语

- "我没收到回复" → 会议预约率下降 **12%**（Gong）
- "只是跟进一下" → 零价值，暗示懒惰
- "顶到收件箱顶部" → 自以为是
- "看到我上一封邮件了吗？" → 制造愧疚感
- "跟进我之前的消息" → 通用，没增加任何内容

## 按资历调整 CTA

**高管/创始人：** 超低摩擦、好奇心驱动。"好奇吗？"或"值得 2 分钟吗？"

**中层管理者：** 更具体的价值。"想让我演示[公司]如何每周节省 15 小时吗？"

组织架构越高 = 能要求的摩擦越低。
