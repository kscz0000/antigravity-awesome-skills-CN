# 文档即营销

将文档打造成一个强大的营销渠道，吸引、转化并留住开发者。

## 本技能涵盖的内容

- **信息架构**：围绕可发现性与转化率组织文档
- **快速上手优化**：在 5 分钟内让开发者拿到首个成功
- **API 参考最佳实践**：可直接复制粘贴、真正可运行的代码
- **搜索优化**：让文档在开发者查询中排到前列
- **效果衡量**：在文档驱动的增长中真正重要的指标

## 何时使用本技能

- 规划或重构开发者文档
- 优化快速上手流程以提升转化率
- 改进 API 参考的易用性
- 将文档作为开发者获客渠道来搭建
- 通过更完善的自助内容减少支持工单

## 核心原则

优秀的文档是无形的营销。它在帮助开发者取得成功的同时降低摩擦、建立信任，并把访客变成拥护者。

## 速赢清单

1. 确保你的快速上手能在 5 分钟内交付首次成功
2. 为每个 API 接口提供可复制粘贴的代码示例
3. 在每个页面加上"下一步"链接
4. 测试所有代码示例，确保它们真的能跑起来
5. 加入"本页是否有帮助"反馈来衡量内容质量

## 相关技能

- `api-onboarding` —— 缩短从零到首次 API 调用的时间
- `sdk-dx` —— 让 SDK 设计反过来简化文档
- `developer-sandbox` —— 与文档互补的交互式演练环境
