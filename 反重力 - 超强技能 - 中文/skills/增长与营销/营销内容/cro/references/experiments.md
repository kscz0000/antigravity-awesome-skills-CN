# 页面 CRO 实验思路

按页面类型整理的全面 A/B 测试与实验清单。

## 内容目录
- 首页实验（首屏区块、信任与社会证明、功能与内容、导航与 UX）
- 价格页实验（价格呈现、价格 UX、异议处理、信任信号）
- 演示申请页实验（表单优化、页面内容、CTA 与路径）
- 资源/博客页实验（内容 CTA、资源板块）
- 落地页实验（信息匹配、转化聚焦、页面长度）
- 功能页实验（功能呈现、转化路径）
- 跨页面实验（全站测试、导航测试）

## 首页实验

### 首屏区块

| 测试 | 假设 |
|------|------------|
| 标题变体 | 具体 vs 抽象信息 |
| 副标题清晰度 | 增加/打磨以承接标题 |
| 首屏 CTA | 是否加入显眼 CTA |
| 首屏视觉格式 | 截图 vs GIF vs 插画 vs 视频 |
| CTA 按钮颜色 | 对比度与可见度 |
| CTA 按钮文字 | "开始免费试用" vs "立即开始" vs "查看演示" |
| 交互式演示 | 立即让访客上手体验产品 |

### 信任与社会证明

| 测试 | 假设 |
|------|------------|
| Logo 摆放位置 | 首屏 vs 首屏之下 |
| 案例研究置入首屏 | 立即展示成果 |
| 信任徽章 | 增加安全、合规、奖项 |
| 标题中的社会证明 | "加入 10,000+ 团队"的信息 |
| 推荐语摆放 | 首屏之上 vs 独立板块 |
| 视频推荐语 | 比文字引述更具吸引力 |

### 功能与内容

| 测试 | 假设 |
|------|------------|
| 功能呈现方式 | 图标+描述 vs 详细板块 |
| 板块排序 | 把高价值功能前移 |
| 次级 CTA | 在页面中加/减 |
| 收益 vs 功能导向 | 以成果开篇 |
| 对比板块 | 对比竞品或现状 |

### 导航与 UX

| 测试 | 假设 |
|------|------------|
| 悬浮导航 | 含 CTA 的常驻导航 |
| 导航菜单顺序 | 高优先项位于两端 |
| 导航中的 CTA 按钮 | 加入显眼按钮 |
| 在线支持组件 | 真人客服 vs AI 机器人 |
| 页脚优化 | 更清晰的次级转化 |
| 退出意图弹窗 | 挽回即将离开的访客 |

## 价格页实验

### 价格呈现

| 测试 | 假设 |
|------|------------|
| 按年 vs 按月展示 | 突出节省或简化 |
| 价格点 | $99 vs $100 vs $97 的心理效应 |
| "最受欢迎"徽章 | 高亮目标套餐 |
| 套餐档数 | 3 vs 4 vs 2 个可见选项 |
| 价格锚定 | 通过排序设定预期 |
| 定制企业套餐 | 是否展示 vs "联系销售" |

### 价格 UX

| 测试 | 假设 |
|------|------------|
| 价格计算器 | 让用量计费更清晰 |
| 引导式价格流程 | 多步向导 vs 对比表 |
| 功能对比形式 | 表格 vs 可展开板块 |
| 月/年切换开关 | 突出节省金额 |
| 套餐推荐测验 | 帮助访客选择 |
| 结账流程长度 | 选完套餐之后还需几步 |

### 异议处理

| 测试 | 假设 |
|------|------------|
| FAQ 板块 | 回应关于价格的异议 |
| ROI 计算器 | 论证价值 vs 成本 |
| 退款保证 | 醒目的展示位 |
| 按用户数计费 | 团队套餐的可读性 |
| 功能包含说明 | 每档各自包含什么 |
| 竞品对比 | 并排的价值对比 |

### 信任信号

| 测试 | 假设 |
|------|------------|
| 价值型推荐语 | 专门针对 ROI 的引述 |
| 客户 logo | 紧邻价格板块 |
| 评分 | G2/Capterra 评分 |
| 案例研究片段 | 具体的价格/价值成果 |

## 演示申请页实验

### 表单优化

| 测试 | 假设 |
|------|------------|
| 字段数量 | 字段越少，完成率越高 |
| 多步 vs 单步 | 进度条带来的激励 |
| 表单位置 | 首屏之上 vs 内容之后 |
| 电话字段 | 是否包含 |
| 字段自动填充 | 隐藏可自动填充的字段 |
| 表单标签 | 在字段内 vs 在字段上方 |

### 页面内容

| 测试 | 假设 |
|------|------------|
| 表单之上阐述收益 | 在请求前强化价值 |
| 演示预览 | 视频/GIF 展示演示体验 |
| "你将了解什么" | 明确设定预期 |
| 表单旁推荐语 | 在决策点降低摩擦 |
| 表单下方 FAQ | 回应常见异议 |
| 视频 vs 文字 | 解释价值的载体 |

### CTA 与路径

| 测试 | 假设 |
|------|------------|
| CTA 文字 | "预约演示" vs "安排 15 分钟通话" |
| 自助选项 | 与人工选项并存的即时演示 |
| 个性化文案 | 基于访客数据/来源 |
| 去除导航 | 减少页面干扰 |
| 日历集成 | 内联预约 vs 外部链接 |
| 资格分流 | 有人自助，有人走销售 |

## 落地页实验

### 信息匹配

| 测试 | 假设 |
|------|------------|
| 标题匹配 | 完全对齐广告文案 |
| 视觉匹配 | 与广告素材对齐 |
| 优惠对齐 | 与广告承诺一致 |
| 面向不同受众的页面 | 按细分人群使用不同落地页 |

### 转化聚焦

| 测试 | 假设 |
|------|------------|
| 去除导航 | 单一焦点页面 |
| CTA 重复 | 在页内多处放置 |
| 表单 vs 按钮 | 直接收集 vs 跳转 |
| 紧迫感/稀缺性 | 如真实有效，可测试话术 |
| 社会证明密度 | 数量与摆放 |
| 视频加入 | 用视频解释优惠 |

### 页面长度

| 测试 | 假设 |
|------|------------|
| 短 vs 长 | 快速转化 vs 完整说服 |
| 仅首屏之上 | 极少滚动 |
| 板块排序 | 最重要的内容前置 |
| 去除页脚 | 消除导航 |
