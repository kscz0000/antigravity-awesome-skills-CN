# 当前状态评分表 — 17 节评分量表

本 17 节评分表是每份营销计划第 3 节（"当前状态"）的单一事实来源。根据可用材料对每节评 0–5 分，然后撰写 2–4 句"形态解读"，命名优势和缺口聚集的位置。

## 如何评分

**基于丰富材料。** 当团队分享了演示文稿、先前内容审计、品牌声音文档、启动会议记录、App Store 和分析快照时——根据这些制品对每节评分。在章节标题中标注"基于材料评分"，以便团队在有更好数据时提出异议。

**基于独立评分的审计。** 如果团队已运行过评分的当前状态评估（任何格式），直接纳入这些分数。不要重做工作——记录评分表评分日期，并标注此后材料发生变化的章节。

无论哪种方式，输出相同：一张 17 行评分表、总分 85 分制、一段形态解读。

## 17 个评分节（每节 0–5 分）

### 1. 定位
**评分内容：** 品类声明清晰度、差异化、跨表面一致性（首页、App Store、融资演示文稿、创始人传播信息）。

**评分指南：**
- 0 = 无任何定位
- 2 = 跨表面不一致；团队无法即兴表述
- 4 = 清晰、原创、基本一致；小表面缺口
- 5 = 独特、品类定义级、每个表面一致

**映射到 AARRR：** 跨切面 — 喂养每个阶段。

### 2. 客户研究
**评分内容：** 客户研究的深度和时效性、ICP 清晰度、客户原声捕获。

**评分指南：**
- 0 = 无正式研究，仅创始人直觉
- 2 = 有些研究但过时或一次性的
- 4 = 活跃的研究实践，客户语言已捕获
- 5 = 持续研究，客户语言流入文案/产品/传播信息

**映射到 AARRR：** 跨切面 — 尤其喂养获取（渠道选择）和激活（引导声音）。

### 3. 首页
**评分内容：** 标题清晰度、声音一致性、转化架构、移动端体验。

**评分指南：**
- 0 = 通用/损坏/偏离品牌
- 2 = 功能正常但表现不佳；声音基本缺失
- 4 = 清晰、声音一致、有转化；小优化机会
- 5 = 独特、转化强劲、完全声音一致

**映射到 AARRR：** 获取 + 激活。

### 4. 销售/产品页
**评分内容：** 专用产品/定价/功能页面的存在和质量。SKU 是否有文档？定价是否可扫描？追加销售是否可见？

**评分指南：**
- 0 = 无专用页面
- 2 = 页面存在但过时或偏离声音
- 4 = 主要产品质量页面；次要产品有缺口
- 5 = 每个产品、层级和追加销售都有高转化页面

**映射到 AARRR：** 获取 + 收入。

### 5. 转化页
**评分内容：** 针对特定活动、渠道或用例的落地页。`/partner`、`/science`、`/ambassadors`、`/eye-mask` 类型的页面。

**评分指南：**
- 0 = 无转化页
- 2 = 存在一两个；其余需要的页面缺失
- 4 = 大部分需要的转化页存在；质量良好
- 5 = 完整转化页库，每页高转化

**映射到 AARRR：** 获取 + 激活。

### 6. 竞品对比
**评分内容：** "vs. {竞品}"页面、对比内容的存在。品牌是否承认替代方案，还是假装它们不存在？

**评分指南：**
- 0 = 无任何内容 — 刻意回避竞品提及
- 2 = 有些内容但薄弱或隐藏
- 4 = 前 2–3 个竞品有扎实对比页
- 5 = 全面对比库；SEO 定向；高转化

**映射到 AARRR：** 获取（考虑阶段 SEO + 销售赋能）。

### 7. 资源/内容
**评分内容：** 博客、知识库、科学页、白皮书、研究、创始人文章、播客。

**评分指南：**
- 0 = 无内容表面
- 2 = 博客存在但过时或单薄
- 4 = 活跃内容生产；多种格式
- 5 = 内容是护城河 — 专有研究、命名支柱、日级产出量

**映射到 AARRR：** 获取。

### 8. 引导
**评分内容：** 新用户引导（应用内 + 邮件）。达到价值时间、完成率、品牌声音一致性。

**评分指南：**
- 0 = 无引导流程
- 2 = 引导存在但损坏、偏离声音或表现不佳
- 4 = 扎实引导；明确瓶颈已识别
- 5 = 已测试、已优化、品牌一致；激活率达到品类前四分位

**映射到 AARRR：** 激活。

### 9. 邮件生命周期
**评分内容：** 生命周期邮件程序的存在和质量。欢迎/引导/购后/流失/召回。

**评分指南：**
- 0 = 无生命周期邮件
- 2 = 有些流程已起草但未上线，或已上线但过时
- 4 = 核心流程在线并表现良好；次要流程有缺口
- 5 = 完整生命周期在线、已分群、表现高于品类基准

**映射到 AARRR：** 留存（+ 引导邮件的激活）。

### 10. 销售材料
**评分内容：** 销售演示文稿、一页纸、演示、案例研究、定价表。（适用于 B2B/混合公司 — 纯 D2C 可标为 N/A 或低分无负面含义。）

**评分指南：**
- 0 = 无销售材料
- 2 = 创始人使用演示文稿但其他材料单薄
- 4 = 扎实的销售工具包；销售代表可自助获取内容
- 5 = 全面材料；季度更新；异议处理库存在

**映射到 AARRR：** 获取 + 收入（B2B）。

### 11. 传播信息
**评分内容：** 跨表面的声音、语调、词汇、信息层级。品牌声音是否有文档、一致、独特？

**评分指南：**
- 0 = 无声音文档；表面不一致
- 2 = 声音存在于创始人脑中但未操作化
- 4 = 有文档的声音；跨表面基本一致
- 5 = 独特声音；有文档；每个表面都尊重它；声音是护城河

**映射到 AARRR：** 跨切面。

### 12. 定价
**评分内容：** 定价结构清晰度、打包逻辑、近期压力测试、标价 vs 实际价格对账。

**评分指南：**
- 0 = 定价超过一年未压力测试；结构不清
- 2 = 标价存在但计划组合/折扣混淆了读数
- 4 = 清晰定价；近期测试；LTV 数学已知
- 5 = 定价季度测试；打包优化；扩展杠杆已知

**映射到 AARRR：** 收入。

### 13. CRO（转化率优化）
**评分内容：** 测试节奏、埋点、A/B 历史、统计严谨性。

**评分指南：**
- 0 = 无测试运行；无埋点
- 2 = 有些临时测试；无统计严谨性
- 4 = 定期测试节奏；有些胜出
- 5 = 持续测试程序；实验文化；已记录的胜出

**映射到 AARRR：** 跨切面（在激活 + 收入影响最大）。

### 14. GTM 发布
**评分内容：** 过去发布执行的质量。产品发布、功能发布、活动发布。

**评分指南：**
- 0 = 无结构化发布；仅"软发布"
- 2 = 有些发布但执行不均
- 4 = 近期发布扎实；有操作手册
- 5 = 可重复的发布动作；Product Hunt #1；按需获得媒体覆盖

**映射到 AARRR：** 获取 + 激活。

### 15. 广告（付费）
**评分内容：** 付费获取状态。活跃活动、渠道、CAC 追踪、创意质量。

**评分指南：**
- 0 = 无付费获取
- 2 = 有些付费但无结构/浪费
- 4 = 付费在 2–3 个渠道投放，单位经济为正
- 5 = 成熟的付费栈；CAC/LTV 已理解；创意每周迭代

**映射到 AARRR：** 获取。

**注意：** 对于无付费预算的 Pre-seed 客户，评 0 分*不应*视为弱点 — 这反映融资阶段，而非营销失败。

### 16. SEO
**评分内容：** 有机搜索表现。域名评级、排名关键词、有机流量、内容集群策略。

**评分指南：**
- 0 = 无 SEO；新域名或零权威
- 2 = 有些内容但无策略；仅品牌词排名
- 4 = 已建立内容集群；有机流量增长；DR 25+
- 5 = SEO 是护城河；DR 40+；千余排名关键词；持续内容生产

**映射到 AARRR：** 获取。

### 17. 国际化
**评分内容：** 地理扩张、语言本地化、区域特定定价。

**评分指南：**
- 0 = 仅美国/英文；无国际考量
- 2 = 国际用户存在但未被服务（一种语言、一种货币）
- 4 = 多语言、区域特定定价、新市场的 GTM 操作手册
- 5 = 国际化是优势；多区域收入；本地化 GTM

**映射到 AARRR：** 获取。

**注意：** 对大多数早期公司，国际化得分 0–1 是合适的。不要因早期公司还没有国际化操作手册而惩罚它们。

## 如何计算总分 + 解读形态

**总分 = 所有 17 项分数之和。满分 85 分。**

总分不如*形态*重要。在评分表之后，撰写 2–4 句"形态解读"：

> *"在{强项章节}得分高，在{弱项章节}得分低。该形态就是本计划其余部分要填补的缺口 — 第 X 节（AARRR 阶段）最长，因为那里的缺口最大。"*

## 常见形态

### "强声音/传播信息，弱分发"
- 高分：定位（#1）、客户研究（#2）、传播信息（#11）
- 低分：SEO（#16）、广告（#15）、GTM 发布（#14）
- 解读：创始人是强叙事者但分发未跟上。计划侧重获取 + 付费层准备。

### "强获取，弱转化"
- 高分：SEO（#16）、资源（#7）、广告（#15）
- 低分：首页（#3）、引导（#8）、转化页（#5）、定价（#12）
- 解读：流量进来但不转化。计划侧重激活 + 收入。

### "强转化，弱留存"
- 高分：引导（#8）、首页（#3）、定价（#12）
- 低分：邮件生命周期（#9）、CRO（#13）
- 解读：用户注册并付费但流失。计划侧重留存。

### "强产品，弱其他一切"
- 高分：仅定位（#1）和客户研究（#2）— 创始人了解客户
- 低分：所有运营项
- 解读：前营销阶段。计划以基础建设为主。第一季度是基石修复。

### "强近期收入，弱复利"
- 高分：广告（#15）、销售材料（#10）、定价（#12）
- 低分：SEO（#16）、资源（#7）、推荐机制
- 解读：绩效营销支撑业务。计划侧重在付费进一步规模化前建设复利渠道。

## 当分数存在主观性时

有些章节比其他章节更容易从外部评分。主观性层级：

- **客观（数据驱动）：** SEO（#16）、广告（#15）、邮件生命周期（#9）、引导（#8）— 有分析支撑
- **半客观：** 定价（#12）、CRO（#13）、转化页（#5）、销售材料（#10）— 有可见制品可评估
- **主观（判断）：** 定位（#1）、传播信息（#11）、客户研究（#2）、资源（#7）— 需解读

对主观章节，将理由写入"备注"列，以便团队在有异议时提出。

## 当存在先前评分的审计时

如果团队已有来自任何当前状态评估的评分输出，直接纳入这些分数 — 不要重做工作。将先前评分视为其覆盖章节的基准事实。

如果先前评分是在数周前进行的且此后材料已变化（新上线的流程、新上线的内容、重新定位等），标注"评分日期 YYYY-MM-DD；此后材料已变化"，并更新你有当前证据的特定分数。
