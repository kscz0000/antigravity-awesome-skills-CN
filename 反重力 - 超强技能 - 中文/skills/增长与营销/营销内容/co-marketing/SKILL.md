## 何时使用此技能

- 寻找潜在的联营营销合作伙伴
- 与特定合作伙伴头脑风暴营销活动创意
- 规划联合发布或促销活动
- 评估合作伙伴契合度
- 构建联营营销协议

## 合作伙伴识别框架

### 1. 受众重叠分析

最佳合作伙伴与您共享受众，但不会竞争同一预算。

**理想合作伙伴特征：**
- 相同的买家画像，但解决的问题不同
- 在工作流中相邻（之前、之后或与您的工具并行）
- 公司阶段和客户规模相似
- 互补而非竞争

**识别合作伙伴的问题：**
- 您的客户已经在使用哪些工具？
- 他们在使用您的产品之前/之后用什么？
- 谁还在向您的 ICP 销售？
- 客户最常请求哪些集成？

### 2. 合作伙伴评分标准

对潜在合作伙伴按以下维度打分（1-5）：

| 标准 | 评估内容 |
|----------|------------------|
| **受众契合度** | 他们的受众与您的 ICP 匹配度如何？ |
| **受众规模** | 他们是否具有值得合作的覆盖范围？ |
| **品牌一致性** | 您是否愿意与他们建立关联？ |
| **互动质量** | 他们是否拥有活跃且参与度高的受众？ |
| **互惠潜力** | 您能否为他们提供同等的价值？ |
| **执行难度** | 他们是否有合作团队？有联营营销的历史吗？ |

### 3. 在哪里寻找合作伙伴

**集成生态系统：**
- 您现有的集成合作伙伴
- 同一应用市场中同品类的工具
- 您的产品所接入的平台

**相邻品类：**
- 解决您之前问题的工具
- 解决您之后问题的工具
- 同一角色使用但工作流不同的工具

**社区信号：**
- 谁赞助同一播客/Newsletter？
- 谁在同一会议上参展？
- 谁在相同社区中活跃？
- 您的受众分享谁的内容？

**数据来源：**
- 使用 Crossbeam 或 Reveal 进行账户重叠分析
- 客户调研（"您还使用什么？"）
- G2/Capterra 相邻品类
- 招聘启事中同时提及您的工具与其他工具

## 联营营销活动类型

### 内容合作

| 形式 | 投入 | 线索共享 | 最适合 |
|--------|--------|--------------|----------|
| **联合署名博客文章** | 低 | 共享署名、链接互换 | 思想领导力、SEO |
| **联合电子书/指南** | 中 | 留资后分配线索 | 线索获取、更深入的主题 |
| **研究报告** | 高 | 留资后分配线索 | 权威性、公关 |
| **互访 Newsletter 互换** | 低 | 各自保留自己的线索 | 受众曝光 |
| **播客互访** | 低 | 各自保留自己的线索 | 关系建立 |

### 网络研讨会与活动

| 形式 | 投入 | 最适合 |
|--------|--------|----------|
| **联合网络研讨会** | 中 | 线索获取、产品教育 |
| **虚拟峰会小组讨论** | 中 | 多合作伙伴曝光 |
| **联合工作坊** | 高 | 动手教育、更深入的互动 |
| **联合展位** | 中 | 成本分摊、受众重叠 |
| **联合酒会/晚宴** | 低 | 活动期间关系建立 |

### 产品与集成营销

| 形式 | 投入 | 最适合 |
|--------|--------|----------|
| **集成发布** | 中 | 现有的集成合作伙伴 |
| **联合案例研究** | 中 | 共享客户 |
| **"强强联合"落地页** | 低 | 集成发现 |
| **打包或折扣** | 中 | 转化提升、交叉销售 |
| **应用内交叉推广** | 中 | 用户激活 |

### 社区与社交

| 形式 | 投入 | 最适合 |
|--------|--------|----------|
| **社交媒体接管** | 低 | 受众曝光 |
| **联合赠品/活动** | 低 | 列表建设、互动 |
| **Slack/Discord 社区合作** | 低 | 社区建设 |
| **联合 AMA 或 Twitter Space** | 低 | 思想领导力 |

## 合作伙伴营销活动头脑风暴

与特定合作伙伴进行头脑风暴时，请考虑：

### 1. 共同的受众时刻

- 哪些触发事件对双方受众都很重要？
- 哪些季节性时刻与双方产品契合？
- 哪些行业趋势会影响双方的客户群？

### 2. 组合价值主张

- 客户使用两个工具可以实现哪些单独无法实现的目标？
- 这种组合能实现什么工作流？
- 这种集成能解决哪些痛点？

### 3. 双方各自带来的独特资产

| 您的资产 | 他们的资产 |
|-------------|--------------|
| 您的受众规模/互动 | 他们的受众规模/互动 |
| 您的内容专长 | 他们的内容专长 |
| 您的产品能力 | 他们的产品能力 |
| 您的品牌信誉 | 他们的品牌信誉 |
| 您的客户故事 | 他们的客户故事 |

### 4. 营销活动创意提示

提出以下问题以激发创意：
- "如果我们必须在 2 周内推出一个项目，我们会创造什么？"
- "我们双方的受众迫切需要什么内容？"
- "什么会让客户说'终于，有人做了这件事'？"
- "我们可以联合提供哪些独家内容？"
- "我们都拥有哪些数据可以讲述引人入胜的故事？"

## 联系潜在合作伙伴

### 冷邮件模板

```
Subject: [Your Company] + [Their Company] co-marketing idea

Hey [Name],

I'm [Role] at [Your Company]. We [one-line description].

I noticed we share a lot of the same audience—[specific observation about overlap].

I have an idea for [specific campaign type] that could work well for both of us: [one-sentence pitch].

Would you be open to a quick call to explore?

[Your name]
```

### 通话前需要准备的内容

1. **账户重叠数据**（可通过 Crossbeam/Reveal 获取）
2. **2-3 个具体的营销活动创意**（不只是"我们做点什么吧"）
3. **您的受众指标**（列表规模、流量、参与度）
4. **过往合作案例**（展示您能够执行）
5. **明确的需求**（您想从他们那里得到什么，您将提供什么）

## 构建合作伙伴关系

### 关键对齐问题

- **线索归属**：如何分配或共享线索？
- **推广承诺**：各方将如何进行推广？
- **资产创建**：谁创建什么？谁来审批？
- **时间表**：每个阶段何时进行？
- **成功指标**：如何衡量成功？
- **后续跟进**：如果效果良好，是否会继续合作？

### 简单的联营营销协议大纲

1. **活动描述**：你们将共同做什么
2. **职责分工**：谁负责什么
3. **时间表**：关键日期和截止时间
4. **线索处理**：线索如何捕获、共享、跟进
5. **推广**：各方的最低承诺
6. **品牌**：Logo 使用、审批流程
7. **成本**：谁支付哪些费用（如有）
8. **指标共享**：活动后将共享哪些数据

## 衡量联营营销成功

### 量化指标

- 线索数量（总数及每位合作伙伴）
- 线索质量（MQL/SQL 转化率）
- 归因收入
- 受众增长（新订阅者、关注者）
- 内容互动（浏览量、下载量、分享量）

### 定性指标

- 协作顺畅程度
- 合作伙伴响应速度
- 受众接受度
- 品牌提升
- 关系强化，为未来活动铺路

## 联营营销清单

### 合作伙伴识别
- [ ] 列出客户已经在使用的工具
- [ ] 通过 Crossbeam/Reveal 检查账户重叠
- [ ] 为前 5 名潜在合作伙伴评分
- [ ] 研究他们过去的联营营销活动

### 营销活动规划
- [ ] 就营销活动类型和目标达成一致
- [ ] 定义线索共享安排
- [ ] 分配职责和截止时间
- [ ] 设定成功指标

### 执行
- [ ] 创建共享资产（落地页、内容等）
- [ ] 协调推广计划
- [ ] 为双方团队提供沟通要点简报

### 活动后
- [ ] 与合作伙伴共享指标
- [ ] 复盘有效和无效之处
- [ ] 讨论未来合作机会

## 任务特定问题

1. 您是在寻找合作伙伴，还是正在与特定合作伙伴规划营销活动？
2. 您最感兴趣哪种类型的联营营销？（内容、活动、集成、社区）
3. 您的受众规模是多少？（邮件列表、社交关注、流量）
4. 您是否有现有的集成合作伙伴？
5. 您之前做过联营营销吗？哪些有效，哪些无效？
6. 您的联营营销时间表和预算是怎样的？
