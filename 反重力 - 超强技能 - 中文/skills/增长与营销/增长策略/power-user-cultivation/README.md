# power-user-cultivation

**将用户转化为倡导者与贡献者** — 识别、培育和赋能你的开发者冠军。

## 功能说明

帮助你识别最活跃的开发者，并创建将他们转化为倡导者、内容创作者和贡献者的项目。不是强制传道 — 而是为热爱你产品的开发者提供真正的价值交换。

## 何时使用

- 识别用户群中的潜在倡导者
- 构建冠军或大使计划
- 为开源项目创建贡献者体验
- 启动推荐计划
- 发起用户生成内容倡议
- 当你说"高级用户"、"开发者倡导者"、"冠军计划"、"UGC"时

## 快速开始

```
/power-user-cultivation

Help me identify potential advocates from our user base and design
a champion program. We have about 5,000 active developers using our API.
```

## 高级用户光谱

| 层级 | 行为 | 你的回应 |
|------|------|----------|
| 活跃用户 | 定期使用产品 | 持续交付 |
| 参与用户 | 参与社区互动 | 快速响应 |
| 倡导者 | 自发推荐产品 | 优先体验，直接沟通渠道 |
| 冠军 | 创建内容，帮助他人 | 正式计划 |
| 贡献者 | 贡献代码/文档 | 贡献者计划 |

## 核心原则

1. **不要推动所有人沿光谱上升 — 在他们所在的位置与之相遇**
2. **不要设内容配额 — 倡导者是因为想做才创作**
3. **推荐用额度而非现金（减少传销感）**
4. **认可必须具体、公开、及时**
5. **从小处着手（5-10 名冠军）再逐步扩大**

## 识别倡导者

寻找以下信号：
- 使用量排名前 10%
- 回答社区问题
- 创建了关于你产品的内容
- 提交了有深度的 Bug 报告
- 正面的社交媒体提及
