# 社区建设（community-building）

**构建蓬勃发展的开发者社区。**

## 功能说明

引导你在 Discord、Slack 以及各类论坛中构建并管理开发者社区。涵盖频道结构、新成员引导、活跃度运营、处理有害行为以及社区驱动的增长策略。

## 适用场景

- 启动一个新的开发者社区
- 改进已有的社区
- 提升成员活跃度
- 应对内容审核挑战
- 当你提及"开发者社区"、"Discord 服务器"、"Slack 社区"或"社区互动"时

## 快速开始

```
/community-building

I'm starting a Discord community for our CLI tool.
Target: DevOps engineers using Kubernetes.
Goal: Build engaged community for support and feedback.
```

该技能将依次完成：
1. 加载你的开发者受众画像
2. 设计频道结构
3. 创建新成员引导体验
4. 规划活跃度运营活动
5. 制定社区管理规范

## 核心框架

### 平台选型

| 平台 | 最适合 |
|------|--------|
| Discord | 开发者工具、开源软件、实时交流 |
| Slack | 企业级、B2B SaaS |
| GitHub Discussions | 开源软件、异步交流、可被搜索 |
| Discourse | 长文内容、利于 SEO |

### 频道结构

```
INFORMATION: welcome, announcements, rules
GENERAL: general, off-topic, show-what-you-built
SUPPORT: help, troubleshooting, feature-requests
COMMUNITY: jobs, events, content
```

### 活跃度运营计划

| 星期 | 话题 |
|------|------|
| 周一 | "你这周在做什么？" |
| 周三 | 技术问答 |
| 周五 | "分享你这周上线的东西" |

### 社区准则的执行

1. 警告（首次违规且出于善意）
2. 临时禁言（重复出现问题）
3. 临时封禁（严重违规）
4. 永久封禁（性质恶劣或屡次违规）
