# 平台规格参考

各广告平台的完整字符限制、格式要求和最佳实践。

---

## Google Ads

### 响应式搜索广告 (RSA)

| 元素 | 字符限制 | 必填 | 备注 |
|------|----------|------|------|
| 标题 | 30字符 | 最少3条，最多15条 | 任意3条可能同时展示 |
| 描述 | 90字符 | 最少2条，最多4条 | 任意2条可能同时展示 |
| 显示路径1 | 15字符 | 可选 | 出现在 URL 域名后 |
| 显示路径2 | 15字符 | 可选 | 出现在路径1后 |
| 最终 URL | 无限制 | 必填 | 落地页 URL |

**组合规则：**
- Google 选择最多3个标题和2个描述进行展示
- 标题以 " | " 分隔或堆叠显示
- 除非固定，任何标题可出现在任何位置
- 固定会降低 Google 的优化能力 — 谨慎使用

**固定策略：**
- 如品牌指南要求，将品牌名固定在位置1
- 将最强 CTA 固定在位置2或3
- 大多数标题保持不固定，以便机器学习

**标题组合建议（15条标题）：**
- 3-4条关键词导向（匹配搜索意图）
- 3-4条利益导向（用户获得什么）
- 2-3条社会证明（数字、奖项、客户）
- 2-3条 CTA 导向（行动号召）
- 1-2条差异化（为什么选你而非竞品）
- 1条品牌名标题

**描述组合建议（4条描述）：**
- 1条利益 + 证明点
- 1条功能 + 结果
- 1条社会证明 + CTA
- 1条紧迫感/优惠 + CTA（如适用）

### 效果最大化广告 (Performance Max)

| 元素 | 字符限制 | 备注 |
|------|----------|------|
| 标题 | 30字符（必填5条） | 用于各种广告位的短标题 |
| 长标题 | 90字符（必填5条） | 用于展示、视频、发现 |
| 描述 | 90字符（必填1条，最多5条） | 配合各种广告格式 |
| 商家名称 | 25字符 | 必填 |

### 展示广告

| 元素 | 字符限制 |
|------|----------|
| 标题 | 30字符 |
| 长标题 | 90字符 |
| 描述 | 90字符 |
| 商家名称 | 25字符 |

## Meta Ads (Facebook & Instagram)

### 单图 / 视频 / 轮播

| 元素 | 推荐 | 最大 | 备注 |
|------|------|------|------|
| 正文文本 | 125字符 | 2,200字符 | 图片上方文字；约125字符后截断 |
| 标题 | 40字符 | 255字符 | 图片下方；约40字符后截断 |
| 描述 | 30字符 | 255字符 | 标题下方；可能不显示 |
| URL 显示链接 | 40字符 | 无 | 可选的自定义显示 URL |

**广告位特定说明：**
- **信息流**：所有元素都显示；正文文本最显眼
- **Stories/Reels**：正文文本叠加显示；控制在72字符以内
- **右侧栏**：仅标题可见；跳过描述
- **Audience Network**：因发布商而异

**最佳实践：**
- 在正文文本开头放置吸引点（前125字符）
- 较长正文文本使用换行提高可读性
- 表情符号：测试，但不要过度使用 — 每条广告最多1-2个
- 正文文本中的提问可增加互动
- 标题应为清晰的 CTA 或价值声明

### 潜在客户广告（即时表单）

| 元素 | 限制 |
|------|------|
| 问候标题 | 60字符 |
| 问候描述 | 360字符 |
| 隐私政策文本 | 200字符 |

## LinkedIn Ads

| 元素 | 推荐 | 最大 | 备注 |
|------|------|------|------|
| 介绍文本 | 150字符 | 600字符 | 图片上方；约150字符后截断 |
| 标题 | 70字符 | 200字符 | 图片下方 |
| 描述 | 100字符 | 300字符 | 仅在 Audience Network 显示 |

### 轮播广告

| 元素 | 限制 |
|------|------|
| 介绍文本 | 255字符 |
| 卡片标题 | 45字符 |
| 卡片数量 | 2-10张 |

### 消息广告 (InMail)

| 元素 | 限制 |
|------|------|
| 主题行 | 60字符 |
| 消息正文 | 1,500字符 |
| CTA 按钮 | 20字符 |

### 文字广告

| 元素 | 限制 |
|------|------|
| 标题 | 25字符 |
| 描述 | 75字符 |

**LinkedIn 特定指南：**
- 专业语调，但不枯燥
- 使用受众认可的行业术语
- 数据和统计表现良好
- 避免消费者式的夸张（"太棒了！""不可思议！"）
- 同行第一人称证言更有共鸣

## TikTok Ads

### 信息流广告

| 元素 | 推荐 | 最大 | 备注 |
|------|------|------|------|
| 广告文本 | 80字符 | 100字符 | 视频上方 |
| 显示名称 | 无 | 40字符 | 品牌名称 |
| CTA 按钮 | 平台选项 | 预定义 | 从 TikTok 选项中选择 |

### Spark Ads（助推原生内容）

| 元素 | 备注 |
|------|------|
| 文案 | 使用原帖文案 |
| CTA 按钮 | 由广告主添加 |
| 显示名称 | 原创作者账号 |

**TikTok 特定指南：**
- 原生内容优于精致广告
- 前2秒决定是否继续观看
- 使用热门音乐和格式
- 文字叠加必不可少（大多数人静音观看）
- 仅支持竖版视频（9:16）

## 字符计数技巧

- **空格** 在所有平台都算字符
- **表情符号** 根据平台算1-2个字符
- **特殊字符**（|、& 等）算1个字符
- **URL** 在正文中计入字符限制
- **动态关键词插入**（`{KeyWord:default}`）可能超出限制 — 设置安全的默认值
- 发布前务必在平台的广告预览中验证

## 多平台创意适配

同时为多个平台创作时，从最严格的格式开始：

1. **Google 搜索标题**（30字符）—— 迫使信息最精炼
2. **扩展到 Meta 标题**（40字符）—— 增加一两个词
3. **扩展到 LinkedIn 介绍文本**（150字符）—— 增加背景和证明
4. **扩展到 Meta 正文文本**（125+字符）—— 完整吸引点和价值主张

这种级联方法确保你的核心信息在各处都有效，然后为允许更多空间的平台进行丰富。
