# 市场规模测算数据源

市场研究和规模分析的可信来源精选列表。

## 行业研究报告

### 付费研究机构

**Gartner** (https://www.gartner.com)
- 技术市场预测和规模测算
- 魔力象限用于竞争定位
- 典型价格：$5K-$50K/份
- 最适用于：企业软件、IT 服务、新兴技术

**Forrester** (https://www.forrester.com)
- 商业技术和数字化转型
- Wave 评估用于供应商对比
- 典型价格：$3K-$30K/份
- 最适用于：营销技术、客户体验、B2B

**IDC** (https://www.idc.com)
- IT 市场情报和规模测算
- 详细的细分拆分
- 典型价格：$4K-$40K/份
- 最适用于：硬件、软件、IT 服务

**McKinsey** (https://www.mckinsey.com/featured-insights)
- 免费洞察和报告
- 战略性行业分析
- 最适用于：行业趋势、宏观经济背景

### 可获取的研究

**Statista** (https://www.statista.com)
- 价格：$39/月个人版，$199/月企业版
- 覆盖：80,000+ 跨行业主题
- 最适用于：快速市场规模估算、图表、趋势

**CB Insights** (https://www.cbinsights.com)
- 价格：定制报价（通常 $10K+/年）
- 覆盖：风险投资、创业市场
- 最适用于：新兴市场、竞争情报

**PitchBook** (https://pitchbook.com)
- 价格：机构定价
- 覆盖：私营公司估值、并购、VC
- 最适用于：创业公司估值、融资趋势

**Grand View Research** (https://www.grandviewresearch.com)
- 价格：$2K-$5K/份
- 覆盖：B2C 和新兴市场
- 最适用于：消费市场、医疗、清洁技术

## 政府和公共数据

### 美国政府来源

**U.S. Census Bureau** (https://www.census.gov)
- 免费、权威的人口统计数据
- 每 5 年一次经济普查
- 最适用于：企业数量、人口统计、消费支出

**Bureau of Labor Statistics** (https://www.bls.gov)
- 免费的就业和经济数据
- 分行业统计数据
- 最适用于：就业趋势、工资、生产率

**SEC EDGAR** (https://www.sec.gov/edgar)
- 免费的上市公司文件
- 10-K、10-Q 报告含细分收入
- 最适用于：用上市公司数据验证市场规模

**Data.gov** (https://www.data.gov)
- 免费政府数据集
- 跨机构汇总
- 最适用于：专业行业数据

### 国际来源

**OECD** (https://data.oecd.org)
- 免费国际经济数据
- 最适用于：跨国比较

**World Bank** (https://data.worldbank.org)
- 免费全球发展数据
- 最适用于：新兴市场、宏观趋势

**Eurostat** (https://ec.europa.eu/eurostat)
- 免费欧盟统计数据
- 最适用于：欧洲市场规模测算

## 数据质量检查清单

评估数据源时：

**权威性**
- [ ] 谁发布的研究？
- [ ] 他们的声誉如何？
- [ ] 是否具备行业专业能力？

**方法论**
- [ ] 数据如何采集？
- [ ] 样本量多大？
- [ ] 研究何时进行？

**时效性**
- [ ] 数据是否最新（< 2 年）？
- [ ] 市场是否发生了重大变化？
- [ ] 增长率是否仍然适用？

**一致性**
- [ ] 多个来源是否一致？
- [ ] 定义是否统一？
- [ ] 数字能否三角校验？

**相关性**
- [ ] 是否匹配你的市场定义？
- [ ] 地理范围是否合适？
- [ ] 细分是否对齐？

## 免费与付费策略

**从免费来源开始：**
1. 政府数据用于客户计数
2. 上市公司文件用于细分收入
3. 行业协会用于行业趋势
4. Google Scholar 用于学术研究

**升级到付费的条件：**
- 融机构资金时（投资人期望付费来源）
- 需要详细细分拆分
- 市场属于利基或新兴领域
- 免费来源过时或不足

**性价比方案：**
- 购买 1-2 份覆盖核心市场的关键报告
- 用免费来源做三角校验
- 用一手研究（客户访谈）补充
- 混合引用免费和付费来源

## 引用最佳实践

市场规模测算中务必引用来源：

**格式：**
```
Market Size: $X.XB
Source: [Publisher], [Report Name], [Date]
URL: [link if available]
```

**示例：**
```
Email Marketing Software TAM: $7.5B (2024)
Source: Gartner, "Market Share: Email Marketing Software, Worldwide, 2024"
Note: Includes all email marketing software revenue globally
```

**应包含：**
- 出版商和报告名称
- 发布日期
- 地理和范围
- 所做的任何调整
- 来源链接（如公开可获取）

## 紧急研究指南

**2 小时内需要市场规模？**

1. **查 Statista**（15 分钟）— 快速行业概览
2. **找上市公司**（30 分钟）— 从 10-K 获取细分收入
3. **LinkedIn 搜索**（20 分钟）— 统计潜在 B2B 客户
4. **Google Scholar**（20 分钟）— 查找学术论文
5. **自下而上计算**（30 分钟）— 客户数 × 价格
6. **三角校验**（15 分钟）— 比较各来源

**记录一切：**
- 写下所有来源
- 记录所有假设
- 展示方法论
- 标注数据质量限制

有明确限制的可辩护估算，胜过完全没有数据。
