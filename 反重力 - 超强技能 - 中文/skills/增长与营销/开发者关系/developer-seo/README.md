# 开发者 SEO

面向技术型查询和开发者受众的 SEO 策略。

## 本技能涵盖内容

- 面向"如何在 Y 中做 X"类查询的关键词研究
- 错误信息 SEO（捕获复制粘贴式搜索）
- 与 Stack Overflow 和官方文档竞争
- 技术长尾关键词策略
- 在开发者搜索中获得排名的内容形式

## 关键概念

**开发者搜索行为有本质不同**——他们带着明确的技术意图搜索，会复制粘贴错误信息，并会立刻跳出浅薄内容。

**错误信息是 SEO 金矿**——带有原文错误信息和真实修复方案的专页，能捕获高意图流量。

**在官方文档薄弱处竞争**——官方文档往往缺少"为什么"的解释、真实场景示例和故障排查指南。

## 何时使用

- 为开发者工具规划 SEO 策略
- 创作应当获得排名的技术型内容
- 制作错误信息或故障排查类内容
- 竞争"如何做"和对比类查询

## 快速开始

1. 从支持渠道挖掘开发者真实的问题
2. 识别开发者会遇到的错误信息
3. 创作真正能解决问题的有用内容
4. 结构上便于快速扫读（代码前置、解释在后）
5. 追踪特定技术型查询的排名
