# 开发者倾听

跨 GitHub、Hacker News、Reddit、Stack Overflow、Twitter、Discord 等平台监测开发者对话，帮助理解情绪、发现互动机会、收集竞争情报。

## 本技能涵盖内容

- 为品牌、竞品和问题域搭建关键词监测
- 平台针对性监测策略（GitHub、HN、Reddit、SO、Twitter、Discord）
- 情绪分析与优先级排序框架
- 发现高价值互动机会（挫败用户、购买意向、提问）
- 从开发者对话中提炼竞争情报

## 推荐工具

- **社媒监听工具** —— 覆盖开发者平台（GitHub、HN、Reddit、SO、Twitter）
- **平台原生搜索** —— 各平台自带搜索能力，便于针对性监测

## 快速上手

1. 定义关键词分类：品牌、竞品、问题、购买意向
2. 在监测工具中配置这些关键词
3. 每日检查负面情绪和购买意向提及
4. 为常见互动场景建立回复剧本
5. 长期跟踪趋势，提炼战略洞察
