## 🛠️ 怎么"安装"技能到 AI 助手？

> 不同的 AI 助手安装方式不一样，下面列几种常见的：

### Cursor（推荐新手）

1. 把整个 `skills/` 文件夹复制到你的项目根目录
2. Cursor 会自动识别所有 `SKILL.md`
3. 在聊天框直接说技能名（如 `@claude-api`）就能调用

### Claude Code / Cline

1. 把 `skills/` 放到 `~/.claude/skills/`（macOS/Linux）或 `C:\Users\你的名字\.claude\skills\`（Windows）
2. 重启 Claude Code / Cline
3. 在对话里说"使用 xxx 技能"

### ChatGPT / Claude 网页版

1. 打开某个 `SKILL.md`，把把把全文复制把把到聊天框
2. 告诉 AI："请按照以上说明回答我的问题"
3. 之后对话就遵循这个技能的规则

### 其他 AI 工具

参考它们的"自定义指令 / System Prompt / Knowledge Base"功能，把 `SKILL.md` 内容填进去即可。
